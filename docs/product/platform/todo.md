# 账号、平台与客户端架构 — 待办与完成情况

本页面记录账号、通用设置、多语言、客户端本地/云端服务分层及数据同步的已完成和未完成任务，整合了客户端与服务端的技术指标。

## 一、已完成任务列表

- [x] **JWT 账号鉴权与隔离**：实现了手机号/密码注册与登录接口，生成 JWT Bearer 鉴权凭证，并支持全业务的 `user_id` 行级数据隔离。
- [x] **后台设置基础面板**：`/settings` 页面已完成，支持主题切换、系统语言（中/英）以及 API Key 配置等基础参数。
- [x] **通用侧边栏布局**：完成了 `dashboard.vue` 布局重构，优化了底部的账号和设置入口。

### 2. 离线缓存、同步与冲突检测

#### 客户端 (Client)
- [ ] **冲突检测 Modal 开发**：当网络状态恢复或用户成功登录时，拉取本地脏数据列表，显示对比弹窗，提供“以本地为准”和“以云端为准”的控制选项。
