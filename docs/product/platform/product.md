# 账号、平台与客户端架构 — 产品设计

## 一、产品定位

MoonLOL 是一个基于 Bevy 引擎（Rust ECS）的《英雄联盟》复刻项目之上的 **AI Agent 电竞平台**。

用户扮演"电竞经理"的角色：配置、培养、发布 Agent / 选手（英雄 + 决策配置），让 Agent 在 Summoner's Rift 上自动对战、自动上分。

平台同时服务三类人群：
- **研究者**：调试 Agent 策略、做对照实验、回溯决策链。
- **普通玩家**：在公共房间社交观战、引入社区 Agent 组局。
- **竞技作者**：打磨 Agent 冲天梯、追求排名。

---

## 二、双端形态与定位

平台支持桌面客户端 (Desktop) 和浏览器网页端 (Web) 两种形态：

| 客户端 | 定位 | 能力范围 |
|---|---|---|
| **Desktop（Tauri + Vue）** | 专业调试版 + 全功能 | 本地对局（独占） + 房间 + Rank + 观战 + 完整调试面板 |
| **Web（浏览器）** | 轻量消费端 | 房间 + Rank + 观战 + Agent/预设管理（不含本地对局） |

Desktop 端本地对局以本地预设库为主，可离线运行；登录后可手动把选手、场景等配置同步到云端。

---

## 三、账号与社交

- **注册与登录**：直接向公众开放注册。目前支持且仅支持**手机号 + 验证码**的注册、登录与密码找回逻辑。
- **社交系统（规划中）**：
  - 好友 / 关注系统。
  - 战队系统（未来用于“职业比赛模式”，支持战队之间组织对局）。

---

## 四、客户端系统设置

提供完善的双端系统设置界面：
- **常规设置**：系统主题切换、语言选择（中/英）、窗口缩放比例、终端配置、HTTP 代理地址以及任务通知。
- **模型设置**：管理模型供应商与模型列表，预设国内外厂商目录（智谱、DeepSeek、火山方舟、Kimi 等）并支持完全自定义供应商；配置后供 Agent 选手编辑页以「选供应商 + 选模型」的方式选用，而非手填。详见 [模型供应商与模型设置](../llm-provider-setting/product.md)。
- **高级配置（规划中）**：代码预览、技能配置、MCP 服务器管理、自定义插件、内置命令行、索引库以及使用情况数据统计。
