#  Samira (莎弥拉)

## 基础信息

- **称号**: 莎弥拉 (The Desert Rose)
- **类型**: 物理 / 远程
- **难度**: 3/3
- **角色**: 射手、刺客
- **位置**: ADC、辅助

## 技能介绍

### 被动 - 勇敢冲动 (Daredevil Impulse)

**机制**: Samira攻击生成Style层数，达到满层可以释放大招。

**Style层数**: 6层
**移速加成**: 每层2.75-3.5%
**刀锋攻击**: 近距离普攻变成刀锋攻击

### Q - 飞镖 (Flair)

- **冷却**: 6/5/4/3/2 秒
- **消耗**: 30 法力
- **范围**: 950

Samira发射子弹或挥刀。

**伤害**: 0/5/10/15/20 (+ 110% AD)
**可暴击**: 150%伤害

### W - 刀锋回旋 (Blade Whirl)

- **冷却**: 30/28/26/24/22 秒
- **消耗**: 60 法力
- **范围**: 325

Samira旋转刀锋，摧毁飞行道具并伤害周围敌人。

**每击伤害**: 20/35/50/65/80 (+ 50% 额外AD)
**持续时间**: 0.75秒

### E - 狂野冲刺 (Wild Rush)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 40 法力
- **范围**: 600

Samira冲向敌人，伤害并获得攻速。

**伤害**: 50/60/70/80/90 (+ 20% 额外AD)
**攻速加成**: 20/25/30/35/40%
**助攻刷新**: 参与击杀后刷新

### R - 炼狱扳机 (Inferno Trigger)

- **冷却**: 5 秒
- **消耗**: 6 Style
- **范围**: 600

Samira释放大规模射击，造成高额伤害。

**每击伤害**: 20/40/60 (+ 30% AD)
**射击次数**: 最多10次
**持续时间**: 2.277秒

## 连招技巧

### 基础连招

**标准连招**:
- Q(飞镖) → E(狂野冲刺) → W(刀锋回旋) → 普攻
- 近身后持续输出

**Style积攒**:
- Q(飞镖) → 普攻 → E(狂野冲刺) → Q(飞镖)
- 快速积攒Style

### 进阶连招

**团战连招**:
- E(狂野冲刺) → W(刀锋回旋) → R(炼狱扳机) → Q(飞镖)
- 冲刺入场后开大

**单杀连招**:
- Q(飞镖) → E(狂野冲刺) → 普攻 → W(刀锋回旋) → R(炼狱扳机)
- 快速接近后爆发

## 隐藏机制

1. **Style系统**: 攻击生成层数，满层开大
2. **刀锋攻击**: 近距离普攻变成刀锋
3. **W挡技能**: W可以挡飞行道具
4. **E位移**: E可以位移和刷新
5. **R爆发**: R是主要爆发技能
