# 亚索 (Yasuo)

## 基础信息

- **称号**: 永诀 (The Unforgiven)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、刺客
- **位置**: 上单、中单、ADC

## 技能介绍

### 被动 - 浪客之道 (Way of the Wanderer)

**暴击转化**:
- 亚索的总暴击率翻倍计算
- 超过100%的暴击率每1%转化为0.5额外攻击力
- 暴击只造成180%伤害（受到10%惩罚）

**护盾机制**:
- 亚索每移动一定距离获得一层"气流"（Flow）
- 满100层时受到敌方英雄或野怪伤害会消耗护盾

**护盾值**: 125-674 (基于等级)

### Q - 钢影斩 (Steel Tempest)

- **冷却**: 4-1.33秒（基于额外攻速）
- **消耗**: 无
- **范围**: 450 / 1150（龙卷风形态）

亚索向前突刺，造成物理伤害并触发攻击特效。

**伤害**: 20/45/70/95/120 (+ 105% AD)

**风暴掌握（Gathering Storm）**:
- 命中敌人获得一层风暴（持续6秒），最多2层
- 2层时Q变为强化龙卷风，击飞敌人0.9秒

### W - 风之障壁 (Wind Wall)

- **冷却**: 25/23/21/19/17 秒
- **消耗**: 无
- **范围**: 320-600（宽度随技能等级变化）

亚索创造一道风墙，持续4秒，阻挡所有敌方飞行道具。

**机制**:
- 风墙会向施法方向移动0.6秒，然后漂浮50单位
- 提供风墙周围地区的视野

### E - 踏前斩 (Sweeping Blade)

- **冷却**: 0.5-0.1 秒（基于技能等级）
- **消耗**: 无
- **范围**: 475 / 625

亚索向目标突进，对目标造成魔法伤害。每次突进获得一层"乘风"效果（持续5秒），最多4层。

**伤害**: 70/85/100/115/130 (+ 20% bonus AD)(+ 60% AP)
**每层伤害加成**: 17.5/21.25/25/28.75/32.5 (+ 5% bonus AD)(+ 15% AP)

### R - 狂风绝息斩 (Last Breath)

- **冷却**: 70/50/30 秒
- **消耗**: 无
- **范围**: 1400

亚索闪烁到附近空中敌人旁边，将其击飞1秒并造成物理伤害。

**伤害**: 200/350/500 (+ 150% bonus AD)

**额外效果**:
- 获得最大气流（Flow）
- 接下来15秒内，暴击无视目标60%额外护甲

## 连招技巧

### 基础连招

**吹飞接R**:
- Q(龙卷风) → R(狂风绝息斩)
- 基础的控制连招

**消耗连招**:
- E → Q → E → 普攻
- 利用E的位移接近并触发伤害

### 进阶连招

**击飞控制**:
- Q(龙卷风) → R → Q → E → Q → 普攻
- 利用R的击飞延长控制时间

**安全输出**:
- W(风墙) → 普攻 → E → Q
- 风墙阻挡敌方飞行道具后输出

**持续输出**:
- E × 4层 → Q(龙卷风) → R
- 叠满乘风层数后打出最高伤害

## 隐藏机制

1. **暴击机制**: 亚索的暴击率翻倍，但暴击伤害只有180%（有10%惩罚）
2. **E技能CD**: E技能冷却随等级减少，满级只有0.1秒
3. **风墙位置**: 风墙在施法方向移动后会在新位置停留
4. **R距离**: R的实际距离比显示的1400更远
5. **Q可以暴击**: Q技能的伤害可以暴击
6. **E不触发攻击特效**: E技能的伤害不触发攻击特效（如贪欲猎手）
7. **R必须目标浮空**: R需要目标处于浮空状态才能施放
