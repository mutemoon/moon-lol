# 刀妹 (Irelia)

## 基础信息

- **称号**: 舞剑精灵 (the Blade Dancer)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、刺客
- **位置**: 上单、中单

## 技能介绍

### 被动 - 艾欧尼亚热诚 (Ionian Fervor)

艾瑞莉娅用技能命中敌人时获得叠加的攻击速度。
- 每层获得8%攻击速度
- 最高4层
- 满层时获得额外伤害
- 移速加成与层数相关

**效果数值**:
- 攻击速度: 8/12/16/20% 每层（满级4层）
- 额外伤害: 30/50/70/90 (1-18级)

### Q - 利刃冲击 (Bladesurge)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 20/25/30/35/40 法力
- **范围**: 600

艾瑞莉娅冲向敌人，造成物理伤害并治疗自身。
- 如果敌人死亡或处于"不稳"状态，冷却刷新
- 对小兵伤害降低

**效果数值**:
- 伤害: 20/40/60/80/100 (+0.6 AD)
- 小兵伤害: 60%
- 治疗量: 15 (+0.2 AP)

**隐藏机制**:
- Q可以重置平A
- Q击杀小兵刷新冷却是核心续航机制
- Q可以穿越薄墙
- Q目标优先级：可击杀单位 > 悬停目标

### W - 防御之舞 (Defiant Dance)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 825

开始蓄力：艾瑞莉娅进入防御姿态，最多2秒，无法行动但减免伤害。
释放：艾瑞莉娅挥动双刃，造成基于蓄力时间的物理伤害。

**效果数值**:
- 蓄力时间: 最多2秒
- 物理减伤: 30/35/40/45/50%
- 魔法减伤: 30/35/40/45/50%
- 最小伤害: 40/80/120/160/200 (+0.6 AD)
- 最大伤害: 80/160/240/320/400 (+1.2 AD)

**隐藏机制**:
- W蓄力期间无法行动，但可以被打断
- W可以取消Q的后摇
- W的伤害基于蓄力时间，满蓄力才有最大伤害
- W蓄力可以被E、R等技能取消

### E - 完美合奏 (Flawless Duet)

- **冷却**: 16/14.5/13/11.5/10 秒
- **消耗**: 30 法力
- **范围**: 850

艾瑞莉娅向地面投掷一把匕首，可以重施。
重施：投掷第二把匕首，两把匕首飞向彼此，眩晕并造成魔法伤害。
- 英雄和大型野怪变为"不稳"状态

**效果数值**:
- 眩晕时间: 0.75/1/1.25/1.5/1.75 秒
- 伤害: 40/90/140/190/240 (+0.7 AP)
- 不稳标记持续: 4秒

**隐藏机制**:
- E第一段可以标记地形（墙体、果实）
- E两段之间有4秒窗口
- E命中后自己会跳向两把刀的交点
- 不稳状态下的敌人被Q命中会刷新Q冷却

### R - 先锋之刃 (Vanguard's Edge)

- **冷却**: 125/105/85 秒
- **消耗**: 100 法力
- **范围**: 950

艾瑞莉娅发射一排匕首，造成魔法伤害并标记英雄和大型野怪为"不稳"。
- 匕首在第一个英雄处爆炸形成禁锢领域
- 领域持续造成伤害并减速

**效果数值**:
- 伤害: 100/200/300 (+0.6 AP)
- 眩晕/减速持续: 1.5/2/2.5 秒
- 禁锢领域持续: 2.5/3/3.5 秒
- 领域内伤害: 50/90/130 (+0.5 AP)
- 减速: 30/40/50%

**隐藏机制**:
- R可以触发两次E（因为R会标记"不稳"）
- R命中的敌人被E命中会刷新Q冷却
- R的禁锢可以被控制技能打断
- R可以用来清兵

## 连招技巧

### 基础连招

**E+E连招**:
- E1投掷 → 等待落点 → E2
- 命中后自己会跳向交点

**对线换血**:
- E+E → Q → 平A → W → 撤退
- 利用E眩晕打一波换血

**Q追击**:
- Q → 平A → Q → 平A
- 对不稳状态敌人Q可以刷新

### 进阶连招

**E+R+F Combo**:
- E1投掷 → R（立即） → F（闪向敌人） → E2眩晕
- 超远距离开团

**E+E+Q连招**:
- E1 → E2 → 立即Q追击
- E2眩晕后Q可以刷新

**RWQ Combo**:
- R → W（蓄力1秒） → Q → E → E
- R标记"不稳"，Q跟进

**R闪+E Combo**:
- R → F → Q → E → E
- R命中的敌人立即进入"不稳"

### 团战连招

**团战入场**:
- E+E → R → Q → 平A → W → 平A
- 先手E眩晕，然后R标记

**追人连招**:
- R命中的敌人 → Q → E → E → Q
- 利用Q刷新持续追击

## 不稳状态详解

"不稳"是刀妹的核心机制：
- 被E或R标记
- 持续4秒
- 不稳状态下被Q命中，Q冷却刷新
- 可以利用这一点进行无限Q追击

**刷新Q连招**:
- E+E眩晕 → Q → 平A → E+E → Q → 平A...
- 只要保持不稳状态，就可以无限Q追击

---

## 隐藏机制

1. **Q重置平A**: Q后可以立即接平A
2. **E可以标记地形**: E第一段可以扔到墙上、果实上
3. **R可以触发E**: R会标记"不稳"，Q可以刷新
4. **W取消Q后摇**: W可以取消Q的后摇
5. **E命中后跳向交点**: 刀妹自己会跳向两把刀的交叉点
6. **Q可以穿墙**: 一些薄墙可以被Q穿越
7. **被动层数与移速**: 满层被动提供额外移速
8. **W蓄力可被取消**: W蓄力时可以用R等技能取消
9. **R的范围**: R可以命中路径上所有敌人
10. **E两段窗口**: 两段E之间有4秒时间
11. **Q对小兵伤害**: Q对小兵只有60%伤害
12. **W减伤机制**: 蓄力时减免所有物理和魔法伤害

## 常见问题

**Q: 刀妹怎么保持连招流畅？**
A: 核心是利用E眩晕和R标记的"不稳"状态。保持不稳时Q会刷新，可以无限追击。

**Q: E的第一段怎么提高命中率？**
A: E第一段可以扔到墙体或果实上，然后等敌人走到合适位置再E2。也可以配合R先标记敌人。

**Q: 刀妹对线怎么换血？**
A: 利用E先手眩晕，打出E+E+Q+A+W+A，然后撤退。等E冷却好了再打一波。

**Q: W什么时候用？**
A: W主要用于减免伤害和取消Q后摇。蓄力时间越长伤害越高，但要注意被打断。

**Q: 刀妹的被动移速怎么利用？**
A: 满层被动提供大量移速追击或撤退。对线期保持4层被动可以获得很大优势。
