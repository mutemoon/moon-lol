#  Ekko (艾克)

## 基础信息

- **称号**: 艾克 (The Boy Who Shattered Time)
- **类型**: 魔法 / 近战
- **难度**: 3/3
- **角色**: 刺客
- **位置**: 打野、中单

## 技能介绍

### 被动 - Z型驱动共振 (Z-Drive Resonance)

**机制**: Ekko的普攻和技能对敌人施加共振层数，最多3层。第三层触发时造成额外魔法伤害。

**额外伤害**: 30-150 (基于等级) (+ 80% AP)
**移速加成**: 50-80% (基于等级)，持续2-3秒

### Q - 时间卷曲器 (Timewinder)

- **冷却**: 9/8.5/8/7.5/7 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 1100

Ekko向目标方向投掷时间手雷，对敌人造成伤害并留下减速区域。手雷返回时再次造成伤害。

**伤害**: 80/95/110/125/140 (+ 30% AP)
**返回伤害**: 40/65/90/115/140 (+ 60% AP)
**减速**: 40/45/50/55/60%

### W - 平行世界 (Parallel Convergence)

- **冷却**: 22/20/18/16/14 秒
- **消耗**: 30/35/40/45/50 法力
- **范围**: 1600

Ekko创造一个幻像，2秒后变成一个领域，减速敌人。如果Ekko进入领域，触发爆炸眩晕敌人。

**护盾值**: 100/120/140/160/180 (+ 150% AP)
**眩晕时间**: 2.25秒
**减速**: 40%

### E - 相位冲刺 (Phase Dive)

- **冷却**: 9/8.5/8/7.5/7 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 325/550

Ekko向目标方向突进，强化下一次普攻。

**伤害**: 50/75/100/125/150 (+ 40% AP)

### R - 时空断裂 (Chronobreak)

- **冷却**: 110/80/50 秒
- **消耗**: 100 法力
- **范围**: 全地图

Ekko进入凝滞状态，然后返回4秒前的位置，对附近敌人造成伤害并治疗自己。

**伤害**: 200/350/500 (+ 175% AP)
**治疗量**: 100/150/200 (+ 60% AP)

## 连招技巧

### 基础连招

**标准爆发**:
- Q(时间卷曲器) → E(相位冲刺) → 普攻 → W(平行世界)
- Q先手，E接近，W控制

**反手连招**:
- W(平行世界) → E(相位冲刺) → R(时空断裂)
- W放置领域，E进入眩晕

### 进阶连招

**团战连招**:
- R(时空断裂) → W(平行世界) → Q(时间卷曲器) → E(相位冲刺)
- R返回战场，W控制

**追杀连招**:
- Q(时间卷曲器) → E(相位冲刺) → 普攻 × 3
- Q减速，E追击

---

## 隐藏机制

1. **被动移速**: 对英雄触发被动时获得大量移速
2. **W眩晕**: Ekko进入W领域可以触发眩晕
3. **E强化普攻**: E后获得强化普攻
4. **R返回**: R可以返回4秒前的位置
5. **Q返回伤害**: Q手雷返回时造成额外伤害
