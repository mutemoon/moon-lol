#  Gragas (格雷福斯)

## 基础信息

- **称号**: 格雷福斯 (The Rabble Rouser)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 战士、坦克
- **位置**: 打野

## 技能介绍

### 被动 - 欢乐时光 (Happy Hour)

**机制**: Gragas施放技能后定期回复生命值。

**治疗量**: 5.5% 最大生命值

### Q - 滚动酒桶 (Barrel Roll)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 80 法力
- **范围**: 850

Gragas放置一个酒桶，可以再次激活引爆。酒桶会随时间增强。

**伤害**: 80-240 (+ 80% AP)
**减速**: 40-60%

### W - 醉酒狂暴 (Drunken Rage)

- **冷却**: 5 秒
- **消耗**: 30 法力

Gragas引导0.75秒，获得伤害减免。引导完成后，强化下一次普攻。

**伤害减免**: 10-18% (+ 4% AP)
**额外伤害**: 20-140 (+ 7% 目标最大生命值) (+ 70% AP)

### E - 肉弹冲击 (Body Slam)

- **冷却**: 14/13.5/13/12.5/12 秒
- **消耗**: 50 法力
- **范围**: 600

Gragas向目标方向冲撞，击飞并眩晕命中的敌人。

**伤害**: 80/125/170/215/260 (+ 60% AP)
**眩晕时间**: 1秒

### R - 爆炸酒桶 (Explosive Cask)

- **冷却**: 100/85/70 秒
- **消耗**: 100 法力
- **范围**: 1000

Gragas投掷一个爆炸酒桶，将敌人击退。

**伤害**: 200/300/400 (+ 80% AP)
**击退距离**: 900

## 连招技巧

### 基础连招

**标准消耗**:
- Q(滚动酒桶) → E(肉弹冲击) → W(醉酒狂暴)
- Q先手，E接近

**团控连招**:
- R(爆炸酒桶) → Q(滚动酒桶) → E(肉弹冲击)
- R先手击退

### 进阶连招

**团战连招**:
- R(爆炸酒桶) → E(肉弹冲击) → Q(滚动酒桶) → W(醉酒狂暴)
- R控制，E击飞

**追杀连招**:
- E(肉弹冲击) → Q(滚动酒桶) → W(醉酒狂暴)
- E先手接近

---

## 隐藏机制

1. **Q引爆**: Q可以再次激活引爆酒桶
2. **W减伤**: W引导时获得伤害减免
3. **E击飞**: E可以击飞并眩晕敌人
4. **被动治疗**: 施放技能后回复生命
5. **R击退**: R可以击退大量敌人
