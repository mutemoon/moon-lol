#  Rammus (拉莫斯)

## 基础信息

- **称号**: 拉莫斯 (The Armordillo)
- **类型**: 物理 / 近战
- **难度**: 1/3
- **角色**: 坦克、辅助
- **位置**: 打野、辅助

## 技能介绍

### 被动 - 尖刺外壳 (Spiked Shell)

**机制**: Rammus获得额外攻击力，基于护甲和魔抗。

**额外AD**: 15% 护甲 + 15% 魔抗

### Q - 动力球 (Powerball)

- **冷却**: 12/10.5/9/7.5/6 秒
- **消耗**: 60 法力
- **范围**: 250

Rammus变成滚动球，加速并击飞敌人。

**伤害**: 80/120/160/200/240 (+ 100% AP)
**减速**: 40/50/60/70/80%
**击飞时间**: 0.4秒

### W - 防御姿态 (Defensive Ball Curl)

- **冷却**: 7 秒
- **消耗**: 40 法力
- **范围**: 自助

Rammus缩成球状，获得高额双抗并反弹伤害。

**护甲加成**: 27/32/37/42/47 (+ 30/37.5/45/52.5/60% 护甲)
**魔抗加成**: 20/25/30/35/40 (+ 30/37.5/45/52.5/60% 魔抗)
**反伤**: 15 (+ 10% 护甲) (+ 10% 魔抗)

### E - 狂乱嘲讽 (Frenzying Taunt)

- **冷却**: 12 秒
- **消耗**: 50 法力
- **范围**: 325

Rammus嘲讽敌人，使其攻击自己。

**嘲讽时间**: 1.2/1.4/1.6/1.8/2秒
**野怪伤害**: 80/100/120/140/160 (+ 70% AP)

### R - 冲天一击 (Soaring Slam)

- **冷却**: 120/105/90 秒
- **消耗**: 100 法力
- **范围**: 800

Rammus跃向目标，造成范围伤害和减速。

**伤害**: 150/250/350 (+ 60% AP)
**减速**: 30/40/50%
**持续时间**: 1.5秒

## 连招技巧

### 基础连招

**Gank连招**:
- Q(动力球) → E(狂乱嘲讽) → W(防御姿态)
- 滚动接近后嘲讽

**团战连招**:
- Q(动力球) → R(冲天一击) → W(防御姿态)
- 滚动起手跳跃

### 进阶连招

**开团连招**:
- Q(动力球) → E(狂乱嘲讽) → R(冲天一击) → W(防御姿态)
- 滚动嘲讽后跳跃

**追击连招**:
- Q(动力球) → W(防御姿态) → E(狂乱嘲讽)
- 滚动追击

---

## 隐藏机制

1. **被动AD**: 被动提供基于双抗的攻击力
2. **Q击飞**: 滚动可以击飞敌人
3. **W反伤**: 防御姿态反弹伤害
4. **E嘲讽**: 强制敌人攻击自己
5. **R跳跃**: 大招可以超远距离开团
