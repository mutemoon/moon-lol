#  Shen (慎)

## 基础信息

- **称号**: 慎 (The Eye of Twilight)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 坦克、战士
- **位置**: 上单、辅助

## 技能介绍

### 被动 - 气之壁垒 (Ki Barrier)

**机制**: Shen技能命中敌人后获得护盾。

**护盾值**: 47-128 (+ 13% 额外生命值)
**冷却**: 11秒
**冷却减免**: 命中敌人减少4-8秒

### Q - 黄昏冲击 (Twilight Assault)

- **冷却**: 8/7.25/6.5/5.75/5 秒
- **消耗**: 140/130/120/110/100 能量
- **范围**: 全图

Shen召回灵刃，对路径上敌人造成伤害和减速。

**伤害**: 10-40 (+ 2-4% 最大生命值)
**减速**: 25/30/35/40/45%
**强化普攻**: 3次普攻获得额外伤害

### W - 灵魂庇护 (Spirit's Refuge)

- **冷却**: 16/14.5/13/11.5/10 秒
- **消耗**: 40 能量
- **范围**: 350

Shen创造一个区域，阻挡友军受到的普攻。

**持续时间**: 1.75秒
**区域大小**: 350范围

### E - 影缚 (Shadow Dash)

- **冷却**: 18/16/14/12/10 秒
- **消耗**: 150 能量
- **范围**: 600

Shen冲刺，伤害并嘲讽命中的敌人。

**伤害**: 60/85/110/135/160 (+ 11% 额外生命值)
**嘲讽时间**: 1.5秒
**能量回复**: 30/40/50

### R - 一起 (Stand United)

- **冷却**: 200/180/160 秒
- **消耗**: 自助
- **范围**: 全图

Shen为友军提供护盾，然后传送至友军身边。

**护盾值**: 120/220/320 (+ 135% AP) (+ 15% 额外生命值)
**增盾效果**: 基于友军损失生命值增加

## 连招技巧

### 基础连招

**对线连招**:
- E(影缚) → Q(黄昏冲击) → A(普攻)
- 嘲讽后跟进

**团战连招**:
- R(一起) → E(影缚) → Q(黄昏冲击) → W(灵魂庇护)
- 大招保护后团控

### 进阶连招

**全球支援**:
- R(一起) → E(影缚) → Q(黄昏冲击)
- 大招全球支援友军

**防守连招**:
- W(灵魂庇护) → E(影缚) → Q(黄昏冲击)
- 保护友军后反打

---

## 隐藏机制

1. **气刃系统**: 灵刃可以远程控制
2. **W挡普攻**: W可以挡友军的普攻
3. **E嘲讽**: E可以嘲讽多个敌人
4. **R全球**: 大招可以全图支援
5. **被动护盾**: 技能命中敌人后获得护盾
