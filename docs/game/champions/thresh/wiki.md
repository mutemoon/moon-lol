# 锤石 (Thresh)

## 基础信息

- **称号**: 魂锁典狱官 (The Chain Warden)
- **类型**: 物理 / 远程
- **难度**: 3/3
- **角色**: 辅助
- **位置**: 辅助

## 技能介绍

### 被动 - 诅咒 (Damnation)

**机制**: 锤石周围死亡的敌方英雄、大型小兵和大型野怪会掉落灵魂。

- 灵魂可以被锤石自动收集
- 每个灵魂提供1点法术强度和1点护甲加成
- 灵魂存在8秒

**注意**: 锤石的护甲不会随等级增长

### Q - 死亡判决 (Death Sentence)

- **冷却**: 19/16.5/14/11.5/9 秒
- **消耗**: 70 法力
- **范围**: 1100

锤石投出镰刀，对第一个命中的敌人造成魔法伤害，眩晕1.5秒并击飞0.4秒。

**伤害**: 100/150/200/250/300 (+ 90% AP)

**额外效果**:
- 命中后减少2秒冷却
- 可以再次激活将目标拉向自己
- 再次激活（Deathly Leap）：锤石会冲向被束缚的敌人

### W - 魂引之灯 (Dark Passage)

- **冷却**: 21/20/19/18/17 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 950

锤石投出灯笼，为范围内第一个友军提供护盾。友军可以点击灯笼向锤石位移。

**护盾值**: 50/70/90/110/130 (+ 每灵魂+2)

**灯笼机制**:
- 持续6秒
- 锤石离灯笼太远时灯笼会返回
- 提供视野

### E - 厄运钟摆 (Flay)

- **冷却**: 13/12.25/11.5/10.75/10 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 525（主动）/ 270（被动）

**被动**: 锤石的普攻造成额外魔法伤害，伤害随时间增加（10秒无攻击时伤害最高）

**主动**: 锤石挥舞链条，击退并减速命中的敌人。

**伤害**: 75/120/165/210/255 (+ 70% AP)
**减速**: 20/25/30/35/40%

### R - 幽魂监牢 (The Box)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 470（五边形墙壁）

锤石创造一个五边形幽魂墙壁，每个墙壁持续5秒。第一个触碰墙壁的敌人受到伤害和减速，后续墙壁只造成减速。

**伤害**: 250/400/550 (+ 100% AP)
**减速**: 99% (第一个墙壁) / 50% (后续墙壁)

## 连招技巧

### 基础连招

**标准勾人**:
- Q(死亡判决) → 再次激活Q(拉近) → E(厄运钟摆)
- Q先手勾人，拉近后用E击退

**辅助连招**:
- W(魂引之灯) → Q(死亡判决)
- 先给队友护盾再勾人

### 进阶连招

**团控连招**:
- Q(死亡判决) → R(幽魂监牢) → E(厄运钟摆)
- Q勾人进R的墙壁范围，E将敌人留在墙壁内

**追击连招**:
- E(厄运钟摆) → Q(死亡判决) → 再次激活Q
- E先减速，Q勾人

**保护连招**:
- R(幽魂监牢) → W(魂引之灯)
- R保护自己，W保护队友

## 隐藏机制

1. **Q蓄力**: Q有短暂的施法时间，期间锤石不能移动或攻击
2. **Q拉人**: Q命中后可以再次激活将目标拉向自己
3. **E可以打断**: E可以打断某些引导技能
4. **W位移**: 友军点击灯笼可以位移到锤石身边
5. **R墙壁**: R的墙壁会按顺序触发，第一个伤害最高
6. **被动收集**: 锤石可以自动收集附近的灵魂
7. **E被动伤害**: E的被动伤害随时间增加，10秒不攻击最高
