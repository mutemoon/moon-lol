#  Nautilus (诺提勒斯)

## 基础信息

- **称号**: 诺提勒斯 (The Titan of the Depths)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 坦克、辅助
- **位置**: 辅助

## 技能介绍

### 被动 - 猛冲重击 (Staggering Blow)

**机制**: Nautilus的普攻造成额外伤害并击飞敌人。

**额外伤害**: 14-128 (基于等级)
**击飞时间**: 0.75-1.5秒
**冷却**: 6秒

### Q - 疏浚之线 (Dredge Line)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 60 法力
- **范围**: 1122

Nautilus投掷锚子，造成伤害、眩晕并拖拽目标。

**伤害**: 85/130/175/220/265 (+ 90% AP)
**眩晕时间**: 1秒
**拖拽时间**: 0.9秒
**地形命中**: 返还50%冷却和蓝耗

### W - 泰坦的愤怒 (Titan's Wrath)

- **冷却**: 12 秒
- **消耗**: 60 法力
- **范围**: 250

Nautilus为自己提供护盾，普攻附加持续伤害。

**护盾值**: 50/60/70/80/90 (+ 8/9/10/11/12% 最大生命值)
**持续伤害**: 30/40/50/60/70 (+ 40% AP)

### E - 潮汐 (Riptide)

- **冷却**: 7/6.5/6/5.5/5 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 350

Nautilus发出三波爆炸，伤害并减速敌人。

**伤害**: 55/90/125/160/195 (+ 50% AP)
**减速**: 30/35/40/45/50%
**减速持续**: 1.5秒

### R - 深渊处刑 (Depth Charge)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 825

Nautilus发射深渊导弹追踪敌人，命中后击飞。

**伤害**: 125/175/225 (+ 40% AP)
**击飞时间**: 1/1.5/2秒
**强化伤害**: 150/275/400 (+ 80% AP)

## 连招技巧

### 基础连招

**标准连招**:
- Q(疏浚之线) → W(泰坦的愤怒) → E(潮汐)
- 锚起手后护盾

**对线连招**:
- E(潮汐) → Q(疏浚之线) → W(泰坦的愤怒)
- 减速后钩取

### 进阶连招

**团控连招**:
- R(深渊处刑) → Q(疏浚之线) → W(泰坦的愤怒) → E(潮汐)
- 大招起手团控

**保护连招**:
- W(泰坦的愤怒) → Q(疏浚之线) → E(潮汐)
- 护盾后控制

## 隐藏机制

1. **被动击飞**: 普攻周期性击飞敌人
2. **Q拖拽**: Q可以拖拽敌人或钩地形位移
3. **W护盾**: 护盾值基于最大生命值
4. **E三段伤害**: E有三波伤害
5. **R追踪**: R自动追踪目标
