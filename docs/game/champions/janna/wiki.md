#  Janna (风暴之怒)

## 基础信息

- **称号**: 风暴之怒 (The Storm's Fury)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 辅助
- **位置**: 辅助
- **资源**: 法力
- **攻击距离**: 550

## 技能介绍

### 被动 - 顺风而行 (Tailwind)

**类型**: 环境型被动
**效果范围**: 1200

Janna为附近面向她位置的友方英雄提供6%额外移动速度。

Janna的普通攻击和Zephyr技能的命中效果会造成额外魔法伤害，等于她额外移动速度的30%。

### Q - 呼啸风暴 (Howling Gale)

- **冷却**: 14 秒
- **消耗**: 90/95/100/105/110 法力
- **施法时间**: 无
- **范围**: 1100-1760 (基于充能秒数)
- **宽度**: 240
- **速度**: 880-1408 (基于充能秒数)

Janna在当前位置召唤一个旋风，充能3秒，在持续时间内增加其范围、速度、伤害和击飞持续时间，每秒增加一次。

**最小魔法伤害**: 55/90/125/160/195 (+ 50% AP)
**每秒额外伤害**: 10/15/20/25/30 (+ 10% AP)

Howling Gale可以在持续时间内随时再次施放，并在持续时间结束后自动施放。

**再次施放**: Janna向目标方向发射旋风，对命中的敌人造成魔法伤害，并将其击飞0-1.25秒（基于充能秒数）。

### W - 鹰风 (Zephyr)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 50/55/60/65/70 法力
- **施法时间**: 0.245秒
- **目标范围**: 550
- **速度**: 1600

**被动**: Janna是幽灵单位，获得额外移动速度。

**额外移动速度**: 6/7/8/9/10% (+ 2% 每100 AP)

**主动**: Janna向目标敌人发送空气元素，造成魔法伤害并减速2秒，上限为99%。

**魔法伤害**: 55/85/115/145/175 (+ 50% AP) (+ 30% 额外移动速度)
**减速**: 20/24/28/32/36% (+ 6% 每100 AP)

### E - 风暴之眼 (Eye of the Storm)

- **冷却**: 16/15/14/13/12 秒
- **消耗**: 70/75/80/85/90 法力
- **施法时间**: 无
- **目标范围**: 800

**被动**: 每当Janna的技能命中至少一个敌方英雄并将其减速或击飞时，Eye of the Storm的20%冷却时间会被返还。这每次技能施放只能发生一次。

**主动**: Janna授予目标友方英雄或防御塔，或她自己，一个持续4秒的护盾。

**护盾强度**: 80/120/160/200/240 (+ 55% AP)

护盾存在期间，目标获得额外攻击力。

**额外攻击力**: 10/15/20/25/30 (+ 10% AP)

### R - 孟番 (Monsoon)

- **冷却**: 130/115/100 秒
- **消耗**: 100 法力
- **施法时间**: 无
- **效果范围**: 700

**主动**: Janna释放一阵冲击波，将所有附近敌人击退到最多875单位之外，基于他们与她的接近程度，但不会穿过地形。

Janna也会引导最多3秒，每0.25秒治疗她和附近友军。

**每跳治疗**: 25/37.5/50 (+ 12.5% AP)

## 连招技巧

### 基础连招

**保护连招**:
- E(风暴之眼) → W(鹰风)
- 护盾后减速敌人

**进攻连招**:
- Q(呼啸风暴) → W(鹰风) → R(孟番)
- 击飞后追击

### 进阶连招

**团战连招**:
- R(孟番) → Q(呼啸风暴) → E(风暴之眼)
- 大范围控制保护

**反开连招**:
- E(风暴之眼) → R(孟番) → Q(呼啸风暴)
- 反手保护并控制

## 隐藏机制

1. **Q可蓄力**: Q可以蓄能增加距离、速度、伤害和击飞时间
2. **W幽灵**: W被动使Janna永久幽灵状态
3. **E冷却返还**: 技能命中敌方英雄并造成减速或击飞时返还20%冷却
4. **R击退**: R可以将敌人击退并引导治疗
5. **被动伤害**: 普攻和W的命中效果基于额外移速造成额外魔法伤害
