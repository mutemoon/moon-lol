# 塞拉斯 (Sylas)

## 基础信息

- **称号**: 解放者 (the Unshackled)
- **类型**: 魔法 / 近战
- **难度**: 3/3
- **角色**: 法师、刺客
- **位置**: 中单、打野

## 技能介绍

### 被动 - 佩特里克爆发 (Petricite Burst)

塞拉斯施放技能后会存储一层佩特里克爆发充能。
- 塞拉斯的普通攻击会消耗充能，旋转 energized chains 造成额外魔法伤害
- 拥有充能时获得攻击速度加成

**效果数值**:
- 攻击速度加成: 30%
- 额外伤害: 20-60 (基于等级) (+0.25 AP)
- 充能持续: 4秒

**隐藏机制**:
- 被动技能伤害享受AP加成
- 充能可以被多个小兵/英雄同时命中
- 攻速加成有助于快速消耗充能

### Q - 锁链鞭打 (Chain Lash)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 775

塞拉斯甩出锁链，造成魔法伤害并减速。
- 锁链交汇处会爆炸，造成额外伤害
- Q有两段伤害

**效果数值**:
- 基础伤害: 50/70/90/110/130 (+0.75 AP)
- 爆炸伤害: 70/100/130/160/190 (+0.75 AP)
- 减速: 25/30/35/40/45%，持续2秒

**隐藏机制**:
- Q的两段伤害可以叠加
- 减速效果从命中时开始计算
- Q可以用来清兵和poke

### W - 弑君者 (Kingslayer)

- **冷却**: 12/10.5/9/7.5/6 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 400

塞拉斯向敌人突进，造成魔法伤害。
- 对英雄：根据已损失生命值回复生命
- 低血量时治疗效果最大化

**效果数值**:
- 伤害: 60/95/130/165/200 (+0.5 AP)
- 最小治疗: 20/30/40/50/60 (+0.2 AP)
- 最大治疗: 80/120/160/200/240 (+0.6 AP)
- 斩杀阈值: 40% 生命值

**隐藏机制**:
- W位移约400距离
- W可以重置平A
- W治疗效果在低血量时最大化
- W斩杀线约40%生命值

### E - 劫持/绑架 (Abscond / Abduct)

- **冷却**: 13/12/11/10/9 秒
- **消耗**: 40 法力
- **范围**: 400

第一段：塞拉斯快速冲刺，准备3.5秒内可以重施。
第二段：投掷锁链，将自己拉向第一个命中的敌人，造成伤害并击飞。

**效果数值**:
- 位移距离: 400
- 伤害: 60/85/110/135/160 (+0.4 AP)
- 击飞时间: 0.6/0.7/0.8/0.9/1.0 秒

**隐藏机制**:
- E可以穿墙
- E第一段是位移，第二段是拉人
- E可以用来取消其他技能后摇
- E两段之间有3.5秒窗口

### R - 盗取 (Hijack)

- **冷却**: 80/55/30 秒
- **消耗**: 50 法力
- **范围**: 950

塞拉斯盗取敌方英雄的大招，可以使用一次复制的大招。
- 使用目标的终极技能（基于塞拉斯自己的大招等级）
- 使用塞拉斯的属性（AD、AP、穿甲等）
- 对目标施加冷却

**效果数值**:
- 冷却缩减: 目标大招冷却的 70/55/40% (基于塞拉斯R等级)
- 最小冷却: 10/8/6 秒

**可以盗取的大招类型**:
- 伤害型大招：造成塞拉斯AP加成的伤害
- 增益型大招：获得相同增益效果
- 位移型大招：复制位移效果
- 召唤物型大招：召唤相应单位

**不能盗取的大招**:
- 妮蔻的大招（变身后的大招无法复制）
- 赛娜的大招（机制特殊）

**隐藏机制**:
- 塞拉斯的大招享受自己属性的加成（AP、穿甲等）
- 塞拉斯可以偷取任何敌方英雄的大招
- 偷取后目标有大招冷却（最低10秒）
- 塞拉斯的大招没有加成系数给偷取的大招

## 连招技巧

### 基础连招

**对线换血**:
- Q → W → 平A → E撤退
- 利用W位移接近

**E连招**:
- E第一段 → E第二段 → W → 平A
- 两段E追击

**QW连招**:
- Q命中 → 立即W
- 减速后W更容易命中

### 进阶连招

**E+W+E Combo**:
- E第一段位移 → W → E第二段拉回
- 利用W取消E后摇

**QEW连招**:
- Q → E → W
- Q减速后W更容易追上

**E闪连招**:
- E第一段 → F → E第二段
- 超远距离开团

### 大招使用

**偷取大招后连招**:
- 偷取 → R → Q/W/E → 平A
- 根据偷取的大招类型决定连招

**团战大招使用**:
- 团战中偷取敌方关键大招
- 利用偷取的大招收割或保命

## 隐藏机制

1. **被动充能机制**: 每次技能后获得一个充能，最多2层
2. **被动攻速**: 拥有充能时+30%攻速
3. **Q两段伤害**: Q的两段伤害可以叠加
4. **W位移**: W约400位移距离
5. **W斩杀治疗**: 低于40%生命值时W治疗最大化
6. **E两段机制**: 第一段位移，第二段拉人
7. **E穿墙**: E两段都可以穿越地形
8. **R无加成**: 偷取的大招没有额外加成
9. **E可以取消后摇**: E可以取消Q、W的后摇
10. **被动伤害触发**: 被动伤害可以触发法术特效（回声等）

## 常见问题

**Q: 塞拉斯应该偷谁的大招？**
A: 根据局势选择。团战优先偷取高伤害AOE（厄运小姐R、赛恩R），保命优先偷取位移技能。

**Q: 塞拉斯对线怎么换血？**
A: 利用Q poke减速，然后E接近打W撤退。被动的充能也可以消耗敌人。

**Q: 塞拉斯的被动怎么用？**
A: 技能后获得充能，充能时攻速加成。尽量在有充能时与敌人对拼。

**Q: 塞拉斯打团怎么定位？**
A: 作为刺客型法师，团战目标是切入后排或利用偷取的大招收割。

**Q: 哪些大招不能偷？**
A: 妮蔻的大招（变身的那个）无法复制。艾希R可以偷但效果可能不佳。
