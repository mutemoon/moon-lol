# Tristana (崔丝塔娜)

## 基础信息

- **称号**: 崔丝塔娜 (The Yordle Gunner)
- **类型**: 物理 / 远程
- **难度**: 2/3
- **角色**: 射手
- **位置**: ADC

## 技能介绍

### 被动 - 瞄准 (Draw a Bead)

**机制**: Tristana随等级获得攻击距离。

**额外射程**: 0-167.65 (基于等级)
**总射程**: 550-717.65

### Q - 快速射击 (Rapid Fire)

- **冷却**: 20/19/18/17/16 秒
- **消耗**: 30/35/40/45/50 法力
- **范围**: 自助

Tristana强化自身攻击速度。

**攻速加成**: 60/75/90/105/120%
**持续时间**: 7秒

### W - 火箭跳跃 (Rocket Jump)

- **冷却**: 22/20/18/16/14 秒
- **消耗**: 30/35/40/45/50 法力
- **范围**: 900

Tristana跳跃至目标位置。

**伤害**: 70/105/140/175/210 (+ 100% 额外AD) (+ 50% AP)
**减速**: 40% (持续2秒)
**击杀重置**: 参与击杀刷新冷却

### E - 爆炸射击 (Explosive Charge)

- **冷却**: 16/15.5/15/14.5/14 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 550-700 (基于等级)

Tristana投掷炸弹，叠层后爆炸。

**基础伤害**: 60/85/110/135/160 (+ 80% 额外AD) (+ 50% AP)
**叠加伤害**: 每层+25% (最大100%)
**塔伤害**: 爆炸范围扩大

### R - 毁灭射击 (Buster Shot)

- **冷却**: 100 秒
- **消耗**: 100 法力
- **范围**: 550-700 (基于等级)

Tristana发射巨型炮弹。

**伤害**: 225/275/325 (+ 70% 额外AD) (+ 100% AP)
**击退距离**: 600/800/1000
**眩晕时间**: 0.4/0.55/0.7秒

## 连招技巧

### 基础连招

**对线连招**:
- E(爆炸射击) → Q(快速射击) → W(火箭跳跃)
- 叠层后跳脸

**击杀连招**:
- W(火箭跳跃) → E(爆炸射击) → R(毁灭射击)
- 位移击杀

### 进阶连招

**团战连招**:
- Q(快速射击) → E(爆炸射击) → W(火箭跳跃) → R(毁灭射击)
- 爆发输出

**逃脱连招**:
- R(毁灭射击) → W(火箭跳跃)
- 击退后跳跃

## 隐藏机制

1. **被动射程**: 随等级提升射程
2. **E塔伤害**: E对塔有额外范围
3. **W重置**: 参与击杀刷新W冷却
4. **R击退**: R可以击退敌人
5. **E叠加**: E最多叠4层
