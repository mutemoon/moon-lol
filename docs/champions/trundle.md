# Trundle (特朗德尔)

## 基础信息

- **称号**: 特朗德尔 (The Troll King)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、坦克
- **位置**: 上单、打野

## 技能介绍

### 被动 - 国王的贡献 (King's Tribute)

**机制**: 附近敌人死亡时Trundle获得治疗。

**治疗量**: 1.8-5.94% (基于等级) 目标最大生命值
**范围**: 1400

### Q - 咬 (Chomp)

- **冷却**: 3.5 秒
- **消耗**: 20 法力
- **范围**: 175

Trundle强化下一次普攻。

**伤害**: 10/30/50/70/90 (+ 15-55% AD)
**减速**: 75% (持续0.1秒)
**获得AD**: 20/25/30/35/40
**偷取AD**: 10/12.5/15/17.5/20

### W - 冰霜领域 (Frozen Domain)

- **冷却**: 18/17/16/15/14 秒
- **消耗**: 40 法力
- **范围**: 750

Trundle在目标位置创造冰霜领域。

**攻速加成**: 30/45/60/75/90%
**移速加成**: 20/28/36/44/52%
**治疗加成**: 25%
**持续时间**: 8秒

### E - 冰柱 (Pillar of Ice)

- **冷却**: 21/19.5/18/16.5/15 秒
- **消耗**: 75 法力
- **范围**: 1000

Trundle创造冰柱，击退并减速敌人。

**减速**: 34/38/42/46/50%
**持续时间**: 6秒
**击退距离**: 225

### R - 压榨 (Subjugate)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 650

Trundle吸取目标生命值和抗性。

**伤害**: 20/25/30% (+ 2% 每100AP) 目标最大生命值
**治疗**: 等量伤害值
**偷取抗性**: 40% (护甲和魔抗)
**持续时间**: 8秒

## 连招技巧

### 基础连招

**对线连招**:
- E(冰柱) → Q(咬) → W(冰霜领域)
- 冰柱控制

**追击连招**:
- W(冰霜领域) → Q(咬) → E(冰柱)
- 加速追击

### 进阶连招

**团战连招**:
- R(压榨) → E(冰柱) → Q(咬) → W(冰霜领域)
- 吸取坦克

**单杀连招**:
- E(冰柱) → Q(咬) → R(压榨) → W(冰霜领域)
- 持续输出

## 隐藏机制

1. **被动治疗**: 附近敌人死亡治疗
2. **Q偷取**: Q偷取敌人AD
3. **W领域**: W内Trundle获得多重加成
4. **E地形**: E创造地形障碍
5. **R吸取**: R吸取抗性和生命值
