# Taliyah (塔莉垭)

## 基础信息

- **称号**: 塔莉垭 (The Stoneweaver)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师
- **位置**: 打野、中单

## 技能介绍

### 被动 - 岩石滑行 (Rock Surfing)

**机制**: Taliyah在地形附近获得移动速度加成。

**移速加成**: 10-40% (基于等级)
**持续时间**: 离开地形后3秒衰减

### Q - .threaded volley (Threaded Volley)

- **冷却**: 7/6/5/4/3 秒
- **消耗**: 55/60/65/70/75 法力
- **范围**: 1000

Taliyah投掷5个石碎片。

**伤害**: 50/67.5/85/102.5/120 (+ 50% AP)
**强化效果**: 在Worked Ground上施放变为 Boulder
** Boulder伤害**: 90/121.5/153/184.5/216 (+ 90% AP)
** Boulder减速**: 20/25/30/35/40%

### W - 地震推 (Seismic Shove)

- **冷却**: 14/12.5/11/9.5/8 秒
- **消耗**: 40/30/20/10/0 法力
- **范围**: 900

Taliyah创造地震，击退敌人。

**击退距离**: 400单位
**延迟**: 0.792秒

### E - 破碎大地 (Unraveled Earth)

- **冷却**: 14 秒
- **消耗**: 90 法力
- **范围**: 800

Taliyah创造岩石地带，伤害并减速敌人。

**伤害**: 60/105/150/195/240 (+ 60% AP)
**减速**: 20%
**眩晕时间**: 0.75-2秒 (位移触碰岩石时)
**最大石头数**: 22个

### R - 织墙者 (Weaver's Wall)

- **冷却**: 180/150/120 秒
- **消耗**: 100 法力
- **范围**: 2500/4500/6500

Taliyah创造巨大的岩石墙。

**墙宽度**: 700/1000
**持续时间**: 4秒
**二段**: 可以骑墙跳跃

## 连招技巧

### 基础连招

**打野连招**:
- Q(Threaded Volley) → E(破碎大地) → W(地震推)
- 岩石控制

**中单连招**:
- Q(Threaded Volley) → E(破碎大地) → R(织墙者)
- 快速清线

### 进阶连招

**团战连招**:
- R(织墙者) → Q(Threaded Volley) → E(破碎大地) → W(地震推)
- 支援控制

**逃跑连招**:
- R(织墙者) → 骑墙跳跃
- 快速撤离

---

## 隐藏机制

1. **被动加速**: 在地形附近获得移速
2. **Q强化**: 在Worked Ground上Q变为 Boulder
3. **E控制**: 位移触碰岩石可以眩晕
4. **R位移**: R可以穿墙
5. **E+R配合**: R可以创造地形配合E
