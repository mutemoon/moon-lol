# Twitch (图奇)

## 基础信息

- **称号**: 图奇 (The Plague Rat)
- **类型**: 物理 / 远程
- **难度**: 2/3
- **角色**: 射手、刺客
- **位置**: ADC

## 技能介绍

### 被动 - 致命毒液 (Deadly Venom)

**机制**: Twitch的普攻施加毒液。

**每层伤害**: 1/2/3/4/5 (+ 3% AP) 每秒
**最大层数**: 6层
**持续时间**: 6秒

### Q - 埋伏 (Ambush)

- **冷却**: 16 秒
- **消耗**: 40 法力
- **范围**: 500

Twitch进入隐身。

**隐身延迟**: 1秒
**隐身时间**: 10/11/12/13/14秒
**移速加成**: 10% (+ 面对敌人30%)
**出草攻速**: 40/45/50/55/60%

### W - 毒瓶 (Venom Cask)

- **冷却**: 13/12/11/10/9 秒
- **消耗**: 70 法力
- **范围**: 950

Twitch投掷毒瓶。

**减速**: 30/35/40/45/50%
**持续时间**: 3秒
**毒液叠加**: 每秒1层

### E - 污染 (Contaminate)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 1200

Twitch引爆毒液。

**基础伤害**: 20/30/40/50/60
**每层伤害**: 15/20/25/30/35 (+ 35% 额外AD)

### R - 超级死亡 (Spray and Pray)

- **冷却**: 90 秒
- **消耗**: 100 法力
- **范围**: 1100

Twitch获得额外射程和伤害。

**额外AD**: 30/45/60
**额外射程**: 300
**持续时间**: 6秒
**穿透效果**: 攻击穿透多个敌人

## 连招技巧

### 基础连招

**对线连招**:
- Q(埋伏) → W(毒瓶) → E(污染)
- 隐身消耗

**击杀连招**:
- Q(埋伏) → R(超级死亡) → E(污染)
- 隐身爆发

### 进阶连招

**团战连招**:
- Q(埋伏) → W(毒瓶) → R(超级死亡) → E(污染)
- 隐身穿透

**逃跑连招**:
- R(超级死亡) → Q(埋伏)
- 加速隐身

---

## 隐藏机制

1. **被动叠加**: 毒液最多6层
2. **Q隐身**: 隐身可以先手
3. **W扩散**: W可以扩散毒液
4. **E引爆**: E引爆所有毒液层数
5. **R穿透**: 大招攻击穿透多个敌人
