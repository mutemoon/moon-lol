# 瑟提 (Sett)

## 基础信息

- **称号**: 荒蔓之拳 (the Boss)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、坦克
- **位置**: 上单

## 技能介绍

### 被动 - 沙场战心 (Pit Dukker)

瑟提将受到的所有物理伤害储存为"斗心"。
- 斗心最高可储存50%最大生命值的伤害
- 每秒钟斗心会衰减一定比例
- 瑟提的普通攻击会用斗心额外造成物理伤害

**斗心机制**:
- 被动储存受到伤害的80%（物理伤害）
- 斗心随时间衰减，脱战后衰减更快
- 斗心提供的额外伤害基于储存值

### Q - 沙场破敌 (Knuckle Down)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 0
- **范围**: 0

瑟提的下两次攻击造成基于目标最大生命值的额外伤害。瑟提在朝向敌方英雄移动时获得移动速度。
- 两次额外攻击各持续4秒
- 额外伤害基于目标最大生命值

**效果数值**:
- 最大生命值伤害: 3/3.5/4/4.5/5%
- 移速加成: 30/35/40/45/50%

**隐藏机制**:
- Q的重置效果可以取消平A后摇
- Q提供的移速加成让他追人更强
- Q的额外伤害享受穿甲加成
- 优先触发有破绽的敌人

### W - 强腕重击 (Haymaker)

- **冷却**: 18/16.5/15/13.5/12 秒
- **消耗**: 0
- **范围**: 25000

瑟提被动地将受到的伤害储存为"灰心"。施放时，瑟提消耗所有储存的灰心获得护盾，然后重拳攻击一个区域，在中心造成真实伤害，两侧造成物理伤害。
- 中心区域伤害更高
- 灰心转化为护盾
- 护盾持续0.75秒

**效果数值**:
- 灰心->护盾转化率: 50%（每100灰心=50护盾）
- 中心真伤: 60/80/100/120/140 + 灰心*35%
- 两侧物伤: 40/55/70/85/100 + 灰心*25%

**隐藏机制**:
- W的蓄力时间约0.25秒
- W期间可以取消，但不能闪避
- 灰心在脱战后衰减很快
- W中心比边缘伤害更高
- 护盾可以吸收技能和普通攻击

### E - 断崖肩袭 (Facebreaker)

- **冷却**: 16/14.5/13/11.5/10 秒
- **消耗**: 0
- **范围**: 490

瑟提将身侧所有敌人拉向自己，造成伤害并眩晕他们。如果敌人只在某一侧，则只造成减速而非眩晕。
- 需要两侧都有敌人才会眩晕
- 单侧敌人会被减速50%，持续0.5秒

**效果数值**:
- 伤害: 60/80/100/120/140 (+0.8 AD)
- 眩晕时长: 0.5/0.75 秒
- 减速: 50%

**隐藏机制**:
- E的判定范围是以瑟提为圆心的左右两侧
- E可以拉取小兵和英雄
- E可以被闪现打断
- E配合W可以取消后摇
- E拉取多个敌人时伤害会分散

### R - 豪腕重摔 (The Show Stopper)

- **冷却**: 120/100/80 秒
- **消耗**: 0
- **范围**: 400

瑟提抱起一个敌方英雄，向空中跃起，将其砸向地面，对落地点附近所有敌人造成伤害并减速。
- 抱起的敌方英雄会一起落地
- 落地范围造成AOE伤害
- 抱起期间该目标无法移动

**效果数值**:
- 伤害: 200/300/400 (+1.5 AD)
- 减速: 40/50/60%，持续1.5秒

**隐藏机制**:
- R可以抱走正在攻击瑟提的敌人
- R可以用来保护队友
- R落地后可以接W和E
- R的方向由鼠标位置决定
- 被抱的敌人在落地前无法被其他技能影响
- R可以配合E使用

## 连招技巧

### 基础连招

**对线消耗**:
- Q起手 → W蓄力 → E拉回 → 平A
- 利用Q的移速加成追人

**击杀连招**:
- E拉近 → 平A → Q → W → 平A
- 确保灰心足够再开W

**消耗+反击**:
- 承受伤害积攒灰心 → E拉回 → W打出护盾和伤害

### 进阶连招

**E+W取消后摇**:
- E拉回 → 立即W → 取消E后摇

**R+E+W**:
- R抱起敌人 → E拉取更多敌人 → W打出范围伤害

**闪现+R**:
- 闪现到敌人身边 → R抱起 → 调整落地点

**WQ+AE**:
- W蓄力 → Q → 冲向敌人 → E拉取 → 打出W高伤害

### 团战进场

**R切入后排**:
- R抱起敌方后排 → 砸向己方阵容
- 利用R的AOE减速分割战场

**E开团**:
- E拉取多个敌人 → W范围伤害 → R抱走威胁目标

**反手连招**:
- 等待敌人先手 → E拉取 → W反打

## 隐藏机制

1. **Q加速机制**: Q提供的移速加成只在朝向敌方英雄时生效
2. **灰心储存**: 物理伤害储存80%，魔法伤害不储存
3. **W护盾生效**: 灰心转化护盾的比例是50%
4. **E两侧判定**: E必须在身体两侧都有敌人才会眩晕
5. **R抱取优先级**: R会抱取鼠标悬停的目标，没有悬停则抱取最近敌人
6. **W中心真伤**: 中心伤害享受灰心加成，两侧不享受
7. **Q重置平A**: Q可以取消平A后摇
8. **灰心衰减**: 脱战后灰心每秒衰减10%
9. **E可以被控制打断**: E的前摇可以被眩晕等控制技能打断
10. **R砸地范围**: R落地的AOE范围比视觉效果略大

## 常见问题

**Q: 瑟提如何快速积攒灰心？**
A: 主动承受伤害可以积攒灰心，但对线时要注意不被消耗太多血量。

**Q: 瑟提的E如何使用最有效？**
A: E需要两侧都有敌人才会眩晕，单侧只有减速。

**Q: 瑟提的核心连招是什么？**
A: E+W取消后摇是核心技巧，可以最大化伤害输出。
