#  Seraphine (萨勒芬妮)

## 基础信息

- **称号**: 萨勒芬妮 (The Starry-Eyed Songstress)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 辅助、法师
- **位置**: ADC、辅助

## 技能介绍

### 被动 - 舞台存在感 (Stage Presence)

**机制**: Seraphine的技能生成Echo，强化后可以再次释放。

**Echo层数**: 2层
**音符效果**: 普攻发射音符造成伤害

### Q - 高音 (High Note)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 900

Seraphine发射音符，造成范围伤害。

**伤害**: 60/85/110/135/160 (+ 50% AP)
**增伤**: 基于目标损失生命值，最高75%

### W - 环绕音效 (Surround Sound)

- **冷却**: 22 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 800

Seraphine为自身和友军提供护盾和移速。

**护盾值**: 60/80/100/120/140 (+ 20% AP)
**自加速**: 20% (+ 2% 每100 AP)
**友军加速**: 8% (+ 0.8% 每100 AP)
**治疗效果**: 护盾结束时治疗周围友军

### E - 节拍下降 (Beat Drop)

- **冷却**: 11/10.5/10/9.5/9 秒
- **消耗**: 60 法力
- **范围**: 1300

Seraphine发射音波，伤害并减速敌人。

**伤害**: 70/100/130/160/190 (+ 50% AP)
**减速/禁锢**: 99% (对已减速敌人变成禁锢)

### R - 再来 (Encore)

- **冷却**: 160/140/120 秒
- **消耗**: 100 法力
- **范围**: 1300

Seraphine发射魅惑冲击波，伤害并魅惑敌人。

**伤害**: 150/200/250 (+ 40% AP)
**魅惑时间**: 1.25/1.5/1.75秒
**减速**: 40% (逐渐增加至99%)
**友军增益**: 被命中友军获得4层音符

## 连招技巧

### 基础连招

**消耗连招**:
- Q(高音) → E(节拍下降) → W(环绕音效)
- 伤害后护盾

**控制连招**:
- E(节拍下降) → Q(高音) → W(环绕音效)
- 减速后持续输出

### 进阶连招

**团战连招**:
- R(再来) → Q(高音) → E(节拍下降) → W(环绕音效)
- 大招起手群体控制

**反手连招**:
- W(环绕音效) → E(节拍下降) → Q(高音)
- 护盾后反打

---

## 隐藏机制

1. **Echo系统**: 技能可以生成Echo，满层时强化
2. **音符效果**: 被动让普攻发射音符
3. **W治疗**: W护盾结束后可以治疗
4. **E双重效果**: 对已减速敌人变成禁锢
5. **R群体增益**: 大招给友军提供音符
