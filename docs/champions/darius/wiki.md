# 诺手 (Darius)

## 基础信息

- **称号**: 诺克萨斯之手 (the Hand of Noxus)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、坦克
- **位置**: 上单、打野

## 技能介绍

### 被动 - 血怒 (Hemorrhage)

德莱文斯和敌人的攻击和伤害性技能会导致敌人流血，持续5秒，最多叠加5层。
- 每层造成12物理伤害
- 叠满5层时，德莱文斯进入狂热状态，获得巨额攻击力（+50% 总AD）
- 5层血怒是诺手的核心输出/斩杀条件

### Q - 大杀四方 (Decimate)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 1（自身周围）

德莱文斯挥动战斧，对边缘敌人造成高伤害，对内圈敌人造成较低伤害。
- 内圈（把手）伤害较低，且不会叠加血怒
- 外圈（斧刃）伤害高，叠加血怒
- 边缘命中的每个英雄恢复一定百分比已损失生命值

**效果数值**:
- 斧刃伤害: 90/100/110/120/130/140/150
- 把手伤害: 45/50/55/60/65/70/75
- 生命回复: 17% 已损失生命值/英雄（最多51%）
- AD加成: 0.9

**隐藏机制**:
- Q的内圈范围非常小（约150范围）
- Q的斧刃范围约350-400
- Q可以被打断（被控制时使用Q会被取消）
- Q对小兵和野怪也有类似的内外圈判定
- Q命中多个英雄时回复效果可叠加

### W - 致残 (Crippling Strike)

- **冷却**: 5 秒（所有等级）
- **消耗**: 30 法力
- **范围**: 300

德莱文斯的下次攻击造成额外物理伤害，并减速目标50%，持续1秒。
- 如果击杀目标，返还法力消耗，冷却减少50%
- 这个技能在造成伤害时触发技能效果

**效果数值**:
- 额外伤害: 160/170/180/190/200% 总AD

**隐藏机制**:
- W可以重置平A
- W的减速可以被韧性减少
- W击杀后返蓝+减CD是高机动性的关键
- W的攻击范围比普通平A略远（约300）

### E - 诺克萨斯断头台/无情立场 (Apprehend)

- **冷却**: 26/23.5/21/18.5/16 秒
- **消耗**: 45 法力
- **范围**: 535

被动：德莱文斯获得护甲穿透。
主动：德莱文斯用战斧拉回敌人，击飞并减速40%，持续1秒。

**效果数值**:
- 护甲穿透: 20/25/30/35/40%
- 拉回距离: 535
- 减速: 40%，持续1秒

**隐藏机制**:
- E可以打断引导技能
- E拉回优先级：英雄 > 小兵 > 野怪
- E的拉回效果可以被韧性减少（击飞时间固定0.75秒，不受韧性影响的是减速部分）
- E可以配合Q使用：E拉回 → Q外圈命中
- E在6级前是对线主要控制技能

### R - 诺克萨斯断头台 (Noxian Guillotine)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力（3级时无消耗）
- **范围**: 460

德莱文斯跃向目标，造成真实伤害。每层血怒提升额外伤害。
- 如果击杀目标，可以在6秒内再次施放
- 3级R无蓝耗，且击杀后完全刷新冷却
- 可以打断引导技能

**效果数值**:
- 基础伤害: 100/200/300
- 每层血怒: +30% 伤害
- 最大伤害（5层）: 100/200/300 + 450% = 550/700/800
- AD加成: 1.2

**隐藏机制**:
- R斩杀后重置是立即的，可以在6秒内随时使用
- R的斩杀距离约460，需要贴身
- R的跃起有约0.3秒不可选中
- 血怒层数在R落地瞬间计算
- 即使目标无敌/不可选中，R仍然可以斩杀（例如：薇恩Q期间）
- R享受全额AD加成和血怒加成

## 连招技巧

### 基础连招

**标准对线换血**:
- AWA（平A + W致残 + 平A）→ Q外圈 → 拉开
- 利用W重置平A打快速伤害

**五层血怒连招**:
- E拉回 → 平A → W → Q外圈 → 平A → R
- 目标死亡后R重置追杀

**满层斩杀连招**:
- 平A × 4（叠满5层血怒）→ R（最大化斩杀伤害）
- 或者：E → 平A → W → Q → 平A → 平A → R

### 进阶连招

**E+W+AA+Q Combo**:
- E拉回 → 立即W → 平A → Q外圈
- 确保Q的外圈命中

**Q闪（Q Flash）**:
- Q起手 → 立即F调整位置
- 用于让Q的外圈命中更多目标
- Q的引导约0.25秒，可以在此时闪现

**RW Combo**:
- R起手 → 立即W
- R的不可选中可以用来躲技能

**F+E Combo**:
- 闪现过墙 → E拉回
- 超远距离开团

### 团战连招

**团战进场**:
- E → 平A → W → Q（外圈命中多个） → 平A → R
- 等待血怒叠满后用R斩杀

**追杀连招**:
- E拉回 → 追击 → Q → 平A → W → R
- 目标逃跑时用R追杀

## 隐藏机制

1. **Q外圈判定**: 斧刃范围约350-400，内圈只有约150
2. **Q内圈不叠血怒**: 只有外圈命中才叠加被动
3. **W重置平A**: W可以取消平A后摇
4. **E可以打断**: E可以打断厄运小姐、卡莎等引导大招
5. **血怒狂热AD**: 叠满5层获得+50% 总AD，持续5秒
6. **R斩杀刷新**: 3级R击杀后冷却完全刷新
7. **R不可选中**: R跃起时有约0.3秒不可选中
8. **E拉回距离**: 535，实际生效范围略大
9. **Q回血机制**: 边缘命中的每个英雄回复17%已损生命
10. **W击杀返蓝**: W击杀小兵/英雄返还蓝耗和50%冷却
11. **E护甲穿透**: 被动护甲穿透对R也生效
12. **血怒持续时间**: 每层持续5秒，可叠加
13. **R起跳无敌**: 可以用来躲一些技能（如艾希R）

## 常见问题

**Q: 如何快速叠满5层血怒？**
A: 使用普攻叠层是最稳定的。对线期可以用AWQA连招快速叠加，团战时尽量先用E拉住多个目标再Q。

**Q: Q外圈怎么提高命中率？**
A: 利用草丛视野优势，或者E拉回后立即Q。敌人被墙边时可以预判走位。

**Q: 诺手怎么打团？**
A: 团战目标是叠满血怒后用R斩杀敌方脆皮。可以先手E开团，或者等队友控制后再进场。

**Q: R斩杀的最高伤害是多少？**
A: 满级R（300基础）+ 5层血怒（450%加成）+ 1.2 AD加成。假设300 AD，理论最高约1000+伤害。
