#  Rakan (洛)

## 基础信息

- **称号**: 洛 (The Charmer)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 辅助
- **位置**: 辅助

## 技能介绍

### 被动 - 羽毛之舞 (Fey Feathers)

**机制**: Rakan周期性获得护盾，脱战时恢复。

**护盾值**: 30-247 (+ 95% AP)
**冷却**: 40-12.75秒

### Q - 闪亮之羽 (Gleaming Quill)

- **冷却**: 11/10/9/8/7 秒
- **消耗**: 45 法力
- **范围**: 900

Rakan投掷羽毛，命中后标记区域。

**伤害**: 70/115/160/205/250 (+ 70% AP)
**治疗范围**: 200
**治疗量**: 40-230 (+ 55% AP)

### W - 华丽登场 (Grand Entrance)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 650

Rakan冲向目标，延迟后击飞敌人。

**伤害**: 70/120/170/220/270 (+ 80% AP)
**击飞时间**: 1秒

### E - 战斗舞步 (Battle Dance)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 700/1000

Rakan为友军提供护盾，然后冲向他们。

**护盾值**: 50/75/100/125/150 (+ 70% AP)
**可再次释放**: 5秒内

### R - 速度之舞 (The Quickness)

- **冷却**: 130/110/90 秒
- **消耗**: 100 法力
- **范围**: 150

Rakan获得巨额移速，碰撞敌人时魅惑并减速。

**伤害**: 100/200/300 (+ 50% AP)
**魅惑时间**: 1/1.25/1.5秒
**减速**: 75%

## 连招技巧

### 基础连招

**保护连招**:
- E(战斗舞步) → W(华丽登场)
- 护盾后控制

**进攻连招**:
- R(速度之舞) → W(华丽登场) → Q(闪亮之羽)
- 大招加速控制

### 进阶连招

**团战连招**:
- R(速度之舞) → W(华丽登场) → E(战斗舞步) → Q(闪亮之羽)
- 大招起手团控

**反手连招**:
- W(华丽登场) → E(战斗舞步) → Q(闪亮之羽)
- 快速反打

## 隐藏机制

1. **被动护盾**: 周期性获得护盾
2. **W延迟**: W有延迟后生效
3. **E双跳**: 可以给友军护盾后跳向另一个友军
4. **R魅惑**: 大招可以魅惑多个敌人
5. **Xayah配合**: 与Xayah有特殊联动
