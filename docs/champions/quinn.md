#  Quinn (奎因)

## 基础信息

- **称号**: 奎因 (Demacia's Wings)
- **类型**: 物理 / 远程
- **难度**: 2/3
- **角色**: 射手、刺客
- **位置**: 上单、打野

## 技能介绍

### 被动 - 猎物标记 (Harrier)

**机制**: Quinn和Valor周期性标记敌人，造成额外伤害。

**标记伤害**: 15-132 (+ 40% 额外AD)
**标记持续**: 4秒
**冷却**: 基于暴击率

### Q - 致盲突击 (Blinding Assault)

- **冷却**: 11/10.5/10/9.5/9 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 1050

Quinn派出Valor攻击，致盲并伤害敌人。

**伤害**: 65/100/135/170/205 (+ 80/85/90/95/100% 额外AD) (+ 50% AP)
**致盲时间**: 1.75秒

### W - 高度感知 (Heightened Senses)

- **冷却**: 50/45/40/35/30 秒
- **消耗**: 自助
- **范围**: 2100

Valor提供周围视野，并获得攻速和移速加成。

**攻速加成**: 28/41/54/67/80%
**移速加成**: 20/25/30/35/40%
**持续时间**: 2秒

### E -  vaults (Vault)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 50 法力
- **范围**: 600

Quinn跳向敌人，将他们击退并减速。

**伤害**: 40/65/90/115/140 (+ 20% 额外AD)
**击退距离**: 短距离
**减速**: 50%

### R - 深入敌后 (Behind Enemy Lines)

- **冷却**: 3 秒
- **消耗**: 100/50/0 法力
- **范围**: 700

Quinn和Valor合体，大幅提升移速。

**移速加成**: 70/100/130%
**持续时间**: 2秒
**技能**: 可释放Skystrike

### R - 天袭 (Skystrike)

- **冷却**: 3 秒
- **消耗**: 自助
- **范围**: 700

Valor从空中攻击，伤害并标记敌人。

**伤害**: 60/90/120 (+ 35% 额外AD)

## 连招技巧

### 基础连招

**对线连招**:
- Q(致盲突击) → E( vaults) → W(高度感知)
- 致盲后追击

**追杀连招**:
- R(深入敌后) → E( vaults) → Q(致盲突击)
- 大招加速追击

### 进阶连招

**团战连招**:
- R(深入敌后) → E( vaults) → Q(致盲突击) → W(高度感知)
- 大招切入后控制

**单杀连招**:
- Q(致盲突击) → E( vaults) → 普攻 → R(天袭)
- 致盲后持续输出

## 隐藏机制

1. **Valor协作**: Valor协助攻击和提供视野
2. **被动标记**: 周期性标记敌人
3. **E击退**: E可以击退敌人
4. **R移速**: 大招提供高额移速
5. **W攻速**: W提供大量攻速加成
