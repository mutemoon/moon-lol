#  Akshan (正义仲裁者)

## 基础信息

- **称号**: 正义仲裁者 (The Rogue Sentinel)
- **类型**: 物理 / 远程
- **难度**: 3/3
- **角色**: 射手、刺客
- **位置**: 中单

## 技能介绍

### 被动 - 不法之战 (Dirty Fighting)

**机制**:  Akshan的普攻和技能对敌人施加"不法"效果，最多3层。第三层触发时造成额外伤害并获得护盾。

**额外伤害**: 15/40/80/150 (基于等级) (+ 60% AP) 魔法伤害
**护盾值**: 40-317 (基于等级) (+ 35% bonus AD)

### Q - 回旋飞轮 (Avengerang)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 850 + 每个敌人命中后扩展500

Akshan向目标方向投掷回旋飞轮，对敌人造成物理伤害并暴露他们。飞轮可以命中多个敌人并扩展范围。

**伤害**: 45/75/105/135/165 (+ 70% bonus AD)
**总伤害**: 90/150/210/270/330 (+ 140% bonus AD)

### W - 行侠仗义 (Going Rogue)

- **冷却**: 18/14/10/6/2 秒
- **消耗**: 40/30/20/10/0 法力

Akshan标记击杀队友的敌人为"恶人"。击杀恶人可获得额外金币并复活队友。

**机制**:
- 标记击杀队友的敌人为"恶人"
- 击杀恶人获得100金币
- 复活被恶人击杀的友军
- 进入伪装状态获得移速加成

### E - 英雄登场 (Heroic Swing)

- **冷却**: 18/16.5/15/13.5/12 秒
- **消耗**: 70 法力
- **范围**: 800

Akshan发射抓钩嵌入地形，然后荡向敌人。最多可使用3次。

**伤害**: 8/16/24/32/40 (+ 25% AD) 每发

### R - 惩恶扬善 (Comeuppance)

- **冷却**: 100/85/70 秒
- **消耗**: 100 法力
- **范围**: 2500

Akshan锁定目标并开始充能子弹，然后发射所有子弹。

**每发伤害**: 25/35/45 (+ 15% AD)
**子弹数量**: 5/6/7 发

## 连招技巧

### 基础连招

**标准消耗**:
- Q(回旋飞轮) → E(英雄登场) → 普攻
- Q起手减速，E接近目标

**击杀连招**:
- W(行侠仗义) → E(英雄登场) → Q(回旋飞轮) → R(惩恶扬善)
- W伪装接近，E突进，R收割

### 进阶连招

**团战连招**:
- W(行侠仗义) → E(英雄登场) × 3 → R(惩恶扬善)
- 伪装接近战场，利用E位移穿梭

**追杀连招**:
- E(英雄登场) → Q(回旋飞轮) → 普攻 × 2
- E快速接近，Q消耗

## 隐藏机制

1. **不法之战护盾**: 第三层不法触发时获得护盾，可用于保命
2. **W标记**: W可以标记击杀队友的敌人，利用它获得金币和复活队友
3. **E第三段**: E第三段可以穿越地形
4. **R充能**: R的子弹会基于目标生命值损失造成额外伤害
5. **Q扩展**: Q可以随着命中敌人不断扩展范围
