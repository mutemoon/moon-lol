#  Annie (安妮)

## 基础信息

- **称号**: 安妮 (The Dark Child)
- **类型**: 魔法 / 远程
- **难度**: 1/3
- **角色**: 法师、爆发
- **位置**: 中单、辅助

## 技能介绍

### 被动 - 嗜火 (Pyromania)

**机制**: Annie的技能命中敌人时获得嗜火层数。满4层时下次伤害技能可以眩晕敌人。

**眩晕时间**: 1.25/1.5/1.75 (基于等级) 秒
**触发条件**: 使用Disintegrate、Incinerate或Summon: Tibbers时

### Q - 碎裂之火 (Disintegrate)

- **冷却**: 4 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 625

Annie向目标投掷火球，造成魔法伤害。如果击杀目标，返还50%冷却和法力消耗。

**伤害**: 80/125/170/215/260 (+ 80% AP)

### W - 焚烧 (Incinerate)

- **冷却**: 7 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 600

Annie释放锥形火焰，对范围内敌人造成魔法伤害。

**伤害**: 70/110/150/190/230 (+ 80% AP)

### E - 熔岩护盾 (Molten Shield)

- **冷却**: 10 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 800

Annie为自己或友军提供护盾，持续3秒。护盾存在期间，攻击护盾的敌人会受到魔法伤害。

**护盾值**: 60/95/130/165/200 (+ 40% AP)
**伤害反弹**: 25/35/45/55/65 (+ 40% AP)

### R - 提伯斯 (Summon: Tibbers)

- **冷却**: 130/115/100 秒
- **消耗**: 100 法力
- **范围**: 600

Annie召唤提伯斯，对范围内敌人造成魔法伤害。提伯斯可以存在最多45秒。

**初始伤害**: 150/275/400 (+ 75% AP)
**提伯斯伤害**: 30/45/60 (+ 10% AP)
**被动**: Annie获得10/15/20%魔法穿透

## 连招技巧

### 基础连招

**标准爆发**:
- 被动满层 → W(焚烧) → Q(碎裂之火) → R(提伯斯)
- 被动满层时先手控制

**消耗连招**:
- Q(碎裂之火) → W(焚烧)
- Q击杀小兵时冷却刷新

### 进阶连招

**团战连招**:
- R(提伯斯) → W(焚烧) → Q(碎裂之火)
- R先手召唤提伯斯

**反手连招**:
- E(熔岩护盾) → R(提伯斯) → W(焚烧)
- 护盾保护后反击

---

## 隐藏机制

1. **被动眩晕**: 被动满层时下次伤害技能眩晕敌人
2. **Q击杀刷新**: Q击杀目标返还冷却和蓝耗
3. **E护盾反弹**: E可以对攻击护盾的敌人造成伤害
4. **提伯斯**: 提伯斯可以控制，有火焰光环
5. **提伯斯复仇**: Annie死亡时提伯斯进入狂暴状态
