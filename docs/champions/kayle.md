#  Kayle (凯尔)

## 基础信息

- **称号**: 凯尔 (The Righteous)
- **类型**: 物理+魔法 / 近战+远程
- **难度**: 2/3
- **角色**: 战士、法师
- **位置**: 上单

## 技能介绍

### 被动 - 神圣崛起 (Divine Ascent)

**机制**: Kayle随等级提升获得四种形态，每6/11/16级进化。

**1级 - 热忱**: 攻速加成，最多5层
**6级 - 崛起**: 获得350额外攻击距离，变成远程
**11级 - 燃烧**: 普攻发射火焰波
**16级 - 超然**: 额外100攻击距离，永远获得热忱效果

### Q - 辐射爆炸 (Radiant Blast)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 900

Kayle召唤星芒，对敌人造成伤害和减速。

**伤害**: 60/90/120/150/180 (+ 60% bonus AD) (+ 50% AP)
**减速**: 25-45%

### W - 天赐祝福 (Celestial Blessing)

- **冷却**: 15 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 900

Kayle治疗自身或友军，并提供移速加成。

**治疗量**: 55/80/105/130/155 (+ 25% AP)
**移速加成**: 24-40%

### E - 星火之刃 (Starfire Spellblade)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 无
- **范围**: 350

Kayle强化下一次普攻，造成基于损失生命值的额外伤害。

**额外伤害**: 8-10% (+ 1.5% per 100 AP) 目标损失生命值

### R - 神圣审判 (Divine Judgment)

- **冷却**: 160/120/80 秒
- **消耗**: 100/50/0 法力
- **范围**: 900

Kayle使自身或友军免疫伤害，2.5秒后降下圣火。

**无敌持续**: 2.5秒
**伤害**: 200/300/400 (+ 100% bonus AD) (+ 70% AP)

## 连招技巧

### 基础连招

**对线消耗**:
- Q(辐射爆炸) → E(星火之刃) → 普攻
- 减速后追击

**保护连招**:
- W(天赐祝福) → E(星火之刃)
- 治疗和强化攻击

### 进阶连招

**团战连招**:
- R(神圣审判) → Q(辐射爆炸) → W(天赐祝福)
- 无敌保护后输出

**后期连招**:
- R(神圣审判) → 普攻 × 5 → Q(辐射爆炸)
- 满级后远程输出

---

## 隐藏机制

1. **形态进化**: 随等级提升获得攻击距离和伤害
2. **E强化**: E强化普攻造成额外魔法伤害
3. **R无敌**: R提供2.5秒无敌
4. **W治疗**: W同时治疗和提供移速
5. **被动远程**: 6级后变成远程英雄
