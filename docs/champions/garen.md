# 盖伦 (Garen)

## 基础信息

- **称号**: 德玛西亚之力 (The Might of Demacia)
- **类型**: 物理 / 近战
- **难度**: 1/3
- **角色**: 战士、坦克
- **位置**: 上单

## 技能介绍

### 被动 - 勇气 (Perseverance)

盖伦被动地增加护甲和魔法抗性，通过击杀敌人来提升。
- 击杀敌人增加0.25护甲和魔法抗性
- 最多叠加30层
- 脱战后快速回复生命值

**生命回复**:
- 每5秒回复0.5%最大生命值
- 脱战后回复速度翻倍

**隐藏机制**:
- 被动叠加的抗性在死亡后保留
- 脱战判定需要一段时间不承受伤害

### Q - 致命打击 (Decisive Strike)

- **冷却**: 8 秒
- **消耗**: 0
- **范围**: 300

盖伦获得移动速度爆发，摆脱所有减速效果。他的下次攻击打击敌人的要害，造成额外伤害并沉默。
- 移除所有减速效果
- 下次平A造成额外伤害和沉默

**效果数值**:
- 伤害: 30/55/80/105/130 (+0.4 AD)
- 沉默时长: 1.5/1.75/2/2.25/2.5 秒
- 移速加成: 30/35/40/45/50%，持续1.5秒

**隐藏机制**:
- Q可以移除所有减速效果
- Q重置平A
- Q+E可以取消Q后摇
- Q+E是盖伦的核心连招之一

### W - 勇气 (Courage)

- **冷却**: 23/21/19/17/15 秒
- **消耗**: 0
- **范围**: 0

盖伦被动地通过击杀敌人增加护甲和魔法抗性。他也可以激活这个技能，在短时间内获得护盾和韧性，然后在一段较长时间内获得伤害减免。
- 被动：击杀叠加双抗
- 主动：短暂护盾 + 韧性 + 后续伤害减免

**效果数值**:
- 被动每层: +0.25 护甲/魔抗
- 最大层数: 30
- 主动护盾: 70/115/160/205/250 (+0.8 AD)
- 韧性: 30/35/40/45/50%，持续0.5-2秒
- 伤害减免: 20/25/30/35/40%，持续2/2.5/3/3.5/4秒

**隐藏机制**:
- W的韧性可以减少控制时间
- W护盾可以用来挡技能
- W的伤害减免在后期很显著
- 被动叠加的抗性在早期提供不错的双抗

### E - 审判 (Judgment)

- **冷却**: 9 秒
- **消耗**: 0
- **范围**: 325

盖伦快速旋转身体，对附近所有敌人造成物理伤害。
- 持续1.5秒
- 每0.5秒造成一次伤害
- 总共最多3次伤害

**效果数值**:
- 每次伤害: 20/35/50/65/80 (+0.7/0.8/0.9/1.0/1.1 AD)
- 最大伤害（3次）: 60/105/150/195/240 (+2.1/2.4/2.7/3.0/3.3 AD)
- 暴击加成: 50%

**隐藏机制**:
- E的范围伤害可以击中多个敌人
- E可以触发提亚玛特/贪欲九头蛇
- E配合Q可以取消Q后摇
- E配合W可以取消E后摇
- E可以被走位拉扯
- E的持续期间可以穿插平A

### R - 德玛西亚正义 (Demacian Justice)

- **冷却**: 120/100/80 秒
- **消耗**: 0
- **范围**: 400

盖伦召唤德玛西亚的力量，试图处决一个敌方英雄。
- 造成基于目标已损失生命值的伤害
- 伤害随技能等级提升

**效果数值**:
- 伤害: 150/300/450 (+0.75 AD）
- 斩杀伤害: 100/200/300 (+1.5 AD) 基于已损失生命值

**隐藏机制**:
- R的伤害基于目标已损失生命值
- 目标血量越低，R伤害越高
- R可以用来抢龙/抢buff
- R斩杀后会获得目标区域的视野

## 连招技巧

### 基础连招

**对线消耗**:
- Q起手 → 平A → E → W → 拉开

**击杀连招**:
- Q → 平A → E → 平A → R

**E+Q取消后摇**:
- E旋转 → Q → 取消E后摇，增加伤害

### 进阶连招

**WQ+E**:
- W护盾 → Q → 平A → E

**R斩杀连招**:
- 将敌人打到低血量 → R处决

**闪现+R**:
- 闪现调整R的距离
- 用于收割逃跑的敌人

**W+R**:
- W护盾 → R处决

### 团战进场

**切后排连招**:
- Q加速 → E切入 → W护盾 → R瞄准后排

**反手连招**:
- W护盾 → Q移除减速 → E反击

**闪现+E**:
- 闪现到敌人身边 → E旋转

## 隐藏机制

1. **Q移除减速**: Q可以移除所有减速效果
2. **Q重置平A**: Q可以取消平A后摇
3. **E范围伤害**: E的范围可以击中多个敌人
4. **W韧性**: W提供的韧性可以减少控制时间
5. **R斩杀机制**: R的伤害基于目标已损失生命值
6. **被动脱战回复**: 被动在脱战后提供生命回复
7. **E暴击加成**: E享受暴击加成
8. **W护盾**: W的主动可以挡技能
9. **Q+E取消**: E+Q可以取消E的后摇
10. **R视野获取**: R斩杀后获得目标区域视野

## 常见问题

**Q: 盖伦如何对线？**
A: 利用Q的加速和沉默消耗，等待敌人血量降低后R处决。

**Q: 盖伦的核心技巧是什么？**
A: Q重置平A，E+Q取消后摇是核心技巧。

**Q: 盖伦如何打团？**
A: 利用Q加速切入后排，E旋转倾泻伤害，R处决低血量敌人。
