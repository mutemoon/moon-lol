#  Evelynn (伊芙琳)

## 基础信息

- **称号**: 伊芙琳 (Agony's Embrace)
- **类型**: 魔法 / 近战
- **难度**: 3/3
- **角色**: 刺客
- **位置**: 打野

## 技能介绍

### 被动 - 恶魔之影 (Demon Shade)

**机制**: Evelynn在4秒不行动后进入恶魔之影伪装。6级后，低于生命值时获得治疗和伪装效果。

**治疗量**: 15-165 (基于等级)
**伪装生命值**: 250-630 (基于等级) (+ 250% AP)

### Q - 仇恨尖刺 (Hate Spike)

- **冷却**: 4 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 800

Evelynn向目标方向发射飞镖，对首个敌人造成伤害并标记。再次激活释放尖刺。

**伤害**: 25/30/35/40/45 (+ 25% AP)
**额外伤害**: 15/25/35/45/55 (+ 25% AP) 标记目标

### W - 魅惑 (Allure)

- **冷却**: 15/14/13/12/11 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 1200/1600

Evelynn诅咒目标敌人5秒。下次攻击或技能引爆目标，减速。持续2.5秒后魅惑敌人。

**减速**: 45%，持续0.75秒
**魅惑时间**: 1.25/1.5/1.75/2/2.25秒
**魔抗削减**: 35/37.5/40/42.5/45%

### E - 鞭打 (Whiplash)

- **冷却**: 8 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 210

Evelynn用鞭子抽打目标，造成魔法伤害。获得移速加成。被动使E可以强化。

**伤害**: 60/90/120/150/180 (+ 3% (+ 1.5% AP) 目标最大生命值)
**移速加成**: 30/35/40/45/50%

### R - 最后轻吻 (Last Caress)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 500

Evelynn显露真身，造成伤害，然后闪烁后退。对低生命值敌人造成额外伤害。

**伤害**: 125/250/375 (+ 75% AP)
**低血量伤害**: 300/600/900 (+ 180% AP)

## 连招技巧

### 基础连招

**标准爆发**:
- Q(仇恨尖刺) → W(魅惑) → E(鞭打) → R(最后轻吻)
- Q标记，W魅惑，E输出

**消耗连招**:
- Q(仇恨尖刺) × 3 → W(魅惑) → E(鞭打)
- Q先手标记

### 进阶连招

**团战连招**:
- W(魅惑) → Q(仇恨尖刺) → R(最后轻吻) → E(鞭打)
- W先手魅惑，R收割

**追杀连招**:
- Q(仇恨尖刺) → E(鞭打) → R(最后轻吻)
- Q减速，E输出

---

## 隐藏机制

1. **被动伪装**: Evelynn在低血量时获得伪装
2. **W魔抗削减**: W对英雄造成魔抗削减
3. **R闪烁**: R造成伤害后闪烁后退
4. **Q标记**: Q标记敌人，后续攻击造成额外伤害
5. **E强化**: 被动使E可以强化
