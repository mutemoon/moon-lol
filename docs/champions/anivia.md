#  Anivia (冰晶凤凰)

## 基础信息

- **称号**: 冰晶凤凰 (The Cryophoenix)
- **类型**: 魔法 / 远程
- **难度**: 3/3
- **角色**: 法师
- **位置**: 中单

## 技能介绍

### 被动 - 蛋生 (Rebirth)

**机制**: Anivia受到致命伤害时进入6秒复活状态，复活后恢复所有生命值。复活期间获得额外双抗。

**额外双抗**: -40-20 (基于等级)
**复活时间**: 6秒

### Q - 寒冰闪耀 (Flash Frost)

- **冷却**: 11/10/9/8/7 秒
- **消耗**: 80/85/90/95/100 法力
- **范围**: 1100

Anivia向目标方向发射冰晶，对敌人造成伤害和减速。再次激活可使冰晶爆炸，造成伤害和眩晕。

**伤害**: 50/70/90/110/130 (+ 25% AP)
**爆炸伤害**: 60/95/130/165/200 (+ 45% AP)
**眩晕时间**: 1.1/1.2/1.3/1.4/1.5 秒

### W - 冰墙 (Crystallize)

- **冷却**: 17 秒
- **消耗**: 70 法力
- **范围**: 1000

Anivia创造一道冰墙，击退所有单位并存在5秒作为地形。

**冰墙宽度**: 600/700/800/900/1000

### E - 霜冻射击 (Frostbite)

- **冷却**: 4 秒
- **消耗**: 50 法力
- **范围**: 600

对目标敌人造成魔法伤害。如果目标被Q或R冻结，伤害翻倍。

**伤害**: 55/80/105/130/155 (+ 55% AP)
**冻结伤害**: 110/160/210/260/310 (+ 110% AP)

### R - 冰川风暴 (Glacial Storm)

- **冷却**: 4/3/2 秒
- **消耗**: 60 法力 + 35/45/55/秒
- **范围**: 750

Anivia在目标区域召唤暴风雪，持续对区域内敌人造成伤害和减速。暴风雪1.5秒后完全形成，造成300%伤害并增强减速效果。

**每跳伤害**: 15/22.5/30 (+ 6.25% AP)
**完全体伤害**: 45/67.5/90 (+ 18.75% AP)

## 连招技巧

### 基础连招

**标准消耗**:
- Q(寒冰闪耀) → E(霜冻射击) → R(冰川风暴)
- Q先手减速，E触发冻结伤害

**控制连招**:
- R(冰川风暴) → Q(寒冰闪耀) → E(霜冻射击)
- R持续减速，Q眩晕

### 进阶连招

**团战连招**:
- R(冰川风暴) → Q(寒冰闪耀) → W(冰墙) → E(霜冻射击)
- R先手，Q眩晕，W分割战场

**击杀连招**:
- Q(寒冰闪耀) → R(冰川风暴) → E(霜冻射击)
- Q先手，R持续输出

---

## 隐藏机制

1. **被动复活**: Anivia的被动可以在死亡后复活，团战中非常强力
2. **Q+W配合**: W可以创造地形阻挡敌人逃跑
3. **E冻结伤害**: E对冻结目标造成双倍伤害
4. **R持续伤害**: R的暴风雪可以持续对敌人造成伤害
5. **Q眩晕**: Q的爆炸可以眩晕敌人
6. **双抗加成**: 被动复活期间获得额外双抗
