# 乌加斯 (Urgot)

## 基础信息

- **称号**: 重装战士 (the Dreadnought)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、射手
- **位置**: 上单

## 技能介绍

### 被动 - 碾核重锤 (Echoing Flames)

乌加斯的普通攻击在踩到技能命中的敌人时会触发额外伤害。
- 每当技能命中敌人，后续平A会附加额外魔法伤害
- 附加伤害基于目标已损失生命值

**效果数值**:
- 额外伤害: 10-50 (基于等级) (+0.25 AD)

### Q - 腐蚀性Charge (Corrosive Charge)

- **冷却**: 10/9.5/9/8.5/8 秒
- **消耗**: 70 法力
- **范围**: 800

向目标位置发射爆炸性Charge，对爆炸范围内的敌人造成物理伤害并减速。
- 爆炸半径约250
- 减速持续0.25秒

**效果数值**:
- 伤害: 40/70/100/130/160 (+0.6 AD)
- 减速: 40/45/50/55/60%

**隐藏机制**:
- Q可以被走位躲避
- Q配合E使用可以增加命中率
- Q的减速很短暂，主要用于追人

### W - 净化 (Purge)

- **冷却**: 12/9/6/3/0 秒
- **消耗**: 40/30/20/10/0 法力
- **范围**: 490

乌加斯降低自己的移动速度，向附近敌人开火。优先攻击近期被其他技能命中的敌方英雄，并触发碾核重锤。
- 激活期间可以移动但移速降低
- 对范围内所有敌人攻击
- 优先攻击被Q/E命中的敌人

**效果数值**:
- 伤害: 20/45/70/95/120 (+0.4 AD)（每发）
- 攻速: 每秒4/5/6/7/8 发
- 移速降低: 25%
- 持续: 直到取消或法力耗尽

**隐藏机制**:
- W的伤害是范围伤害，可以攻击多个敌人
- W可以触发被动的额外伤害
- W可以取消Q的后摇
- W期间乌加斯是近战攻击判定
- W可以触发提亚玛特/贪欲九头蛇的AOE

### E - 蔑视 (Disdain)

- **冷却**: 16/15.5/15/14.5/14 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 475

乌加斯向一个方向冲刺，为自己提供护盾，并踩踏非英雄单位。如果撞到敌方英雄，他会停止并将敌人甩开。
- 冲刺距离约300
- 护盾持续0.75秒
- 撞到英雄会停止冲刺

**效果数值**:
- 护盾值: 80/130/180/230/280 (+0.8 AD)
- 伤害: 60/100/140/180/220 (+1.0 AD)
- 甩开距离: 300

**隐藏机制**:
- E可以穿越薄墙
- E护盾可以用来挡技能
- E可以打断敌方技能
- E配合W可以取消E的后摇
- E闪可以改变冲刺方向

### R - 超越死亡的恐惧 (Fear Beyond Death)

- **冷却**: 100/85/70 秒
- **消耗**: 100 法力
- **范围**: 2500

乌加斯发射一枚化学钻头，穿刺第一个命中的敌方英雄。如果该英雄生命值低于阈值，乌加斯判定其虚弱，可以将其处决。
- 发射后有短暂延迟
- 低于阈值直接处决
- 处决后乌加斯可以控制被处决者的灵魂

**效果数值**:
- 伤害: 150/300/450 (+1.0 AD)
- 斩杀阈值: 25/30/35% 生命值
- 处决后移速加成: 50%，持续3秒

**隐藏机制**:
- R的斩杀阈值基于目标当前生命值百分比
- 处决成功后乌加斯获得敌方位置的视野
- 处决后乌加斯和友军可以看到敌人灵魂
- R可以用来抢龙/抢buff
- R发射可以被走位或闪现躲避

## 连招技巧

### 基础连招

**对线消耗**:
- Q起手 → 命中后W → 平A → E拉开
- 利用Q的减速确保W命中

**EWQ连招**:
- E近身 → W倾泻火力 → Q补伤害 → 平A

**击杀连招**:
- Q → W → 平A → E → 继续W倾泻

### 进阶连招

**E+W取消后摇**:
- E近身 → W → 取消E后摇

**R处决连招**:
- 将敌人打到低血量 → R发射 → 斩杀
- R斩杀后获得敌方位置视野

**闪现+E**:
- 闪现调整E的角度
- 可以用来抓取逃跑的敌人

**E+闪+R**:
- E接近 → 闪到敌人身后 → R发射处决

### 团战进场

**切后排连招**:
- E进入战场 → W倾泻火力 → R瞄准后排 → 处决

**反手连招**:
- 等待敌人先手 → E挡住 → W反击

**R抢龙/抢buff**:
- 计算斩杀伤害 → R发射

## 隐藏机制

1. **W范围伤害**: W可以同时攻击多个敌人
2. **W触发被动**: W命中被Q/E标记的敌人会触发被动额外伤害
3. **E护盾**: E提供的护盾在冲刺结束后仍持续一小段时间
4. **Q减速短暂**: Q的减速只有0.25秒
5. **R可以被躲避**: R发射有延迟，可以走位或闪现躲避
6. **E可以穿墙**: E可以穿越部分薄墙
7. **W降移速**: W激活时乌加斯移速降低25%
8. **R处决优先级**: R只处决第一个命中的英雄
9. **E撞墙停止**: E撞到地形会停止冲刺
10. **W可以取消Q**: 在Q后立即W可以取消Q的后摇

## 常见问题

**Q: 乌加斯如何对线？**
A: 利用Q远程消耗，等待敌人走位失误后E近身，配合W倾泻火力。

**Q: 乌加斯的R如何使用？**
A: R主要用于斩杀低血量敌人，也可以用来获取敌方位置视野。

**Q: 乌加斯的核心技巧是什么？**
A: E+W取消后摇是核心，可以最大化输出。
