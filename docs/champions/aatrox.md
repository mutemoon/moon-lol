# 剑魔 (Aatrox)

## 基础信息

- **称号**: 暗裔魔剑 (the Darkin Blade)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士
- **位置**: 上单、打野

## 技能介绍

### 被动 - 死亡镰刀 (Deathbringer Stance)

亚托克斯周期性获得强化，下一次普通攻击造成基于目标最大生命值的额外魔法伤害，并治疗自身。
- 冷却约15秒
- 额外伤害: 目标最大生命值的 4-10%
- 治疗量: 造成伤害的 40%

### Q - 暗裔之刃 (The Darkin Blade)

- **冷却**: 14/12/10/8/6 秒
- **消耗**: 无
- **范围**: 25000

亚托克斯挥舞巨剑，造成物理伤害。边缘命中会造成额外伤害并击飞。
- 可以施放三次，每段形状不同
- 每段比前一段多25%伤害
- Q可以配合E取消后摇

**Q的三段形态**:
1. 第一段：直线斩击
2. 第二段：斜向斩击
3. 第三段：圆形下劈

**边缘机制**:
- Q的边缘（sweet spot）造成更高伤害
- 边缘命中会短暂击飞敌人0.25秒
- 边缘伤害约为基础伤害的 160%

**隐藏机制**:
- Q的前摇约0.3秒，可以通过E取消
- Q三段可以穿插平A增加伤害
- E可以在Q蓄力过程中使用
- Q的边缘判定比视觉效果略小

### W - 冥府之链 (Infernal Chains)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 无
- **范围**: 825

亚托克斯发射锁链，减速第一个命中目标，持续1秒并造成伤害。
- 命中的英雄和大型野怪有1秒时间离开影响范围
- 否则会被拉回中心并受到等额伤害

**效果数值**:
- 伤害: 100/145/190/235/280 (+1.0 AP)
- 减速: 40%，持续1秒

**隐藏机制**:
- W的拉扯判定有延迟，约1秒后才生效
- 可以利用这个延迟进行走位
- W可以被闪现、沙漏等无敌效果躲避

### E - 暗影冲锋 (Umbral Dash)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 无
- **范围**: 25000

被动：亚托克斯对英雄造成伤害时治疗自身。
主动：亚托克斯向面朝方向冲刺。

**被动效果**:
- 对英雄造成伤害的 15% 转化为治疗

**隐藏机制**:
- E可以在Q蓄力过程中使用（关键连招）
- E可以取消Q的后摇
- E闪可以改变冲刺方向
- E配合Q可以实现超远距离追击

### R - 世界终结者 (World Ender)

- **冷却**: 120/100/80 秒
- **消耗**: 无
- **范围**: 25000

亚托克斯显现真身，持续12秒。
- 恐惧周围小兵1秒
- 获得移动速度加成（衰减）
- 获得攻击力加成
- 增加自疗效果
- 击杀英雄延长持续时间并刷新移速加成

**效果数值**:
- 移动速度: 50/60/70%，持续12秒
- 攻击力加成: 20/25/30%
- 自疗增强: 25/30/35%
- 击杀延长: 5秒

**隐藏机制**:
- R期间亚托克斯体积略大
- R提供的移速可以追杀或撤退
- 击杀后刷新移速是追击关键

## 连招技巧

### 基础连招

**标准对线换血**:
- Q1 → E → Q2 → 平A → Q3
- 利用E取消Q后摇实现更流畅连招

**WEQ连招**:
- W命中 → E接近 → Q1 → Q2 → Q3
- 确保W命中后再用E追击

**Q+E Combo**:
- Q1蓄力 → E → Q2 → 平A → Q3
- 在Q蓄力过程中使用E取消后摇

### 进阶连招

**W+E+Q Combo**:
- W发射 → E取消W后摇 → Q1
- 可以快速接近敌人

**W+E+Q+闪 Combo**:
- W → E → Q1 → F → Q2 → Q3
- 超远距离追击

**RWQ连招**:
- R起手 → W → E → Q → Q → Q
- R期间利用移速加成追击

**R起手斩杀**:
- R → 接近目标 → Q → E → Q → 平A → Q
- 利用R的攻击力加成打出最大伤害

### 团战连招

**团战入场**:
- R起手 → E → W → Q → E → Q → Q
- 利用R的移速快速入场

**收割连招**:
- R → E → Q → W → Q → E → Q
- 击杀一个目标后R延长，追杀下一个

## 隐藏机制

1. **Q边缘击飞**: Q边缘命中会短暂击飞敌人0.25秒
2. **E取消Q后摇**: 关键技巧，大幅提升连招流畅度
3. **Q三段伤害递增**: 每段比前一段多25%伤害
4. **W拉扯延迟**: W命中有1秒后才拉扯，可以利用这个时间走位
5. **R移速刷新**: 击杀后移速加成刷新，可以持续追击
6. **R恐惧小兵**: 只恐惧小兵，不恐惧英雄
7. **E位移距离**: E位移约250-300距离
8. **Q+E Combo**: 在Q蓄力时使用E可以取消后摇，是核心技巧
9. **被动强化普攻**: 被动提供的额外伤害基于目标最大生命值
10. **E不可打断**: E位移过程中不会被控制技能打断
11. **Q范围指示器**: Q的范围比视觉效果略大
12. **R持续时间**: 12秒基础，每击杀延长5秒

## 常见问题

**Q: 剑魔怎么打连招最有效？**
A: 核心是Q+E combo，在Q蓄力时使用E取消后摇，可以实现无缝衔接的三段Q。

**Q: W拉扯的时机怎么把握？**
A: W命中后有1秒时间，敌人需要离开一个圆圈范围才能避免被拉。这个范围比视觉效果略小。

**Q: 剑魔团战怎么进场？**
A: R起手获得移速加成，E突进接W，然后Q三连。使用R追杀时注意刷新移速加成。

**Q: Q的边缘怎么判断？**
A: 巨剑的两侧约20%是边缘区域，命中造成更高伤害并击飞。边缘比视觉效果略小。
