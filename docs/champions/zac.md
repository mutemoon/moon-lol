# Zac (扎克)

## 基础信息

- **称号**: 扎克 (The Secret Weapon)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 坦克、战士
- **位置**: 打野

## 技能介绍

### 被动 - 细胞分裂 (Cell Division)

**机制**: Zac受到致命伤害时复活。

**复活时间**: 8/7/6/5/4秒
**复活效果**: 分裂成4个bloblet
**治疗效果**: 拾取碎片治疗

### Q - 拉伸打击 (Stretching Strikes)

- **冷却**: 14/12.5/11/9.5/8 秒
- **消耗**: 8% 当前生命值
- **范围**: 800

Zac伸展手臂抓住敌人。

**伤害**: 60/90/120/150/180 (+ 30% AP)
**减速**: 40%
**拉回效果**: 拉近敌人

### W - 不稳定物质 (Unstable Matter)

- **冷却**: 5 秒
- **消耗**: 4% 当前生命值
- **范围**: 350

Zac爆炸造成范围伤害。

**伤害**: 40/50/60/70/80 (+ 4-8% 最大生命值)
**冷却减免**: 收集碎片减少1秒

### E - 弹性弹弓 (Elastic Slingshot)

- **冷却**: 22/19/16/13/10 秒
- **消耗**: 4% 当前生命值
- **范围**: 1200-1800

Zac弹射向目标。

**伤害**: 60/105/150/195/240 (+ 80% AP)
**击飞时间**: 0.5-1秒
**蓄力效果**: 增加射程

### R - 弹跳! (Let's Bounce!)

- **冷却**: 120/105/90 秒
- **消耗**: 自助
- **范围**: 300

Zac弹跳造成范围伤害。

**伤害**: 120/190/260 (+ 40% AP)
**总伤害**: 300/475/650 (+ 100% AP)
**减速**: 20%

## 连招技巧

### 基础连招

**打野连招**:
- Q(拉伸打击) → W(不稳定物质) → E(弹性弹弓)
- 控制输出

**对线连招**:
- E(弹性弹弓) → Q(拉伸打击) → W(不稳定物质)
- 弹射控制

### 进阶连招

**团战连招**:
- R(弹跳!) → E(弹性弹弓) → Q(拉伸打击) → W(不稳定物质)
- 大招入场

**逃脱连招**:
- E(弹性弹弓) → R(弹跳!)
- 弹射逃离

## 隐藏机制

1. **被动复活**: 受到致命伤害时复活
2. **Q拉回**: Q拉近敌人
3. **W爆炸**: W范围伤害
4. **E弹射**: E可以穿墙
5. **R弹跳**: R可以多次弹跳
