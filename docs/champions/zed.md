# 劫 (Zed)

## 基础信息

- **称号**: 影流之主 (The Master of Shadows)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 刺客
- **位置**: 打野、中单

## 技能介绍

### 被动 -  contempt for the Weak

**机制**: 劫的普攻对50%血量以下的敌人造成额外魔法伤害。

**额外伤害**: 6/8/10% (基于等级) 目标最大生命值

### Q - 利刃 (Razor Shuriken)

- **冷却**: 6 秒
- **消耗**: 75/70/65/60/55 能量
- **范围**: 925

劫向目标方向投掷飞刃，对命中的敌人造成物理伤害。

**伤害**: 80/120/160/200/240 (+ 100% bonus AD)
**后续目标**: 60%伤害

### W - 影分身 (Living Shadow)

- **冷却**: 20/19/18/17/16 秒
- **消耗**: 40/35/30/25/20 能量
- **范围**: 650

劫创建一个影分身，可以复制Q和E技能。

**能量回复**: 30/35/40/45/50

**再次激活**: 劫和分身交换位置

### E - 鬼斩 (Shadow Slash)

- **冷却**: 5/4.5/4/3.5/3 秒
- **消耗**: 40 能量
- **范围**: 315

劫挥舞利刃，对附近敌人造成物理伤害。被分身命中的敌人会被减速。

**伤害**: 70/95/120/145/170 (+ 80% bonus AD)
**减速**: 20/25/30/35/40%

### R - 死亡标记 (Death Mark)

- **冷却**: 120/110/100 秒
- **消耗**: 无
- **范围**: 625

劫标记目标敌人，3秒后造成基于期间造成伤害的伤害。

**延迟**: 0.6秒后
**标记伤害**: 100% AD (+ 25/40/55% 存储伤害)

## 连招技巧

### 基础连招

**标准消耗**:
- W(影分身) → Q(利刃) → E(鬼斩) → 再次激活W
- W创造分身，Q消耗，E减速，W交换位置

**爆发连招**:
- R(死亡标记) → W(影分身) → Q(利刃) → E(鬼斩) → 再次激活R
- R标记目标，W创造分身，Q和E输出

### 进阶连招

**追杀连招**:
- W(影分身) → 再次激活W → Q(利刃) → E(鬼斩)
- W接近目标，分身交换位置，Q消耗

**团战连招**:
- R(死亡标记) → W(影分身) → E(鬼斩) → 再次激活W
- R标记主要目标，W分身E减速

**能量管理**:
- 利用W命中同一目标时回复能量
- Q命中后接E减少W冷却

---

## 隐藏机制

1. **W分身**: W的分身可以复制Q和E技能
2. **W交换**: 再次激活W可以和分身交换位置
3. **E减速**: E对被分身命中的敌人造成额外减速
4. **R标记**: R的标记在3秒后爆发，存储期间造成的伤害
5. **被动增伤**: 被动对低血量目标有额外伤害
6. **能量回复**: W命中同一目标时回复能量
