# 剑姬 (Fiora)

## 基础信息

- **称号**: 玉剑传说 (the Grand Duelist)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、刺客
- **位置**: 上单

## 技能介绍

### 被动 - 决斗之舞 (Duelist's Dance)

菲奥娜在敌方英雄身上揭露一个要害。对要害发起的攻击会治疗菲奥娜并提供移动速度加成。
- 要害会随英雄面向旋转
- 击破要害后获得8%移动速度，持续1.5秒
- 要害位置每4秒刷新一次

### Q - 前刺 (Lunge)

- **冷却**: 13/11.25/9.5/7.75/6 秒
- **消耗**: 20/25/30/35/40 法力
- **范围**: 400

菲奥娜向一个方向突刺，刺向最近的敌人、守卫或建筑物，造成物理伤害。
- 优先攻击要害和可以击杀的目标
- 如果命中敌人，冷却时间减少40%

**效果数值**:
- 伤害: 40/70/100/130/160 (+0.9 AD)
- 如果击中要害，伤害翻倍计算

**隐藏机制**:
- Q可以被任何伤害类技能重置（如W、E、R的刺击）
- Q的目标优先级：悬停目标 > 可击杀目标 > 最近目标
- Q可以穿越地形（薄墙）

### W - 招架 (Riposte)

- **冷却**: 24/22/20/18/16 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 750

菲奥娜招架所有即将到来的伤害、控制效果和debuff，持续0.75秒，然后刺击。
- 刺击对第一个命中的英雄造成魔法伤害
- 减速50%，持续2秒
- 如果招架了一个硬控（眩晕、禁锢、压制等），目标会被眩晕而非减速

**效果数值**:
- 刺击伤害: 100/140/180/220/260 (+1.0 AP)
- 攻击速度减速: 50%
- 移动速度减速: 50%

**隐藏机制**:
- W的持续时间只有0.75秒，需要精确预判
- W可以格挡：技能伤害，平A伤害、召唤师技能（引燃、虚弱等）
- W格挡后反刺的伤害享受穿刺加成
- 预判失误时W仍然会刺出一刀（只是没有东西可挡）
- W格挡的优先级极高，可以挡掉赛恩的大招

### E - 利刃之舞 (Bladework)

- **冷却**: 11/10/9/8/7 秒
- **消耗**: 40 法力
- **范围**: 425

菲奥娜获得额外攻击速度，持续1.5秒。她的下两次攻击：
- 第一击：减速40%，持续1秒
- 第二击：必定暴击，造成150%伤害

**效果数值**:
- 攻击速度: 50/60/70/80/90%
- 暴击伤害: 150% (+0.3 AD)

**隐藏机制**:
- E的两击都可以触发攻击特效（贪欲吸血、生命偷取等）
- E的两击都消耗被动充能
- 第二击必定暴击但享受额外攻击速度加成
- E期间穿插Q会打断E的动画

### R - 决斗 (Grand Challenge)

- **冷却**: 110/90/70 秒
- **消耗**: 100 法力
- **范围**: 500

菲奥娜揭露目标身上所有四个要害（持续8秒），造成最大生命值真实伤害。
- 如果在4秒内击破全部四个要害，或目标在击破至少一个要害后死亡，菲奥娜对周围友军每秒恢复生命值，持续4秒
- 被大招标记的塔姆可以E走队友吃掉（需要快速击破4个要害）

**效果数值**:
- 要害总伤害: 相当于目标最大生命值的 12/14/16%
- 每个要害: 3/3.5/4% 最大生命值
- 治疗效果: 50/75/100 (+0.25 AP) 生命值/秒

**隐藏机制**:
- 大招期间移动速度加成提升至30%（从被动的8%）
- 如果目标在4秒内没有死，要害会消失
- 击破全部4个要害的治疗范围约为500
- R可以重置Q的冷却（在某些情况下）

## 连招技巧

### 基础连招

**常规换血**:
- Q击中要害 → 平A → E → 平A → W(减速) → 拉开

**Q起手连招**:
- Q → 平A → W → 平A → E → 平A

**E起手连招**:
- 平A → E → 平A → Q → W → 平A

### 进阶连招

**瞬间四破 (Instant 4-Vitals)**:
- R → 平A(击破第一个要害) → E → F(闪现到侧面要害) → Q(击破第二个要害)
- 紧接着走位到下一个要害 → 平A → 平A → Q(击破第三个要害) → 平A → 平A(击破第四个要害)
- 难度极高，需要大量练习

**W格挡反打**:
- 预判敌人关键技能 → W格挡 → 反刺 → Q跟上 → 平A
- 例如：对阵鳄鱼时等他W过来瞬间W格挡

**利用小兵刷新Q**:
- Q远处小兵刷新冷却 → Q跳向敌人 → 平A → E → 平A
- 可以实现超远距离突进

### 团战连招

**R+AA+E+F+Q**:
- 大招标记 → 平A击破第一要害 → E侧跳 → 闪现到另一侧要害
- Q击破第二要害 → 平A第三要害 → Q第四要害

**反手连招**:
- W格挡敌人关键伤害 → R → E追击 → 逐个击破要害

## 隐藏机制

1. **Q可以重置平A**: Q后可以立即接平A，Q的冲刺会取消平A后摇
2. **W格挡优先级**: W可以格挡派克的R、赛恩的R等引导技能
3. **E不重置平A**: E之后需要等平A动画完成，不能像Q那样取消
4. **R期间被动加速**: 大招期间被动移速加成从8%提升到30%
5. **要害旋转速度**: 要害以英雄为中心每4秒旋转180度
6. **Q目标优先级**: 优先攻击鼠标悬停的单位
7. **E第二击必暴击**: 即使不出暴击装备，第二击也必定暴击
8. **W减速不吃韧性**: W的减速效果不受韧性影响
9. **Q可以穿薄墙**: 一些薄墙（如龙坑墙）可以被Q穿越
10. **R的真实伤害**: 大招的全部伤害都是真实伤害，无视护盾和护甲
11. **格挡塔伤害**: W可以格挡防御塔的攻击（0.75秒内）
12. **E后平A不能取消**: E之后必须等平A动画完成，无法接Q取消

## 常见问题

**Q: 如何快速击破四个要害？**
A: 需要大量练习要害位置，预判敌人走位，提前调整站位。高分剑姬玩家会使用E+F来快速切换位置。

**Q: W格挡哪些技能最有效？**
A: 鳄鱼W、瑞文W、潘森W等硬控技能。也可以格挡引燃、虚弱等召唤师技能。

**Q: 剑姬对线怎么换血？**
A: 利用Q远程消耗+击破要害获得移速加成撤退。E的减速留作追击或反打。
