# 阿狸 (Ahri)

## 基础信息

- **称号**: 九尾妖狐 (The Nine-Tailed Fox)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、刺客
- **位置**: 中单

## 技能介绍

### 被动 - 灵魂掠夺 (Essence Theft)

**机制**: 阿狸在击杀小兵或野怪时获得一层灵魂碎片。收集9层后消耗它们来治疗自身。

- 小兵/野怪击杀：35-102 (基于等级) (+ 20% AP)
- 英雄击杀（伤害后3秒内）：75-176 (基于等级) (+ 30% AP)

### Q - 欺诈宝珠 (Orb of Deception)

- **冷却**: 7秒
- **消耗**: 55/65/75/85/95 法力
- **范围**: 900

阿狸放出她的宝珠，对穿过的敌人造成魔法伤害。到达最大距离后返回，对穿过的敌人造成真实伤害。

**伤害（每段）**: 35/60/85/110/135 (+ 50% AP)

### W - 狐火 (Fox-Fire)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 30 法力
- **范围**: 550

阿狸获得40%移速加成（2秒衰减），并召唤三团火焰环绕自身（最多2.5秒）。

**火焰伤害**: 40/60/80/100/120 (+ 40% AP)
- 火焰优先攻击被魅惑的敌人
- 对低血量小兵伤害翻倍

### E - 魅惑 (Charm)

- **冷却**: 12秒
- **消耗**: 60 法力
- **范围**: 1000

阿狸释放一个飞吻，对第一个命中的敌人造成伤害，击飞并魅惑1.2-1.8秒（基于等级），减速65%。

**伤害**: 80/120/160/200/240 (+ 85% AP)

### R - 灵魄冲击 (Spirit Rush)

- **冷却**: 140/120/100 秒
- **消耗**: 100 法力
- **范围**: 500

阿狸向目标位置突进，然后对最多3个可见敌人发射魂魄弹，造成魔法伤害。可在15秒内重施最多2次。

**伤害**: 75/125/175 (+ 35% AP)

## 连招技巧

### 基础连招

**标准消耗连招**:
- E(魅惑) → Q(欺诈宝珠) → W(狐火) → R追击
- 魅惑确保技能命中，提高爆发伤害

**安全清线**:
- Q(欺诈宝珠) → Q返回
- 利用Q的两段伤害清线

### 进阶连招

**爆发连招**:
- E → Q → W → R → Q → R → R
- 魅惑后立刻接所有技能爆发

**追杀连招**:
- R起手接近 → E魅惑 → Q → W → R追击
- 利用R的位移接近目标

## 隐藏机制

1. **Q双倍伤害**: 宝珠去程造成魔法伤害，返回造成真实伤害，可以击中同一目标两次
2. **W自动寻敌**: 火焰会优先攻击被魅惑的目标，然后是英雄，最后是小兵
3. **R治疗机制**: 在R激活期间通过灵魂掠夺获得治疗会延长重施时间
4. **被动叠加**: 补刀小兵叠9层被动可以续航
5. **E命中判定**: 魅惑需要直接命中，非范围效果
