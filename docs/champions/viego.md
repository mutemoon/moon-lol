# 佛耶戈 (Viego)

## 基础信息

- **称号**: 破败之王 (The Ruined King)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士/刺客
- **位置**: 上单、打野

## 技能介绍

### 被动 - 君命难违 (King's Guard)

佛耶戈对被他的技能命中过的敌人造成额外伤害。每次命中会标记目标，一段时间后目标会受到已损失生命值的魔法伤害。

**效果数值**:
- 额外伤害: 目标当前生命的 6-12%（基于等级）
- 标记持续时间: 3 秒
- 已损失生命值伤害: 15% (每 100 AD +6%)

### Q - 破败之剑 (Blade of the Ruined King)

- **冷却**: 0 秒（可重置）
- **消耗**: 无
- **范围**: 500

佛耶戈的鬼魂之剑被动地造成额外伤害并偷取生命。每次攻击两次命中最近被佛耶戈技能命中的敌人。

**被动效果**:
- 额外伤害: 目标当前生命的 6/7/8/9/10%
- 生命偷取: 10/12/14/16/18%
- 双击判定: 命中最近被技能命中的敌人（额外一次普攻）

**主动效果**:
- 向前刺出一道暗影，造成 20/30/40/50/60 (+0.4 AD) 物理伤害
- 冷却: 4 秒

**隐藏机制**:
- Q 的双击可以被格挡（如布甲鞋）
- Q 主动可以穿越地形
- 双击享受攻击特效（如巨型九头蛇）

### W - 幽影随行 (Spectral Maw)

- **冷却**: 14/12/10/8/6 秒
- **消耗**: 无
- **范围**: 850

佛耶戈蓄力后向前冲刺，释放一团浓缩的黑雾，眩晕第一个命中的敌人。

**效果数值**:
- 蓄力时间: 0-0.75 秒（基于技能等级）
- 位移距离: 300
- 眩晕时长: 0.5-1.25 秒
- 伤害: 80/120/160/200/240 (+0.6 AD)

**隐藏机制**:
- W 可以蓄力也可以秒放
- W 蓄力时间越长，眩晕时间越长
- W 位移可以穿越地形
- 闪现可以取消 W 位移

### E - 欺心之魔 (Harrowed Path)

- **冷却**: 22/19/16/13/10 秒
- **消耗**: 无
- **范围**: 0

佛耶戈命令黑雾环绕一块地形。佛耶戈可以在黑雾中隐藏，获得伪装、移速和攻速加成。

**效果数值**:
- 伪装持续: 直到离开黑雾区域
- 移速加成: 30/35/40/45/50%
- 攻速加成: 20/30/40/50/60%
- 黑雾持续: 4 秒
- 冷却: 22/19/16/13/10 秒

**隐藏机制**:
- E 的黑雾可以遮挡视野
- 佛耶戈在黑雾中获得伪装（不是潜行）
- 可以配合 W 使用进行突进
- 黑雾可以放在地形边缘用于埋伏

### R - 心赴 (Heartbreaker)

- **冷却**: 120/90/60 秒
- **消耗**: 无
- **范围**: 500

佛耶戈传送到附近一个位置，执行一个敌方英雄，在命中时造成伤害并击杀目标。

**效果数值**:
- 位移距离: 500
- 斩杀伤害: 目标已损失生命的 15/20/25% (+0.4 AD) 的真实伤害
- 冲击波范围: 300
- 冲击波伤害: 150/225/300 (+0.6 AD)

**限制条件**:
- 只能对生命值低于 35/45/55% 的目标使用
- 需要有目标位置（无法对迷雾使用）

**隐藏机制**:
- R 可以用来追杀低血量目标
- 冲击波会击退周围友军
- R 释放后有一段无敌时间
- 可以配合 E 的伪装接近目标

## 连招技巧

### 基础连招

**标准对线换血**:
- Q → W（蓄力）→ 平A → Q → E 撤退
- W 蓄力眩晕后接 Q 双击

**E+W Combo**:
- E 放黑雾 → 进入黑雾获得伪装 → W 蓄力 → 位移接近 → 眩晕 → Q
- 利用伪装接近目标

### Mobalytics 高阶连招

**WEAAQRAA**:
- W → E（穿黑雾）→ AA → Q → R → AA（爆发连招）

**WEAAQAA**:
- W → E → AA → Q → AA（远程换血）

**EQAAWAAT**:
- E → Q → AA → W → AA（近距离对拼）

**WAAQ**:
- W → AA → Q（快速消耗）

**QEAAWAARAAT**:
- Q → E → AA → W → AA → R → AA → Q（全技能连招）

**WQF（闪现）**:
- W → Q → Flash 中途（远程斩杀/追击）

**WF（闪现）**:
- W 蓄力 → Flash（gank 埋伏）

**QF（闪现）**:
- Q → Flash 中途（低血量斩杀）

### 进阶连招

**Gank 连招**:
- E 放黑雾 → W 蓄力 → 位移 → Q → R 斩杀
- 利用 E 的移速加成接近目标

**R 追杀连招**:
- Q 命中 → 平A → W → Q → R
- R 斩杀低血量目标

**E+R Combo**:
- E 放黑雾 → 进入获得伪装 → R 传送到目标身后 → Q → W
- 利用 R 位移接近并斩杀

### 团战连招

**团战切入**:
- E 放黑雾 → 进入伪装 → W → Q → R → Q
- 利用伪装接近敌方后排

**收割连招**:
- R 斩杀低血量目标 → Q → W → Q
- R 击杀后刷新技能冷却

**反手连招**:
- W 蓄力 → E 放黑雾 → 进入黑雾 → Q
- 利用 E 的伪装躲避技能

## 隐藏机制

1. **Q 双击机制**: Q 的第二次攻击命中最近被技能命中的目标，享受攻击特效
2. **E 伪装不被侦测**: E 的黑雾提供的伪装不会被普通侦测发现
3. **W 蓄力眩晕递增**: W 蓄力时间越长，眩晕时间越长（0.5-1.25秒）
4. **W 可穿越地形**: W 的位移可以穿越地形
5. **R 斩杀条件**: R 需要目标生命值低于一定比例才能使用
6. **R 冲击波击退**: R 的冲击波会击退周围的友军
7. **Q 生命偷取**: Q 的被动提供生命偷取，可以用来续航
8. **被动已损失生命伤害**: 被动对被技能命中的目标造成已损失生命值的额外伤害
9. **E 视野遮挡**: E 的黑雾可以遮挡敌方视野

## 常见问题

**Q: 佛耶戈怎么快速叠被动？**
A: Q 的双击机制可以快速叠被动，每次技能命中的目标都会触发双击判定。

**Q: W 什么时候应该蓄力？**
A: 对线时可以秒 W 快速消耗，团战或 Gank 时建议蓄力以获得更长的眩晕时间。

**Q: 佛耶戈的 R 怎么用？**
A: R 主要用于斩杀低血量目标，需要目标生命值低于 35/45/55% 才能使用。

**Q: E 技能怎么放最好？**
A: E 应该放在地形边缘，让佛耶戈可以在黑雾中隐藏并获得伪装，便于接近目标。

**Q: 佛耶戈打团应该怎么进场？**
A: 利用 E 的伪装接近敌方后排，W 蓄力眩晕后接 Q 输出，R 用于收割低血量目标。
