#  Singed (辛吉德)

## 基础信息

- **称号**: 辛吉德 (The Mad Chemist)
- **类型**: 魔法 / 近战
- **难度**: 3/3
- **角色**: 坦克、法师
- **位置**: 上单

## 技能介绍

### 被动 - 毒气滑行 (Noxious Slipstream)

**机制**: Singed经过敌人时获得移速加成。

**移速加成**: 每层25% (最高625%)
**持续时间**: 2秒
**最大层数**: 25层

### Q - 毒药痕迹 (Poison Trail)

- **冷却**: 持续
- **消耗**: 13 法力/秒
- **范围**: 180

Singed在身后留下毒云。

**每跳伤害**: 5/7.5/10/12.5/15 (+ 10% AP)
**持续时间**: 3.25秒

### W - 超级粘液 (Mega Adhesive)

- **冷却**: 17/16/15/14/13 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 1000

Singed放置粘液区域，减速并禁锢敌人。

**减速**: 50/55/60/65/70%
**持续时间**: 3秒

### E - 强力投掷 (Fling)

- **冷却**: 10/9.5/9/8.5/8 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 125

Singed将敌人扔向身后。

**伤害**: 50/60/70/80/90 (+ 6-8% 最大生命值) (+ 55% AP)
**禁锢时间**: 1/1.25/1.5/1.75/2秒
**配合**: 扔到粘液上禁锢

### R - 疯狂药剂 (Insanity Potion)

- **冷却**: 100 秒
- **消耗**: 100 法力
- **范围**: 自助

Singed喝下药剂，获得大量属性加成。

**加成属性**: 25/55/85 AP、护甲、魔抗
**额外效果**: 毒药施加重伤

## 连招技巧

### 基础连招

**消耗连招**:
- W(超级粘液) → E(强力投掷) → Q(毒药痕迹)
- 粘液控制后投掷

**追杀连招**:
- Q(毒药痕迹) → W(超级粘液) → E(强力投掷)
- 毒雾追击

### 进阶连招

**团战连招**:
- R(疯狂药剂) → W(超级粘液) → E(强力投掷) → Q(毒药痕迹)
- 大招加速追击

**恶心流**:
- Q(毒药痕迹) → R(疯狂药剂) → E(强力投掷) → W(超级粘液)
- 持续毒雾消耗

---

## 隐藏机制

1. **Q毒云**: 持续伤害经过的敌人
2. **W区域**: W创造减速区域
3. **E投掷**: E可以把敌人扔向身后
4. **E+W配合**: 扔到粘液上触发禁锢
5. **R属性加成**: 大招提供全属性加成
