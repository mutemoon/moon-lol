#  Diana (黛安娜)

## 基础信息

- **称号**: 黛安娜 (Scorn of the Moon)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 刺客、战士
- **位置**: 打野、中单

## 技能介绍

### 被动 - 银光刃 (Moonsilver Blade)

**机制**: Diana获得15-37%攻速加成。施放技能后，攻速加成翻倍持续5秒。

**攻速加成**: 15-37% (基于等级)，技能后45-112%

### Q - 新月打击 (Crescent Strike)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 50 法力
- **范围**: 900

Diana释放弧形月牙能量，对敌人造成伤害并施加月光效果。

**伤害**: 70/105/140/175/210 (+ 70% AP)

### W - 月银之盾 (Pale Cascade)

- **冷却**: 15/13.5/12/10.5/9 秒
- **消耗**: 40/45/50/55/60 法力

Diana为自己提供护盾，并创造3个球体环绕她，对敌人造成伤害。

**护盾值**: 45/60/75/90/105 (+ 30% AP) (+ 9% bonus health)
**每球伤害**: 20/32/44/56/68 (+ 18% AP)

### E - 月神冲刺 (Lunar Rush)

- **冷却**: 22/20/18/16/14 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 825

Diana突进到目标，消耗月光效果并减少技能冷却。

**伤害**: 50/70/90/110/130 (+ 60% AP)
**冷却重置**: 消耗月光后冷却至0.25秒

### R - 月陨 (Moonfall)

- **冷却**: 100/90/80 秒
- **消耗**: 100 法力
- **范围**: 475

Diana拉拽所有附近敌人，减速他们。如果命中敌方英雄，引发月光束造成伤害。

**伤害**: 200/300/400 (+ 60% AP)
**额外伤害**: 35/60/85 (+ 15% AP) 每英雄
**减速**: 40/50/60%

## 连招技巧

### 基础连招

**标准爆发**:
- Q(新月打击) → E(月神冲刺) → W(月银之盾) → 普攻
- Q命中后E跟进消耗

**团战连招**:
- R(月陨) → Q(新月打击) → E(月神冲刺) → W(月银之盾)
- R先手拉拽，Q和E输出

### 进阶连招

**追杀连招**:
- Q(新月打击) → E(月神冲刺) → 普攻 × 3 → R(月陨)
- Q先手，E追击

**反手连招**:
- W(月银之盾) → R(月陨) → Q(新月打击) → E(月神冲刺)
- W护盾，R控制

## 隐藏机制

1. **E冷却重置**: 消耗月光后E冷却重置为0.25秒
2. **W护盾叠加**: W的三个球全部命中时护盾叠加
3. **Q月光**: Q对敌人施加月光效果
4. **R拉拽**: R可以拉拽所有附近敌人
5. **被动攻速**: 技能后攻速大幅提升
