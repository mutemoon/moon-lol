# Diana (黛安娜)

## 基础信息

- **称号**: 皎月女神 (Scorn of the Moon)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 刺客、战士
- **位置**: 打野、中单

## 技能介绍

### 被动 - 银光刃 (Moonsilver Blade)

**机制**: 黛安娜获得 15% – 37.35% 攻速加成。施放技能后，攻速加成翻三倍，持续 5 秒。

**额外效果**: 普攻生成技能叠加层数，最多 2 层。2 层时下一次普攻会额外对附近敌人造成 20 – 270 (+50% AP) 魔法伤害。

### Q - 新月打击 (Crescent Strike)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 50 法力
- **范围**: 900

黛安娜释放弧形月牙能量，对敌人造成伤害并施加月光效果。

**伤害**: 70/105/140/175/210 (+70% AP)
**月光持续**: 3 秒

### W - 月银之盾 (Pale Cascade)

- **冷却**: 15/13.5/12/10.5/9 秒
- **消耗**: 40/45/50/55/60 法力

黛安娜为自己提供护盾，并创造 3 个球体环绕她，对敌人造成伤害。

**护盾值**: 45/60/75/90/105 (+30% AP) (+9% bonus health)
**每球伤害**: 20/32/44/56/68 (+18% AP)
**最大护盾**: 90/120/150/180/210 (+60% AP) (+18% bonus health)

### E - 月神冲刺 (Lunar Rush)

- **冷却**: 22/20/18/16/14 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 825

黛安娜突进到目标，消耗月光效果并减少技能冷却。

**伤害**: 50/70/90/110/130 (+60% AP)
**冷却重置**: 消耗月光后冷却至 0.25 秒

### R - 月陨 (Moonfall)

- **冷却**: 100/90/80 秒
- **消耗**: 100 法力
- **范围**: 475

黛安娜拉拽所有附近敌人，减速他们。如果命中敌方英雄，引发月光束造成伤害。

**伤害**: 200/300/400 (+60% AP)
**额外伤害**: 35/60/85 (+15% AP) 每英雄
**减速**: 40/50/60%

## 连招技巧

### 基础连招

**标准爆发**:
- Q(新月打击) → E(月神冲刺) → W(月银之盾) → 普攻
- Q命中后 E 跟进消耗

**团战连招**:
- R(月陨) → Q(新月打击) → E(月神冲刺) → W(月银之盾)
- R 先手拉拽，Q 和 E 输出

### 进阶连招

**追杀连招**:
- Q(新月打击) → E(月神冲刺) → 普攻 × 3 → R(月陨)
- Q 先手，E 追击

**反手连招**:
- W(月银之盾) → R(月陨) → Q(新月打击) → E(月神冲刺)
- W 护盾，R 控制

## 隐藏机制

1. **E 冷却重置**: 消耗月光后 E 冷却重置为 0.25 秒
2. **W 护盾叠加**: W 的三个球全部命中时护盾叠加
3. **Q 月光**: Q 对敌人施加月光效果
4. **R 拉拽**: R 可以拉拽所有附近敌人
5. **被动攻速**: 技能后攻速大幅提升
