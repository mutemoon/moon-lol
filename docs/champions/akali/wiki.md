# 劫 (Akali)

## 基础信息

- **称号**: 刺客 (The Rogue Assassin)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 刺客
- **位置**: 上单、中单

## 技能介绍

### 被动 - 刺客印记 (Assassin's Mark)

**机制**: 阿卡丽用技能伤害敌方英雄时，会在目标周围创建一个环（持续4秒）。

- 离开环时获得30-60%移动速度加成（基于等级），持续2秒
- 离开环后获得强化攻击（Extended Kama），普攻距离翻倍，造成35-212 (+ 60% bonus AD)(+ 55% AP)额外魔法伤害

### Q - 五点射击 (Five Point Strike)

- **冷却**: 1.5秒
- **消耗**: 110/100/90/80/70 能量
- **范围**: 500

阿卡丽向锥形方向释放手里剑，造成魔法伤害。远距离目标被减速50%。

**伤害**: 45/70/95/120/145 (+ 65% AD)(+ 60% AP)

### W - 暮光之刃 (Twilight Shroud)

- **冷却**: 20/19/18/17/16 秒
- **消耗**: 无
- **范围**: 350（烟雾范围）

阿卡丽恢复100能量，获得30-50%移动速度加成（基于等级）。向目标方向投掷烟雾弹，扩散成环形遮罩，持续5-7秒。

**机制**:
- 进入烟雾时阿卡丽进入隐身状态
- 隐身期间不能使用技能或攻击
- 烟雾持续期间能量上限增加100

### E - 手里剑翻转 (Shuriken Flip)

- **冷却**: 16/14.5/13/11.5/10 秒
- **消耗**: 30 能量
- **范围**: 825

阿卡丽向后翻转，投掷手里剑标记第一个命中的敌人。可以再次激活技能冲向标记目标。

**手里剑伤害**: 21/42/63/84/105 (+ 30% AD)(+ 33% AP)
**冲刺伤害**: 49/98/147/196/245 (+ 70% AD)(+ 77% AP)

### R - 完美击杀 (Perfect Execution)

- **冷却**: 120/90/60 秒
- **消耗**: 无
- **范围**: 675

阿卡丽向目标敌方英雄冲刺，对路径上的敌人造成魔法伤害。可在10秒内再次激活。

**第一段伤害**: 110/220/330 (+ 50% bonus AD)(+ 30% AP)
**第二段伤害**: 70/140/210 - 210/420/630 (+ 30% AP)，基于目标已损失生命值增加

## 连招技巧

### 基础连招

**标准消耗**:
- Q(五点射击) → 普攻 → E(手里剑翻转)
- 利用被动环的移速加成追击或撤退

**清线连招**:
- W(暮光之刃) → Q → 在烟雾中等待能量回复 → Q

### 进阶连招

**爆发连招**:
- R(第一段) → E(手里剑翻转) → Q → 普攻 → E(冲刺) → R(第二段)
- 利用R接近后立刻接E标记

**追杀连招**:
- W(暮光之刃) → 进入烟雾隐身 → E(手里剑翻转)标记
- 等待目标使用位移后再次激活E冲刺

## 隐藏机制

1. **被动环移速**: 离开环时获得移速加成，可以用来穿墙
2. **W能量回复**: W可以恢复100能量，是阿卡丽能量管理的重要技能
3. **E标记机制**: E标记可以暴露敌方视野，包括隐身目标
4. **R斩杀伤害**: R第二段的伤害基于目标已损失生命值，残血目标伤害更高
5. **烟雾穿墙**: W的烟雾可以穿过某些薄墙
6. **被动强化攻击**: 强化普攻可以触发攻击特效
