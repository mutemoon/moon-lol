#  Janna (风暴之怒)

## 基础信息

- **称号**: 风暴之怒 (The Storm's Fury)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 辅助、法师
- **位置**: 辅助

## 技能介绍

### 被动 - 顺风而行 (Tailwind)

**机制**: Janna为附近面向她的友军提供移速加成。她的普攻和Zephyr技能基于额外移速造成额外魔法伤害。

**移速加成**: 6%
**额外伤害**: 30% 额外移速

### Q - 呼啸风暴 (Howling Gale)

- **冷却**: 14 秒
- **消耗**: 90/95/100/105/110 法力
- **范围**: 1100-1760

Janna召唤旋风，充能3秒后向目标方向发射。

**最小伤害**: 55/90/125/160/195 (+ 50% AP)
**击飞时间**: 0.5-1.25秒

### W - 鹰风 (Zephyr)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 550

Janna释放空气元素，造成魔法伤害并减速敌人。

**伤害**: 55/85/115/145/175 (+ 50% AP)
**减速**: 20-36%

### E - 风暴之眼 (Eye of the Storm)

- **冷却**: 16/15/14/13/12 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 800

Janna为友军或防御塔提供护盾，持续4秒。

**护盾**: 80/120/160/200/240 (+ 55% AP)
**额外攻击力**: 10/15/20/25/30 (+ 10% AP)

### R - 孟番 (Monsoon)

- **冷却**: 130/115/100 秒
- **消耗**: 100 法力
- **范围**: 700

Janna将附近敌人击退，并持续治疗自己和友军。

**治疗**: 25/37.5/50 (+ 12.5% AP) 每0.25秒
**击退距离**: 875

## 连招技巧

### 基础连招

**保护连招**:
- E(风暴之眼) → W(鹰风)
- 护盾后减速敌人

**进攻连招**:
- Q(呼啸风暴) → W(鹰风) → R(孟番)
- 击飞后追击

### 进阶连招

**团战连招**:
- R(孟番) → Q(呼啸风暴) → E(风暴之眼)
- 大范围控制保护

**反开连招**:
- E(风暴之眼) → R(孟番) → Q(呼啸风暴)
- 反手保护并控制

---

## 隐藏机制

1. **Q可蓄力**: Q可以蓄能增加距离和击飞时间
2. **W幽灵**: W被动使Janna永久幽灵状态
3. **E护盾**: 护盾为友军提供额外攻击力
4. **R击退**: R可以打断引导技能
5. **被动移速**: 面向Janna的友军获得移速加成
