# Talon (泰隆)

## 基础信息

- **称号**: 泰隆 (The Blade's Shadow)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 刺客
- **位置**: 中单

## 技能介绍

### 被动 - 刀锋之末 (Blade's End)

**机制**: Talon技能对敌人施加伤口层数。

**层数上限**: 3层
**最大伤害**: 80-303 (+ 210% 额外AD)
**持续时间**: 6秒
**3层效果**: 造成流血伤害

### Q - 诺克萨斯外交 (Noxian Diplomacy)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 40 法力
- **范围**: 575

Talon冲刺至目标，造成伤害。

**伤害**: 65/85/105/125/145 (+ 100% 额外AD)
**近战效果**: 150%暴击
**击杀治疗**: 9-60生命值
**冷却返还**: 50%

### W - 锄爪 (Rake)

- **冷却**: 9/8.5/8/7.5/7 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 900

Talon投掷匕首，伤害并减速敌人。

**伤害**: 50/60/70/80/90 (+ 40% 额外AD)
**返回伤害**: 60/90/120/150/180 (+ 90% 额外AD)
**减速**: 40/45/50/55/60%

### E - 刺客之道 (Assassin's Path)

- **冷却**: 2 秒
- **消耗**: 自助
- **范围**: 725

Talon翻越地形，无法被选中。

**翻越距离**: 800单位
**地形冷却**: 160/135/110/85/60秒

### R - 暗影袭击 (Shadow Assault)

- **冷却**: 100/80/60 秒
- **消耗**: 100 法力
- **范围**: 550

Talon释放暗影，造成伤害并获得隐身。

**伤害**: 90/135/180 (+ 100% 额外AD)
**移速加成**: 40/55/70%
**隐身时间**: 2.5秒
**总伤害**: 180/270/360 (+ 200% 额外AD)

## 连招技巧

### 基础连招

**对线连招**:
- W(锄爪) → Q(诺克萨斯外交) → E(刺客之道)
- 快速接近

**秒杀连招**:
- R(暗影袭击) → Q(诺克萨斯外交) → W(锄爪) → E(刺客之道)
- 隐身接近

### 进阶连招

**团战连招**:
- E(刺客之道) → R(暗影袭击) → Q(诺克萨斯外交) → W(锄爪)
- 翻墙进场

**逃脱连招**:
- R(暗影袭击) → E(刺客之道) → 翻墙逃跑
- 隐身撤离

## 隐藏机制

1. **被动流血**: 3层时触发流血
2. **Q位移**: Q可以快速接近目标
3. **W减速**: W返回时造成额外伤害
4. **E翻墙**: E可以翻越所有地形
5. **R隐身**: R提供隐身和移速加成
