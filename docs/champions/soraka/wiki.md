# 索拉卡 (Soraka)

## 基础信息

- **称号**: 众星之子 (The Starchild)
- **类型**: 魔法 / 远程
- **难度**: 1/3
- **角色**: 辅助、法师
- **位置**: 辅助

## 技能介绍

### 被动 - 救赎 (Salvation)

**机制**: 索拉卡面向附近40%血量以下的友军时获得大量移速加成。

**移速加成**: 90%

### Q - 星界诅咒 (Starcall)

- **冷却**: 8/7/6/5/4 秒
- **消耗**: 45/50/55/60/65 法力
- **范围**: 800

索拉卡在目标位置召唤星辰，对敌人造成魔法伤害并减速。

**伤害**: 85/120/155/190/225 (+ 35% AP)
**减速**: 30% (持续1.5秒)

**额外效果**:
- 命中英雄时获得"回复"效果
- 回复期间使用W会为目标也提供回复

**回复效果**:
- 治疗量: 60/75/90/105/120 (+ 30% AP)
- 移速加成: 20/22.5/25/27.5/30%

### W - 星之灌注 (Astral Infusion)

- **冷却**: 6/5/4/3/2 秒
- **消耗**: 10% 最大生命值 + 40/45/50/55/60 法力
- **范围**: 550

索拉卡治疗目标友军。

**治疗量**: 90/110/130/150/170 (+ 50% AP)

**额外效果**:
- 在"回复"期间使用时，生命消耗减少80-100%

### E - 星体结界 (Equinox)

- **冷却**: 20/19/18/17/16 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 925

索拉卡在目标位置创造一个结界，对区域内敌人造成伤害并沉默。1.5秒后结界爆发，造成相同伤害并定身敌人。

**沉默伤害**: 70/95/120/145/170 (+ 40% AP)
**总伤害**: 140/190/240/290/340 (+ 80% AP)
**定身时间**: 1/1.25/1.5/1.75/2 秒

### R - 祈愿 (Wish)

- **冷却**: 150/135/120 秒
- **消耗**: 100 法力
- **范围**: 全地图

索拉卡祈祷，治疗所有友军。40%血量以下的目标获得50%额外治疗。

**治疗量**: 150/250/350 (+ 50% AP)
**低血量目标**: 225/375/525 (+ 75% AP)

## 连招技巧

### 基础连招

**标准治疗**:
- W(星之灌注) → Q(星界诅咒)
- W治疗队友，Q伤害敌人

**消耗连招**:
- Q(星界诅咒) → W(星之灌注) → E(星体结界)
- Q减速敌人，W治疗，E控制

### 进阶连招

**团战治疗**:
- R(祈愿) → W(星之灌注) → Q(星界诅咒)
- R全图治疗，W治疗最虚弱的队友

**追击连招**:
- Q(星界诅咒) → E(星体结界) → W(星之灌注)
- Q减速，E沉默并定身，W治疗

**自保连招**:
- Q(星界诅咒) → W(星之灌注)
- 利用Q获得回复效果，W治疗自己

---

## 隐藏机制

1. **被动移速**: 被动在面向低血量友军时提供大量移速
2. **Q回复**: Q命中英雄后获得回复效果，可以传递
3. **W生命消耗**: W消耗自身生命值，需要注意血量
4. **E沉默**: E技能先沉默后定身
5. **R全图**: R可以治疗全地图所有友军
6. **R低血量加成**: R对低血量目标有50%额外治疗
