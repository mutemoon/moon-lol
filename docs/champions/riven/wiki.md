# 锐雯 (Riven)

## 基础信息

- **称号**: 放逐之刃 (the Exile)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、刺客
- **位置**: 上单

## 技能介绍

### 被动 - 符文之刃 (Runic Blade)

锐雯的技能会为她的blade充能，她的普通攻击会消耗充能来造成额外伤害。
- 最多可充能3次
- 每次攻击消耗一层充能

### Q - 破碎 (Broken Wings)

- **冷却**: 13秒
- **消耗**: 无消耗
- **范围**: 275

锐雯向前冲刺，造成物理伤害。这个技能可以在短时间内重施两次，第三次会击飞敌人。

**详细机制**:
- 第一/二次使用：向前挥砍，对接触到的所有敌人造成物理伤害
- 第三次使用：跃起并砸向地面，对周围敌人造成物理伤害并击飞0.75秒
- 第三次跃起可以穿越地形障碍
- 锐雯会以鼠标悬停的单位为目标，若无目标则向当前面向方向攻击

**效果数值**:
- 第一击伤害: 45/75/105/135/165
- 第三次击飞伤害: 75/150/225/300/375/450/525
- AD加成: 0.75

### W - 震击 (Ki Burst)

- **冷却**: 11/10/9/8/7/7秒
- **消耗**: 无消耗
- **范围**: 260

锐雯的剑刃释放一道符文能量波，造成物理伤害并眩晕周围敌人0.75秒。

**效果数值**:
- 伤害: 30/65/95/125/155/185/215
- AD加成: 1.0

### E - 勇斩 (Valor)

- **冷却**: 10/9/8/7/6/6秒
- **消耗**: 无消耗
- **范围**: 250

锐雯快速向鼠标方向冲刺，获得护盾值，护盾持续1.5秒。

**效果数值**:
- 护盾值: 50/70/95/120/145/170/235
- AD加成: 1.1

### R - 放逐之刃 (Blade of the Exile)

- **冷却**: 120/90/60秒
- **消耗**: 无消耗
- **范围**: 200

锐雯的武器被Spirit energy灌注，获得额外攻击力，持续8秒。在此期间，她可以使用一次 Wind Slash。

**主动效果**:
- 获得额外攻击力
- 技能和攻击的攻击范围增加
- 获得一次释放Wind Slash的机会

**Wind Slash**:
- 释放一道冲击波，对命中的所有敌人造成物理伤害
- 伤害基于敌人的已损失生命值，最低造成100/300/450/600伤害

**效果数值**:
- AD加成: 0.6
- 斩杀伤害: 1.8 (基于已损失生命值)

## 连招技巧

### 基础连招

**普通连招**:
- Q起手 × 3（配合平A）
- 在Q之间插入W来控制敌人
- 用E来获得护盾并调整位置

**快速连招 (Fast Combo)**:
- 在平A即将命中的瞬间按Q，取消平A后摇
- 在Q后点击地面取消Q的后摇
- 反复执行：平A → Q(取消后摇) → 点击地面 → 平A → Q...

### 进阶连招

**瞬间爆发**:
- E → R（利用R取消E的后摇，同时开启大招）
- 紧接着：Q → 平A → Q → 平A → Q → 平A
- W眩晕 → 平A × 2
- R重施（Wind Slash）收割

**E+R取消动画**:
- 正常情况下，锐雯开启R后会站立约0.5秒来forge her broken blade
- 如果在按R之后立即按E，可以取消这0.5秒的动画
- 这让敌人没有反应时间

**不可见Wind Slash**:
- E → R → 立即重施R
- Wind Slash的施法动画会被E的位移取消掉
- 用于敌人即将交闪现逃跑时

### 动画取消机制

锐雯的每个技能都有"施法后摇"(Cast Backswing)，可以通过以下方式取消：

1. **平A取消Q后摇**: 在平A动画即将结束时按Q
2. **点击地面取消Q后摇**: Q之后立即点击地面
3. **Q取消平A后摇**: 在平A造成伤害后立即按Q
4. **R+E取消R前摇**: 在R动画过程中使用E
5. **E取消R后摇**: R之后立即E来取消forge动画

**Q取消平A后摇详解**:
- 锐雯的平A有完整的施法动画和后摇
- 在平A即将造成伤害的瞬间按Q
- 平A伤害会正常结算，但动画会被Q替换
- 这个技巧可以让每次Q之间多打一次平A
- 理论上可以造成约30%更多的伤害

### 团战进场

**E+R+Flash进场**:
- E → R → 立即F到敌人身边
- 平A → W → Q → 平A
- 等待R重施冷却，用R+Q收割

## 隐藏机制

1. **Q第三段穿墙**: Q的第三次跃起可以穿越地形，但碰撞半径需要足够通过
2. **Q目标选取**: 优先以鼠标悬停的单位为目标
3. **E位移距离**: E的实际位移距离约为250，但护盾生效范围略大
4. **R后普攻距离**: R提供的攻击距离加成也作用于技能伤害
5. **W眩晕范围**: W是以锐雯为中心的圆形的范围伤害
6. **被动充能机制**: 每次施放技能充能一层，3层满时平A伤害最高
7. **Q可以重置平A**: 在平A动画过程中按Q可以取消平A并立即冲刺
8. **Q不受韧性影响**: Q的第三段击飞不受韧性减少（韧性只影响控制时长）
9. **E护盾类型**: E获得的护盾是自适应护盾，根据伤害类型（物理/魔法）提供对应抗性
10. **R Wind Slash判定**: Wind Slash可以击中R期间所有受到过锐雯技能伤害的目标

## 常见问题

**Q: 锐雯的Fast Q是什么？**
A: Fast Q是一种利用动画取消的技术，在每次Q之间插入平A，利用Q来取消平A的后摇，从而在短时间内打出更多伤害。

**Q: 为什么E+R可以取消动画？**
A: 因为R有约0.5秒的forge动画，而E的冲刺可以在这段时间内发生，等效取消了R的站立动画。

**Q: 锐雯的Q第三段可以穿哪些墙？**
A: 理论上所有薄墙都可以，但实际判定取决于地形厚度和碰撞半径，需要实战练习。
