# Syndra (辛德拉)

## 基础信息

- **称号**: 辛德拉 (The Dark Sovereign)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师
- **位置**: 中单

## 技能介绍

### 被动 - 超越 (Transcendent)

**机制**: Syndra收集愤怒碎片，增强技能效果。

**怒气上限**: 120层
**满层效果**: +15% 技能伤害

### Q - 黑暗法球 (Dark Sphere)

- **冷却**: 7 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 800

Syndra召唤黑暗法球，造成伤害。

**伤害**: 80/115/150/185/220 (+ 65% AP)
**法球持续**: 6秒

### W - 意志之力 (Force of Will)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 925/950/500

Syndra抓取敌人或法球并投掷。

**伤害**: 70/100/130/160/190 (+ 65% AP)
**减速**: 25% (持续1.5秒)
**抓取距离**: 500单位

### E - 散射 (Scatter the Weak)

- **冷却**: 15 秒
- **消耗**: 50 法力
- **范围**: 700/800/1300

Syndra推送敌人和法球。

**伤害**: 60/95/130/165/200 (+ 60% AP)
**击退距离**: 400-800单位
**眩晕时间**: 1.25秒

### R - 释放力量 (Unleashed Power)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 675

Syndra释放所有法球攻击敌人。

**每球伤害**: 80/120/160 (+ 20% AP)
**最低伤害**: 240/360/480 (+ 60% AP)
**目标数量**: 最多7个法球

## 连招技巧

### 基础连招

**对线连招**:
- Q(黑暗法球) → E(散射) → W(意志之力)
- 法球控制

**消耗连招**:
- Q(黑暗法球) → W(意志之力) → E(散射)
- 远程消耗

### 进阶连招

**团战连招**:
- R(释放力量) → Q(黑暗法球) → E(散射) → W(意志之力)
- 大招收割

**单杀连招**:
- Q(黑暗法球) → W(意志之力) → R(释放力量) → E(散射)
- 爆发输出

## 隐藏机制

1. **法球机制**: Q可以放置多个法球
2. **W抓取**: W可以抓取法球和敌人
3. **E击退**: E可以推送法球造成二次伤害
4. **R收集**: R自动收集周围法球
5. **被动增强**: 怒气满层时技能增强
