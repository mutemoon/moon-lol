#  Fiddlesticks (费德提克)

## 基础信息

- **称号**: 费德提克 (The Ancient Fear)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、辅助
- **位置**: 打野

## 技能介绍

### 被动 - 惊讶稻草人 (A Harmless Scarecrow)

**机制**: Fiddlesticks开始游戏时拥有独特的惊讶稻草人。可以伪装成稻草人获得伪装效果。

**伪装效果**: 6级后，放置稻草人时会召唤一个扫描器

### Q - 恐惧 (Terrify)

- **冷却**: 15/14.5/14/13.5/13 秒
- **消耗**: 65 法力
- **范围**: 575

Fiddlesticks向目标发射乌鸦，造成魔法伤害并恐惧敌人。如果Fiddlesticks未被看见，恐惧时间延长。

**伤害**: 4-6% (+ 3% AP) 目标当前生命值
**恐惧时间**: 1.2-2 秒

### W - 丰收祝福 (Bountiful Harvest)

- **冷却**: 10/9.5/9/8.5/8 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 650

Fiddlesticks与附近敌人建立锁链，吸取他们的灵魂。持续造成伤害并治疗自己。

**每秒伤害**: 60/90/120/150/180 (+ 45% AP)
**治疗量**: 25-55% 伤害值

### E - 收割 (Reap)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 850

Fiddlesticks用镰刀劈砍目标区域，对敌人造成魔法伤害并减速。中心敌人被沉默。

**伤害**: 70/105/140/175/210 (+ 50% AP)
**减速**: 30-50%
**沉默时间**: 1.25秒

### R - 乌鸦风暴 (Crowstorm)

- **冷却**: 140/110/80 秒
- **消耗**: 100 法力
- **范围**: 800

Fiddlesticks引导1.5秒后，闪烁到目标位置，周围乌鸦狂舞5秒，对附近敌人造成持续伤害。

**每跳伤害**: 37.5/62.5/87.5 (+ 12.5% AP)

## 连招技巧

### 基础连招

**标准团控**:
- R(乌鸦风暴) → Q(恐惧) → E(收割) → W(丰收祝福)
- R先手，Q控制

**消耗连招**:
- Q(恐惧) → W(丰收祝福) → E(收割)
- Q先手恐惧

### 进阶连招

**团战连招**:
- R(乌鸦风暴) → Q(恐惧) → W(丰收祝福) → E(收割)
- R大范围持续伤害

**埋伏连招**:
- Q(恐惧) → R(乌鸦风暴) → W(丰收祝福)
- 伪装后先手

---

## 隐藏机制

1. **伪装**: Fiddlesticks可以伪装成稻草人
2. **W治疗**: W吸取敌人时治疗自己
3. **Q恐惧延长**: 未被看见时Q恐惧时间更长
4. **E沉默**: E命中中心敌人时造成沉默
5. **R持续伤害**: R的乌鸦风暴持续5秒
