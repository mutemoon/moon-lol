#  Kennen (凯南)

## 基础信息

- **称号**: 凯南 (The Heart of the Tempest)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师
- **位置**: 上单

## 技能介绍

### 被动 - 风暴印记 (Mark of the Storm)

**机制**: Kennen的技能对敌人施加印记。达到3层时眩晕敌人并返还能量。

**眩晕时间**: 1.25秒
**能量返还**: 25

### Q - 雷霆之怒 (Thundering Shuriken)

- **冷却**: 7/6.25/5.5/4.75/4 秒
- **消耗**: 60/55/50/45/40 能量
- **范围**: 1050

Kennen向目标方向投掷手里剑。

**伤害**: 75/125/175/225/275 (+ 75% AP)

### W - 电流涌动 (Electrical Surge)

- **冷却**: 13/11.25/9.5/7.75/6 秒
- **消耗**: 40 能量
- **范围**: 775

Kennen的普攻叠加电涌，满层时下次攻击造成额外伤害。

**额外伤害**: 35/45/55/65/75 (+ 80-120% bonus AD) (+ 35% AP)
**主动伤害**: 70/95/120/145/170 (+ 80% AP)

### E - 闪电冲刺 (Lightning Rush)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 80 能量
- **范围**: 200

Kennen变成闪电，获得移速加成和免疫。

**伤害**: 80/120/160/200/240 (+ 80% AP)
**移速加成**: 100%
**攻速加成**: 40-80%

### R - 风暴龙卷风 (Slicing Maelstrom)

- **冷却**: 120 秒

Kennen召唤风暴，每0.5秒降下闪电。

**伤害**: 40/75/110 (+ 22.5% AP) 每道
**总伤害**: 300/562.5/825 (+ 168.75% AP)
**双抗加成**: 20/40/60

## 连招技巧

### 基础连招

**标准消耗**:
- Q(雷霆之怒) → W(电流涌动) → E(闪电冲刺)
- 手里剑起手，电涌追击

**控制连招**:
- E(闪电冲刺) → R(风暴龙卷风) → Q(雷霆之怒)
- 冲刺进场，大招控场

### 进阶连招

**团战连招**:
- E(闪电冲刺) → R(风暴龙卷风) → W(电流涌动)
- 进场大范围控制

**追杀连招**:
- Q(雷霆之怒) → E(闪电冲刺) → W(电流涌动)
- 持续输出

## 隐藏机制

1. **被动眩晕**: 3层印记眩晕敌人
2. **E免疫**: E提供幽灵状态
3. **R范围伤害**: R持续3秒，每0.5秒造成伤害
4. **W电涌**: W被动可以强化普攻
5. **能量机制**: Kennen使用能量条而非法力
