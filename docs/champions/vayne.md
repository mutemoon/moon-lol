# 薇恩 (Vayne)

## 基础信息

- **称号**: 暗夜猎手 (The Night Hunter)
- **类型**: 物理 / 远程
- **难度**: 3/3
- **角色**: 射手、刺客
- **位置**: ADC

## 技能介绍

### 被动 - 暗夜猎手 (Night Hunter)

**机制**: 薇恩面向附近可见敌方英雄时获得额外移速。

**移速加成**: 30（在R期间为90）

### Q - 翻滚 (Tumble)

- **冷却**: 6/5/4/3/2 秒
- **消耗**: 30 法力
- **范围**: 300

薇恩向目标方向位移一小段距离，强化下一次普攻。

**强化伤害**: 75/85/95/105/115% AD (+ 50% AP)

**进化效果（R期间）**:
- Q冷却减少30/40/50%
- Q位移时获得1秒隐身

### W - 银弹 (Silver Bolts)

**被动**: 薇恩的普攻和E技能施加"银弹"标记（最多3层）。

- 第3层触发时造成基于目标最大生命值的真实伤害
**伤害**: 6/7/8/9/10% 目标最大生命值
**最小伤害**: 50/65/80/95/110

### E - 恶魔审判 (Condemn)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 90 法力
- **范围**: 550

薇恩发射一枚巨型螺栓，对目标造成伤害并将其击退。如果目标撞击地形会受到额外伤害并眩晕。

**伤害**: 50/85/120/155/190 (+ 50% bonus AD)
**撞击地形伤害**: 75/127.5/180/232.5/285 (+ 75% bonus AD)
**眩晕时间**: 1.5秒

### R - 终极时刻 (Final Hour)

- **冷却**: 100/85/70 秒
- **消耗**: 80 法力
- **持续时间**: 8/10/12 秒

薇恩装上弩箭，获得额外攻击力。

**额外攻击力**: 35/50/65

**强化效果**:
- 被动移速加成从30提升至90
- Q冷却减少30/40/50%
- Q位移时获得1秒隐身

**助攻延长**: 击杀敌方英雄后R持续时间延长4秒

## 连招技巧

### 基础连招

**标准消耗**:
- Q(翻滚) → 普攻 → E(恶魔审判) → 普攻
- Q接近目标，E将其钉墙

**追击连招**:
- 普攻 × 2 → Q(翻滚) → 普攻 × 3
- 利用Q重置普攻追击

### 进阶连招

**钉墙连招**:
- Q(翻滚) → 普攻 → E(恶魔审判) → 墙壁 → 普攻 × 2 → Q
- E将敌人钉在墙上触发额外伤害

**爆发连招**:
- R(终极时刻) → Q(翻滚) → 普攻 → E(恶魔审判) → Q → 普攻
- R期间无限隐身和Q冷却减少

**团战收割**:
- R(终极时刻) → Q(翻滚) → 普攻 × 2 → E(恶魔审判)
- 利用R的高机动性追杀脆皮

## 隐藏机制

1. **Q重置普攻**: Q可以重置普攻计时器
2. **E钉墙**: E可以将敌人击退到地形上眩晕
3. **W真实伤害**: W的第三层造成真实伤害，无视护甲
4. **R移速**: R期间被动移速加成翻三倍
5. **Q隐身**: R期间Q位移获得隐身
6. **W层数**: W层数最多3层，需要叠加
7. **E控制**: E命中后会击退目标
