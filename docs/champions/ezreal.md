# 伊泽瑞尔 (Ezreal)

## 基础信息

- **称号**: 探险家 (The Prodigal Explorer)
- **类型**: 物理 / 远程
- **难度**: 2/3
- **角色**: 射手
- **位置**: ADC

## 技能介绍

### 被动 - 咒能高涨 (Rising Spell Force)

**机制**: 伊泽瑞尔的技能每命中一个敌人获得一层"咒能高涨"（持续6秒），最多5层。

**每层效果**:
- 10% bonus attack speed
- 最高50% bonus attack speed

### Q - 秘术射击 (Mystic Shot)

- **冷却**: 5.5/5.25/5/4.75/4.5 秒
- **消耗**: 28/31/34/37/40 法力
- **范围**: 1200

伊泽瑞尔朝目标方向发射能量箭矢，造成物理伤害，触发攻击特效。

**伤害**: 20/45/70/95/120 (+ 130% AD)(+ 15% AP)

**额外效果**:
- 命中敌人时减少所有技能1.5秒冷却

### W - 精华跃动 (Essence Flux)

- **冷却**: 8 秒
- **消耗**: 50 法力
- **范围**: 1200

伊泽瑞尔发射一个能量球，标记命中的第一个敌方英雄、野怪或建筑4秒。下一次对该目标的普攻或技能会引爆标记。

**引爆伤害**: 80/135/190/245/300 (+ 100% bonus AD)(+ 70/75/80/85/90% AP)

**额外效果**:
- 用技能引爆时，返还该技能蓝耗+60法力

### E - 奥术跃迁 (Arcane Shift)

- **冷却**: 26/23/20/17/14 秒
- **消耗**: 70 法力
- **范围**: 475

伊泽瑞尔闪烁到目标位置，然后向最近的敌人发射追踪弹。

**伤害**: 80/130/180/230/280 (+ 60% bonus AD)(+ 75% AP)

**额外效果**:
- 优先攻击被E标记的敌人
- 可以命中不可见的敌人

### R - 精准弹幕 (Trueshot Barrage)

- **冷却**: 120/105/90 秒
- **消耗**: 100 法力
- **范围**: 全地图

伊泽瑞尔朝目标方向发射强大的能量弧，对命中的敌人造成魔法伤害。

**伤害**: 350/550/750 (+ 100% bonus AD)(+ 90% AP)
**小兵/野怪伤害**: 175/275/375 (+ 50% bonus AD)(+ 45% AP)

## 连招技巧

### 基础连招

**标准消耗**:
- Q(秘术射击) → W(精华跃动) → E(奥术跃迁)
- Q起手减少冷却，W标记目标，E追击

**爆发连招**:
- W(精华跃动) → Q(秘术射击) → 普攻 → E
- W先手标记，Q触发标记伤害

### 进阶连招

**追杀连招**:
- E(奥术跃迁) → W(精华跃动) → Q(秘术射击) → 普攻
- E接近目标，W标记后Q引爆

**团战收割**:
- R(精准弹幕) → Q(秘术射击) → W(精华跃动)
- R全图伤害后接Q减少冷却继续输出

**远距离标记**:
- W(精华跃动) → R(精准弹幕) → Q引爆
- W标记后R可以对多个目标造成伤害

## 隐藏机制

1. **Q减少冷却**: Q命中敌人时减少所有技能1.5秒冷却
2. **W标记机制**: W标记可以被任何技能或普攻引爆
3. **E优先目标**: E会自动寻找被W标记的目标
4. **R全图施放**: R可以施放在全地图范围内
5. **被动叠加**: 被动最多5层，提供50%攻速
6. **E可以穿墙**: E可以穿越薄墙到达目标位置
