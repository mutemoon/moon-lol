#  Amumu (阿木木)

## 基础信息

- **称号**: 阿木木 (The Sad Mummy)
- **类型**: 魔法 / 近战
- **难度**: 1/3
- **角色**: 坦克、辅助
- **位置**: 打野、辅助

## 技能介绍

### 被动 - 诅咒之触 (Cursed Touch)

**机制**: Amumu的普攻和技能对敌人施加"诅咒"效果3秒。诅咒目标受到所有魔法伤害的10%额外真实伤害。

### Q - 绷带射击 (Bandage Toss)

- **冷却**: 16/15/14/13/12 秒 (可充能2次)
- **消耗**: 50 法力
- **范围**: 1100

Amumu向目标方向投掷绷带，对第一个敌人造成魔法伤害、眩晕1秒并将自己拉向目标。

**伤害**: 70/95/120/145/170 (+ 85% AP)

### W - 绝望之念 (Despair)

- **冷却**: 1 秒 (开关技能)
- **消耗**: 8 法力/秒
- **范围**: 350

Amumu哭泣产生眼泪池，每0.5秒对附近敌人造成魔法伤害。

**每跳伤害**: 5 (+ 1% 目标最大生命值) (+ 0.25% AP)

### E - 悲伤木乃伊 (Tantrum)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 35 法力
- **范围**: 350

被动：Amumu减少所有承受的物理伤害（最多50%）。
主动：对附近敌人造成魔法伤害。每次被普攻命中减少E冷却0.75秒。

**伤害**: 65/95/125/155/185 (+ 50% AP)

### R - 木乃伊之咒 (Curse of the Sad Mummy)

- **冷却**: 150/125/100 秒
- **消耗**: 100/150/200 法力
- **范围**: 550

Amumu诅咒并束缚附近敌人，造成魔法伤害、击飞和眩晕1.5秒。

**伤害**: 200/300/400 (+ 80% AP)

## 连招技巧

### 基础连招

**标准团控**:
- Q(绷带射击) → R(木乃伊之咒) → W(绝望之念) → E(悲伤木乃伊)
- Q先手眩晕，R大范围控制

**消耗连招**:
- Q(绷带射击) → E(悲伤木乃伊) → W(绝望之念)
- Q接近目标，E造成伤害

### 进阶连招

**团战连招**:
- R(木乃伊之咒) → Q(绷带射击) → W(绝望之念) → E(悲伤木乃伊)
- R先手大范围控制

**蹲草连招**:
- W(绝望之念) → Q(绷带射击) → R(木乃伊之咒)
- 草丛先开W持续伤害

## 隐藏机制

1. **诅咒增伤**: 被动使目标受到10%额外真实伤害
2. **Q充能**: Q有2层充能，可以连续使用两次
3. **E减伤**: E被动减少物理伤害，是Amumu坦度的关键
4. **W消耗**: W持续消耗法力但伤害可观
5. **R范围**: R的范围比指示器显示的更大
