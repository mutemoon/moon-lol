# 女枪 (Miss Fortune)

## 基础信息

- **称号**: 赏金猎人 (The Bounty Hunter)
- **类型**: 物理 / 远程
- **难度**: 1/3
- **角色**: 射手
- **位置**: ADC

## 技能介绍

### 被动 - 轻挑 (Love Tap)

**机制**: 厄运小姐的普攻会标记敌人。攻击新目标时造成50-100% AD额外物理伤害（基于等级）。

- 对小兵的额外伤害减半

### Q - 一箭双雕 (Double Up)

- **冷却**: 7/6/5/4/3 秒
- **消耗**: 40 法力
- **范围**: 550

厄运小姐向目标射击，子弹会弹射到目标身后的敌人。

**伤害**: 20/45/70/95/120 (+ 100% AD)(+ 35% AP)
**弹射暴击**: 如果Q击杀主要目标，弹射会暴击

### W - 大步流星 (Strut)

- **冷却**: 12 秒
- **消耗**: 45 法力

**被动**: 厄运小姐在4秒不受伤害后获得移速加成，3秒后移速加成翻倍。

**移速加成**: 30/35/40/45/50 → 60/70/80/90/100

**主动**: 厄运小姐获得攻速加成（持续4秒）。

**攻速加成**: 40/55/70/85/100%

### E - 弹射 (Make It Rain)

- **冷却**: 18/17/16/15/14 秒
- **消耗**: 80 法力
- **范围**: 1000

厄运小姐在目标区域召唤弹雨，持续2秒，减速并伤害区域内敌人。

**每跳伤害**: 8.75/12.5/16.25/20/23.75 (+ 15% AP)
**减速**: 40% (+ 6% per 100 AP)

### R - 弹幕时间 (Bullet Time)

- **冷却**: 120/110/100 秒
- **消耗**: 100 法力
- **范围**: 1450

厄运小姐引导最多3秒，向前方发射多波弹幕。

**每波伤害**: 20/30/40 (+ 60% AD)(+ 25% AP)
**波数**: 14/16/18
**暴击**: 每波可以暴击（130% + 9% 伤害）

## 连招技巧

### 基础连招

**标准消耗**:
- Q(一箭双雕) → 普攻 → W(大步流星) → 普攻
- Q起手，W提供攻速加成

**清线连招**:
- E(弹射) → Q(一箭双雕) → 普攻
- E先手减速，Q收割

### 进阶连招

**爆发连招**:
- Q(一箭双雕) → 普攻 → R(弹幕时间) → W
- Q起手标记，R引导弹幕

**团战连招**:
- R(弹幕时间) → E(弹射) → Q(一箭双雕) → W
- R先手打出大量伤害

**追击连招**:
- W(大步流星) → Q(一箭双雕) → 普攻
- W的高移速追击敌人

## 隐藏机制

1. **被动标记**: 被动可以触发攻击特效，如贪欲猎手
2. **Q弹射**: Q可以弹射到目标身后的敌人，优先弹向直接位于目标后的敌人
3. **W移速**: W的被动移速在脱战后获得，击杀后获得翻倍移速
4. **R引导**: R可以引导最多3秒，引导期间可以被打断
5. **E减速**: E的减速效果随AP增加
6. **被动伤害**: 被动对英雄和野怪伤害更高，对小兵伤害减半
