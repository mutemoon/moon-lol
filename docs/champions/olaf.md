# 奥拉夫 (Olaf)

## 基础信息

- **称号**: 狂战士 (the Berserker)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、坦克
- **位置**: 打野、上单

## 技能介绍

### 被动 - 狂战之怒 (Berserker Rage)

奥拉夫每次攻击获得攻速加成，最多叠加一定层数。
- 攻速加成: 2/4/6/8%（每层，基于等级）
- 最大层数: 5层

**隐藏机制**:
- 被动提供的攻速在早期清野很有帮助
- 叠加层数需要持续攻击

### Q - 逆流 (Undertow)

- **冷却**: 9 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 1000

奥拉夫将斧头投掷到目标位置，对穿过的敌人造成物理伤害，并减少他们的护甲和移动速度。如果奥拉夫捡起斧头，技能冷却会重置。
- 斧头会在目标位置停留4秒
- 捡起斧头重置Q冷却

**效果数值**:
- 伤害: 70/115/160/205/250 (+0.7 AD)
- 护甲减少: 20/25/30/35/40%，持续2秒
- 减速: 30/35/40/45/50%，持续2秒

**隐藏机制**:
- Q可以穿透小兵
- 捡起斧头重置冷却是奥拉夫的核心机制
- Q配合W的攻速加成可以快速捡斧头
- Q可以用来消耗和推线

### W - 残暴打击 (Tough It Out)

- **冷却**: 16/15/14/13/12 秒
- **消耗**: 30 法力
- **范围**: 700

奥拉夫增加攻击速度，减少受到伤害，并获得护盾。
- 持续6秒

**效果数值**:
- 攻击速度: 25/35/45/55/65%
- 伤害减免: 10/15/20/25/30%
- 护盾值: 60/100/140/180/220 (+0.4 AD)

**隐藏机制**:
- W提供的伤害减免和护盾让奥拉夫可以硬抗伤害
- 攻速加成配合Q可以快速捡斧头
- W护盾可以用来挡技能

### E - 鲁莽挥击 (Reckless Swing)

- **冷却**: 11/10/9/8/7 秒
- **消耗**: 0
- **范围**: 325

奥拉夫以极大的力量攻击，对目标造成真实伤害，也会对自己造成伤害。如果他击杀目标，返还生命值消耗。
- 自残伤害：30/35/40/45/50 (+0.25 AD)
- 击杀返还: 生命消耗的100%

**效果数值**:
- 伤害: 70/115/160/205/250 (+0.4 AD)
- 自残: 30/35/40/45/50 (+0.25 AD)

**隐藏机制**:
- E是真实伤害，可以用来击杀低血量目标
- E的自残可以被W的护盾抵消
- E配合R的无敌可以无视自残
- E可以用来补伤害

### R - 诸神黄昏 (Ragnarok)

- **冷却**: 100/90/80 秒
- **消耗**: 0
- **范围**: 400

奥拉夫被动地获得额外护甲和魔法抗性。他可以激活此技能，在持续攻击期间免疫所有控制效果。
- 被动：护甲和魔抗加成
- 主动：免疫控制，持续6秒

**效果数值**:
- 被动护甲: 30/45/60
- 被动魔抗: 30/45/60
- 免疫控制: 持续6秒（每次攻击延长0.5秒，最多12秒）

**隐藏机制**:
- R提供的双抗让奥拉夫更肉
- R的免疫控制是奥拉夫的核心机制
- R期间每次攻击延长无敌时间
- R配合E可以无视自残
- R可以用来追击或逃跑

## 连招技巧

### 基础连招

**Gank连招**:
- Q起手投斧 → 捡起重置冷却 → W → E真实伤害

**对线消耗**:
- Q投斧 → 减速敌人 → 平A → W → E

**追击连招**:
- Q投斧 → 捡起重置 → 继续Q → W追击

### 进阶连招

**QR+E**:
- Q投斧 → R免疫控制 → 平A追击 → E补伤害

**RW+E**:
- R免疫控制 → W护盾 → E真实伤害

**闪现+E**:
- 闪现接近 → E真实伤害 → Q追击

**WQ+AE**:
- W护盾 → Q投斧 → 平A → E补伤害

### 团战进场

**切后排连招**:
- Q投斧减速 → R免疫控制 → W护盾 → 平A → E

**反手连招**:
- R免疫控制 → W护盾 → E反击

**团战搅局**:
- R进入战场 → 利用免疫控制追击后排

## 隐藏机制

1. **Q重置冷却**: 捡起斧头重置Q冷却
2. **W伤害减免**: W提供的伤害减免可以抵消E的自残
3. **E真实伤害**: E是真实伤害，无视护甲和护盾
4. **R免疫控制**: R可以让奥拉夫无视所有控制效果
5. **R延长机制**: R期间每次攻击延长无敌时间
6. **被动攻速叠加**: 被动攻速最多叠加5层
7. **Q减速叠加**: Q的减速可以叠加
8. **E击杀返还**: E击杀目标返还自残的生命消耗
9. **R双抗加成**: R被动提供的双抗在早期很显著
10. **Q穿兵**: Q可以穿透小兵

## 常见问题

**Q: 奥拉夫如何快速清野？**
A: 利用Q的范围伤害和W的攻速加成，可以快速清理野怪。

**Q: 奥拉夫的核心技巧是什么？**
A: Q捡斧头重置冷却是核心，需要练习走位。

**Q: 奥拉夫如何打团？**
A: 利用R免疫控制切入后排，Q减速配合E真实伤害输出。
