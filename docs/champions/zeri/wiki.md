# Zeri (泽丽)

## 基础信息

- **称号**: 泽丽 (The Spark of Zaun)
- **类型**: 物理 / 远程
- **难度**: 2/3
- **角色**: 射手
- **位置**: ADC

## 技能介绍

### 被动 - 活体电池 (Living Battery)

**机制**: Zeri移动和释放技能获得充能。

**充能上限**: 100
**普攻伤害**: 10-27 (+ 3% AP)
**满充能伤害**: 75-170 (+ 110% AP)

### Q - 爆发火花 (Burst Fire)

- **冷却**: 攻击速度
- **消耗**: 自助
- **范围**: 750

Zeri发射7发子弹。

**伤害**: 15-23 (+ 104-120% AD)
**充能获取**: 10点/每次施放

### W - 超冲击激光 (Ultrashock Laser)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 1200

Zeri发射冲击波。

**伤害**: 30/70/110/150/190 (+ 120% AD) (+ 25% AP)
**减速**: 30/35/40/45/50%
**地形碰撞**: 变成激光

### E - 闪电突袭 (Spark Surge)

- **冷却**: 22/21/20/19/18 秒
- **消耗**: 90/85/80/75/70 法力
- **范围**: 300

Zeri向目标方向冲刺。

**位移距离**: 300+
**穿墙**: 可以穿越地形
**强化效果**: 强化Q穿透敌人

### R - 闪电冲击 (Lightning Crash)

- **冷却**: 80/75/70 秒
- **消耗**: 100 法力
- **范围**: 825

Zeri释放电能。

**伤害**: 150/250/350 (+ 100% 额外AD) (+ 110% AP)
**过载效果**: 攻速和移速加成
**链式闪电**: 伤害连锁

## 连招技巧

### 基础连招

**对线连招**:
- Q(爆发火花) → W(超冲击激光) → E(闪电突袭)
- 技能连招

**消耗连招**:
- W(超冲击激光) → Q(爆发火花) → E(闪电突袭)
- 位移消耗

### 进阶连招

**团战连招**:
- R(闪电冲击) → Q(爆发火花) → W(超冲击激光) → E(闪电突袭)
- 大招过载

**追击连招**:
- E(闪电突袭) → Q(爆发火花) → R(闪电冲击)
- 位移追击

## 隐藏机制

1. **被动充能**: 移动和施放技能获得充能
2. **Q充能**: Q消耗充能
3. **W激光**: W命中地形变成激光
4. **E穿墙**: E可以穿越地形
5. **R过载**: R激活后攻速移速加成
