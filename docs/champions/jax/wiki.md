#  Jax (贾克斯)

## 基础信息

- **称号**: 武器大师 (Grandmaster at Arms)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士
- **位置**: 上单、打野
- **资源**: 法力
- **攻击距离**: 125

## 技能介绍

### 被动 - 无尽攻击 (Relentless Assault)

**类型**: 触发型被动 / 环境型被动

Jax的普通攻击会在2秒内生成一层Relentless Assault堆叠，每次后续攻击刷新持续时间，最多叠加8层。每层堆叠提供5%-14% (基于等级) 额外攻击速度，最高40%-112% (基于等级)。

**大师垂钓**: 当Jax在河道中脱离战斗且闲置10秒时，他会偶尔钓到鱼，授予金币和5秒内1点法强，平均每15秒钓到一条鱼。也有5%概率钓到稀有鱼，授予金币和10点法强，持续5秒。

### Q - 跳斩 (Leap Strike)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 65 法力
- **施法时间**: 无
- **目标范围**: 700

Jax向目标单位位置突进。如果目标是敌人且到达时在范围内，Jax对其造成物理伤害。

**物理伤害**: 65/105/145/185/225 (+ 100% bonus AD)

Jax可以在突进期间施放任何技能。

### W - 强化 (Empower)

- **冷却**: 7/6/5/4/3 秒
- **消耗**: 30 法力
- **施法时间**: 无

Jax强化他的下次普通攻击或Leap Strike，在10秒内对范围内敌人造成额外魔法伤害，对建筑物减少50%。

如果Empower用于普通攻击，将获得50额外攻击距离，并有不可取消的前摇。

**额外魔法伤害**: 50/85/120/155/190 (+ 60% AP)

Empower重置Jax的普通攻击计时器。

### E - 反击风暴 (Counter Strike)

- **冷却**: 17/15/13/11/9 秒
- **消耗**: 50/60/70/80/90 法力
- **施法时间**: 无
- **效果范围**: 375

**主动**: Jax进入闪避状态2秒：一种防御姿态，使其闪避所有即将到来的非防御塔普通攻击，并受到所有来自英雄的范围技能伤害减少25%。

Counter Strike可以在1秒后再次施放，持续时间结束后自动施放。

**再次施放**: Jax对附近敌人造成魔法伤害，每次闪避的攻击增加20%总伤害，上限100%，并眩晕1秒。

**最小魔法伤害**: 40/70/100/130/160 (+ 70% AP) (+ 目标最大生命值的3%)

### R - 宗师之威 (Grandmaster-at-Arms)

- **冷却**: 110/100/90 秒
- **消耗**: 100 法力
- **施法时间**: 0.25秒
- **效果范围**: 375

**被动**: Jax的普通攻击会生成一层Grandmaster-at-Arms命中效果，在2秒内持续，每下次命中刷新叠加，最多2层。达到2层时，他的下次普通攻击的命中效果会获得不可取消的前摇，并消耗所有层数来造成额外魔法伤害，对建筑物减少50%。当Grandmaster-at-Arms激活时，激怒攻击在1层时触发。

**额外魔法伤害**: 75/130/185 (+ 60% AP)

**主动**: Jax挥动灯笼，对附近敌人造成魔法伤害。如果命中英雄，他获得护甲，每超过第一个英雄增加护甲，并获得等于护甲60%的魔法抗性，以及8秒内10%增加的体型。

**魔法伤害**: 100/175/250 (+ 100% AP)
**护甲加成**: 45/60/75 (+ 40% bonus AD)
**每个命中英雄额外护甲**: (See in-game)

## 连招技巧

### 基础连招

**消耗连招**:
- Q跳向敌人 → W → 普通攻击 → E → 拉开
- 利用E的闪避来躲避敌方反打

**击杀连招**:
- Q → W → 普通攻击 → E → 普通攻击 → R → W → 普通攻击

### 进阶连招

**E闪避反打**:
- E开启 → 敌方交关键技能 → 闪避 → 眩晕 → Q → W → 普通攻击

**跳眼穿越**:
- 在墙边插眼 → Q跳眼穿越地形
- 可以用来追击或逃跑

**R+E取消前摇**:
- R → E可以在R动画过程中开启E
- E结束后眩晕接R的范围伤害

### 团战进场

**切后排连招**:
- Q跳向敌方后排 → E闪避 → W → 普通攻击 → R
- 利用E的无敌帧躲过前排技能

**反手连招**:
- 敌方先手 → E闪避 → 眩晕 → R → Q → W → 普通攻击

## 隐藏机制

1. **Q跳眼位移**: Q可以跳向眼位，包括敌方眼，这使得贾克斯可以穿越地形
2. **E闪避优先级**: E的闪避可以躲掉很多技能，如船长Q、VN普通攻击等
3. **W重置普通攻击**: W可以取消普通攻击后摇，配合Q和E可以快速打出伤害
4. **R被动触发**: R的被动额外伤害每第三次攻击触发，包括技能后的普通攻击
5. **Q+E无解追人**: E结束后配合Q可以追杀到天涯海角
6. **提亚玛特取消E后摇**: E结束后使用提亚玛特或贪欲九头蛇可以取消后摇
7. **羊刀配合**: 贾克斯非常适合鬼索的凶刃，被动叠加攻速后输出大幅提升
8. **E结束后自动攻击**: E结束后如果范围内有敌人会自动攻击
9. **大师垂钓**: Jax在河道闲置可以钓鱼获得金币和法强
