# 贾克斯 (Jax)

## 基础信息

- **称号**: 武器大师 (Grandmaster at Arms)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、刺客
- **位置**: 上单、打野

## 技能介绍

### 被动 - 武器专注 (Weapon of the Void)

贾克斯的第三次普通攻击会附加额外魔法伤害。
- 每第三次连续攻击造成额外魔法伤害

### Q - 跳斩 (Leap Strike)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 65 法力
- **范围**: 700

贾克斯跳向一个单位，如果是敌人则造成物理伤害。
- 可以跳向友军、小兵、守卫或敌人
- 跳向守卫可以用于插眼/拆眼的位移

**效果数值**:
- 伤害: 65/100/135/170/205 (+0.6 AD)

**隐藏机制**:
- Q可以跳向眼位，包括己方和敌方眼
- Q落地瞬间平A会被取消，如果目标在范围内
- Q可以用于穿越地形（跳眼）

### W - 强化 (Empower)

- **冷却**: 7/6/5/4/3 秒
- **消耗**: 30 法力
- **范围**: 300

贾克斯为武器充能，使下次攻击造成额外魔法伤害。
- 施放后重置平A
- 可以对防御塔使用

**效果数值**:
- 额外伤害: 40/75/110/145/180 (+0.6 AP)

**隐藏机制**:
- W可以重置平A，配合Q使用：Q → W → 平A
- W+E的组合：E闪避期间可以使用W
- W可以触发攻击特效

### E - 反击风暴 (Counter Strike)

- **冷却**: 15/13.5/12/10.5/9 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 300

贾克斯的战斗技巧使他能够闪避所有即将到来的攻击，持续一段时间，然后快速反击，眩晕周围所有敌人。
- 持续1.2秒的闪避
- 结束时对周围敌人造成物理伤害
- 最多眩晕1秒的敌人

**效果数值**:
- 闪避持续时间: 1.2 秒
- 伤害: 50/85/120/155/190 (+0.5 AD)
- 眩晕时长: 0.75/1/1.25/1.5/1.75 秒

**隐藏机制**:
- E期间贾克斯闪避所有普通攻击和技能伤害
- E结束后立即对周围造成伤害
- 可以配合W重置平A使用：E → W → 平A
- E闪避不受韧性影响（闪避是无敌判定）
- E结束时如果附近没有敌人不会眩晕
- 配合提亚玛特/贪欲九头蛇可以取消E后摇

### R - 宗师之威 (Grandmaster-at-Arms)

- **冷却**: 100/90/80 秒
- **消耗**: 100 法力
- **范围**: 260

每第三次连续攻击造成额外魔法伤害。此外，贾克斯可以激活此技能，对周围造成伤害并增强决心，在短时间内增加护甲和魔法抗性。
- 被动：每第三次攻击附加额外魔法伤害
- 主动：范围伤害 + 护甲/魔抗加成

**效果数值**:
- 被动额外伤害: 100/160/220 (+0.7 AP)
- 主动伤害: 100/150/200 (+0.8 AD)
- 护甲加成: 30/50/70
- 魔抗加成: 30/50/70
- 持续时间: 3 秒

**隐藏机制**:
- R的被动额外伤害与E的眩晕可以配合
- R提供的双抗加成在早期非常显著
- R结束后可以继续追人或撤退
- R+E+W+平A的最大伤害连招

## 连招技巧

### 基础连招

**消耗连招**:
- Q跳向敌人 → W → 平A → E → 拉开
- 利用E的闪避来躲避敌方反打

**击杀连招**:
- Q → W → 平A → E → 平A → R → W → 平A

### 进阶连招

**E闪避反打**:
- E开启 → 敌方交关键技能 → 闪避 → 眩晕 → Q → W → 平A

**跳眼穿越**:
- 在墙边插眼 → Q跳眼穿越地形
- 可以用来追击或逃跑

**R+E取消前摇**:
- R → E可以在R动画过程中开启E
- E结束后眩晕接R的范围伤害

### 团战进场

**切后排连招**:
- Q跳向敌方后排 → E闪避 → W → 平A → R
- 利用E的无敌帧躲过前排技能

**反手连招**:
- 敌方先手 → E闪避 → 眩晕 → R → Q → W → 平A

## 隐藏机制

1. **Q跳眼位移**: Q可以跳向眼位，包括敌方眼，这使得贾克斯可以穿越地形
2. **E闪避优先级**: E的闪避可以躲掉很多技能，如船长Q、VN平A等
3. **W重置平A**: W可以取消平A后摇，配合Q和E可以快速打出伤害
4. **R被动触发**: R的被动额外伤害每第三次攻击触发，包括技能后的平A
5. **Q+E无解追人**: E结束后配合Q可以追杀到天涯海角
6. **提亚玛特取消E后摇**: E结束后使用提亚玛特或贪欲九头蛇可以取消后摇
7. **羊刀配合**: 贾克斯非常适合羊刀，被动叠加攻速后输出大幅提升
8. **E结束后自动攻击**: E结束后如果范围内有敌人会自动攻击

## 常见问题

**Q: 贾克斯如何对线远程？**
A: 利用Q跳小兵近身，E闪避关键技能，寻找反打机会。多兰盾出门，注意走位。

**Q: 贾克斯的核心装备是什么？**
A: 羊刀、破败、三相是核心。后续根据对面阵容补防御装。

**Q: 贾克斯的E如何使用？**
A: E的时机是关键。太早开会被敌方骗掉，太晚开可能被秒。新手建议等敌方交关键技能后再开。
