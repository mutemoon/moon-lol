# 蕾欧娜 (Leona)

## 基础信息

- **称号**: 黎明之刃 (The Radiant Dawn)
- **类型**: 物理 / 近战
- **难度**: 1/3
- **角色**: 坦克、辅助
- **位置**: 辅助

## 技能介绍

### 被动 - 阳光 (Sunlight)

**机制**: 蕾欧娜的技能会对命中的敌人施加"阳光"标记（持续2.5秒）。

- 友军对标记目标的伤害会触发额外魔法伤害
**额外伤害**: 32-165 (基于等级)

### Q - 日蚀 (Shield of Daybreak)

- **冷却**: 5 秒
- **消耗**: 30/35/40/45/50 法力

蕾欧娜强化下一次普攻，造成额外魔法伤害并眩晕目标1秒。

**额外伤害**: 10/35/60/85/110 (+ 30% AP)
**眩晕时间**: 1秒
**额外距离**: +50

### W - 日炎 (Eclipse)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 60 法力
- **范围**: 450

蕾欧娜举起盾牌3秒，获得伤害减免和双抗加成。持续时间结束后对周围敌人造成魔法伤害。

**伤害减免**: 最高50%，基于受到的伤害
**额外护甲**: 20/27.5/35/42.5/50 (+ 20% bonus armor)
**额外魔抗**: 20/27.5/35/42.5/50 (+ 20% bonus magic resistance)
**爆炸伤害**: 55/85/115/145/175 (+ 40% AP)

### E - 天顶之刃 (Zenith Blade)

- **冷却**: 12/10.5/9/7.5/6 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 900

蕾欧娜向目标方向投射剑刃，对命中的敌人造成魔法伤害。如果命中敌方英雄，她会冲向该英雄并将其定身0.5秒。

**伤害**: 50/90/130/170/210 (+ 40% AP)
**定身时间**: 0.5秒

### R - 太阳风暴 (Solar Flare)

- **冷却**: 90/75/60 秒
- **消耗**: 100 法力
- **范围**: 1200

蕾欧娜在目标位置召唤太阳风暴，0.625秒后落下，对区域内的敌人造成魔法伤害并减速。

**伤害**: 150/225/300 (+ 80% AP)
**减速**: 80% (持续1.75秒)
**中心命中**: 眩晕1.75秒

## 连招技巧

### 基础连招

**标准控制**:
- E(天顶之刃) → Q(日蚀) → W(日炎)
- E先手定身，Q眩晕，W承受伤害

**团战进场**:
- R(太阳风暴) → E(天顶之刃) → Q(日蚀) → W(日炎)
- R先手减速，E接近控制

### 进阶连招

**保护连招**:
- W(日炎) → E(天顶之刃) → Q(日蚀)
- W吸收伤害，E控制敌人

**配合ADC**:
- E(天顶之刃) → 普攻 → Q(日蚀) → W(日炎)
- E先手定身，触发被动额外伤害

---

## 隐藏机制

1. **被动标记**: 被动标记可以被友军的任何伤害触发
2. **W伤害减免**: W的伤害减免最高可达50%
3. **E位移**: E命中英雄后蕾欧娜会冲向目标
4. **Q强化普攻**: Q的强化普攻有不可取消的前摇
5. **R中心眩晕**: R的中心区域会造成眩晕而非减速
6. **W双抗加成**: W提供的双抗加成基于额外双抗计算
