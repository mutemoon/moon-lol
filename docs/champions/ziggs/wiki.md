# Ziggs (吉格斯)

## 基础信息

- **称号**: 吉格斯 (The Hexplosives Expert)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、炮台
- **位置**: ADC、中单

## 技能介绍

### 被动 - 短引线 (Short Fuse)

**机制**: Ziggs周期性强化普攻。

**伤害**: 20-184 (+ 50% AP)
**建筑加成**: +75%
**冷却减免**: 施放技能减少4-6秒

### Q - 弹跳炸弹 (Bouncing Bomb)

- **冷却**: 6/5.5/5/4.5/4 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 850/1400

Ziggs投掷弹跳炸弹。

**伤害**: 80/130/180/230/280 (+ 60-80% AP)
**弹跳次数**: 最多2次

### W - 炸药包 (Satchel Charge)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 80 法力
- **范围**: 1000

Ziggs放置炸药包。

**伤害**: 70/105/140/175/210 (+ 50% AP)
**击退效果**: 击退敌人
**自位移**: 可以位移自己

### E - hexplosive Minefield (Hexplosive Minefield)

- **冷却**: 16 秒
- **消耗**: 70/80/90/100/110 法力
- **范围**: 900

Ziggs放置11颗地雷。

**伤害**: 30/70/110/150/190 (+ 25-45% AP)
**减速**: 10/20/30/40/50%
**持续时间**: 10秒

### R - 超级炸弹 (Mega Inferno Bomb)

- **冷却**: 120/95/70 秒
- **消耗**: 100 法力
- **范围**: 5000

Ziggs发射超级炸弹。

**伤害**: 300/500/700 (+ 100% AP)
**外圈伤害**: 65%
**范围**: 525

## 连招技巧

### 基础连招

**对线连招**:
- Q(弹跳炸弹) → W(炸药包) → E(hexplosive Minefield)
- 范围消耗

**消耗连招**:
- E(hexplosive Minefield) → Q(弹跳炸弹) → W(炸药包)
- 地雷控制

### 进阶连招

**团战连招**:
- R(超级炸弹) → Q(弹跳炸弹) → W(炸药包) → E(hexplosive Minefield)
- 大招起手

**逃脱连招**:
- W(炸药包) → 炸药包爆炸 → 位移
- 击退自己

## 隐藏机制

1. **被动强化**: 被动周期性强化普攻
2. **Q弹跳**: Q可以弹跳2次
3. **W击退**: W可以击退敌人和自己
4. **E地雷**: E放置多个地雷
5. **R全图**: R可以攻击全图
