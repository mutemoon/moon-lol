# Tryndamere (泰达米尔)

## 基础信息

- **称号**: 泰达米尔 (The Barbarian King)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、刺客
- **位置**: 上单

## 技能介绍

### 被动 - 战斗愤怒 (Battle Fury)

**机制**: Tryndamere通过攻击产生怒气。

**怒气上限**: 100
**攻击怒气**: 5点
**暴击怒气**: 10点
**怒气消耗**: 8秒无伤害时每秒-5点
**怒气加成**: 0-50% 暴击率

### Q - 嗜血 (Bloodlust)

- **冷却**: 12 秒
- **消耗**: 100% 当前怒气
- **范围**: 自助

Tryndamere消耗怒气治疗自己。

**治疗量**: 30/40/50/60/70 (+ 30% AP)
**额外AD加成**: 最高20/35/50/65/80

### W - 嘲弄 (Mocking Shout)

- **冷却**: 14 秒
- **消耗**: 自助
- **范围**: 850

Tryndamere减少附近敌人攻击力。

**攻击力削减**: 20/35/50/65/80
**减速**: 30/35/40/45/50%

### E - 旋转劈砍 (Spinning Slash)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 自助
- **范围**: 660

Tryndamere冲刺并伤害敌人。

**伤害**: 80/120/160/200/240 (+ 100% 额外AD)
**怒气获取**: 2点(普通) / 5点(英雄)
**冷却减免**: 暴击减少0.75秒

### R - 不朽 (Undying Rage)

- **冷却**: 120/100/80 秒
- **消耗**: 自助
- **范围**: 自助

Tryndamere变得不可阻挡并保留最低生命值。

**怒气获取**: 50/75/100
**最低生命值**: 30/50/70%
**持续时间**: 5秒

## 连招技巧

### 基础连招

**对线连招**:
- E(旋转劈砍) → W(嘲弄) → Q(嗜血)
- 位移追击

**单杀连招**:
- E(旋转劈砍) → Q(嗜血) → W(嘲弄) → E(旋转劈砍)
- 消耗后击杀

### 进阶连招

**团战连招**:
- R(不朽) → E(旋转劈砍) → Q(嗜血) → W(嘲弄)
- 大招进入

**逃脱连招**:
- E(旋转劈砍) → R(不朽)
- 追击后反杀

---

## 隐藏机制

1. **怒气系统**: 怒气影响暴击率
2. **Q治疗**: 消耗怒气治疗
3. **W减速**: 嘲弄附带减速效果
4. **E位移**: 旋转劈砍可以穿墙
5. **R不死**: 大招提供最低生命值
