#  Nocturne (梦魇)

## 基础信息

- **称号**: 梦魇 (The Eternal Nightmare)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 刺客、战士
- **位置**: 打野

## 技能介绍

### 被动 - 夜魔翅膀 (Umbra Blades)

**机制**: Nocturne定期强化普攻，对周围敌人造成伤害并治疗。

**额外伤害**: 120% AD
**治疗量**: 13-32 (+ 30% AP)
**冷却**: 12秒

### Q - 暗影之刃 (Duskbringer)

- **冷却**: 8 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 1200

Nocturne投掷一把暗影之刃，留下暗影之径。

**伤害**: 65/110/155/200/245 (+ 85% bonus AD)
**暗影之径移速**: 20/25/30/35/40%
**暗影之径AD加成**: 15/25/35/45/55

### W - 黑暗庇护 (Shroud of Darkness)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 50 法力
- **范围**: 自助

Nocturne获得攻速加成，成功挡技能后攻速翻倍。

**攻速加成**: 30/35/40/45/50%
**强化攻速**: 60/70/80/90/100%
**持续时间**: 5秒

### E - 无法言喻的恐惧 (Unspeakable Horror)

- **冷却**: 15/14/13/12/11 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 425

Nocturne与目标建立锁链，造成持续伤害后恐惧。

**总伤害**: 80/125/170/215/260 (+ 100% AP)
**恐惧时间**: 1.25/1.5/1.75/2/2.25秒

### R - paranoid (偏执)

- **冷却**: 140/115/90 秒
- **消耗**: 100 法力
- **范围**: 2500/3250/4000

Nocturne恐惧所有敌方英雄并可以冲向一个目标。

**伤害**: 150/275/400 (+ 120% bonus AD)
**致盲时间**: 6秒
**冲刺距离**: 全图

## 连招技巧

### 基础连招

**刷野连招**:
- Q(暗影之刃) → 普攻 → W(黑暗庇护)
- 暗影之径提供加速

**Gank连招**:
- Q(暗影之刃) → E(无法言喻的恐惧) → 普攻
- 恐惧后持续追击

### 进阶连招

**团战连招**:
- R(paranoid) → Q(暗影之刃) → E(无法言喻的恐惧) → W(黑暗庇护)
- 大招切入恐惧

**秒杀连招**:
- R(paranoid) → Q(暗影之刃) → 普攻 → E(无法言喻的恐惧)
- 大招突进秒杀

## 隐藏机制

1. **Q暗影之径**: 站在上面获得移速和AD加成
2. **W挡技能**: 成功挡技能后攻速翻倍
3. **E锁链**: 未挣脱则恐惧
4. **R全图**: 大招可以全图突进
5. **被动治疗**: 被动普攻可以治疗
