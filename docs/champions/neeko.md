#  Neeko (妮蔻)

## 基础信息

- **称号**: 妮蔻 (The Curious Chameleon)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、爆发
- **位置**: 中单、辅助

## 技能介绍

### 被动 - 先天魅力 (Inherent Glamour)

**机制**: Neeko可以伪装成友军英雄，混入敌人中间。

**伪装范围**: 400
**跳跃范围**: 500
**触发条件**: 靠近友军小兵、陷阱、守卫、宠物或草丛植物

### Q - 盛开爆炸 (Blooming Burst)

- **冷却**: 9/8.5/8/7.5/7 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 800

Neeko投掷一颗种子，落地时爆发造成伤害。

**伤害**: 60/110/160/210/260 (+ 60% AP)
**绽放伤害**: 35/60/85/110/135 (+ 25% AP)
**击杀传播**: 如果击杀英雄或大型野怪，会再次绽放

### W - 变形者 (Shapesplitter)

- **冷却**: 16/15/14/13/12 秒
- **消耗**: 自助
- **范围**: 900

Neeko创造一个分身，双方都获得隐身和加速。

**额外伤害**: 30/65/100/135/170 (+ 60% AP)
**加速**: 10/17.5/25/32.5/40%
**分身持续**: 3秒

### E - 纠缠之刺 (Tangle-Barbs)

- **冷却**: 12/11.5/11/10.5/10 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 1000

Neeko投掷螺旋形魔法，造成伤害并禁锢敌人。

**伤害**: 70/105/140/175/210 (+ 65% AP)
**禁锢时间**: 0.7/0.9/1.1/1.3/1.5秒
**强化禁锢**: 1.8/2.1/2.4/2.7/3秒 (击中敌人后)

### R - 绽放 (Pop Blossom)

- **冷却**: 120/105/90 秒
- **消耗**: 100 法力
- **范围**: 590

Neeko蓄力后跃起，击飞并眩晕周围敌人。

**伤害**: 150/350/550 (+ 120% AP)
**击飞时间**: 0.6秒
**眩晕时间**: 0.75秒
**蓄力时间**: 1.25秒

## 连招技巧

### 基础连招

**标准消耗**:
- Q(盛开爆炸) → E(纠缠之刺) → W(变形者)
- 种子伤害后禁锢

**伪装先手**:
- 伪装接近 → R(绽放) → Q(盛开爆炸) → E(纠缠之刺)
- 伪装后突然爆发

### 进阶连招

**团战连招**:
- W(变形者) → R(绽放) → Q(盛开爆炸) → E(纠缠之刺)
- 分身迷惑后团控

**单杀连招**:
- Q(盛开爆炸) → 普攻 → E(纠缠之刺) → W(变形者)
- 持续输出

---

## 隐藏机制

1. **被动伪装**: 可以伪装成友军英雄
2. **Q传播**: 击杀后种子会再次绽放
3. **W分身**: 分身可以移动和普攻
4. **E强化**: 击中敌人后禁锢时间延长
5. **R团控**: 蓄力期间可以获得护盾
