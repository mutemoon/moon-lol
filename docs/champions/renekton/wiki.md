# 鳄鱼 (Renekton)

## 基础信息

- **称号**: 沙漠巨鳄 (the Butcher of the Sands)
- **类型**: 物理 / 近战
- **难度**: 1/3
- **角色**: 战士、坦克
- **位置**: 上单

## 技能介绍

### 被动 - 怒之领域 (Reign of Anger)

雷克顿的攻击会生成怒气，低生命值时怒气生成量大幅提升。
- 普通攻击生成5怒气
- 技能命中英雄时生成额外怒气
- 怒气可用于强化技能（黄条状态下施放）
- 50怒气以上时技能会消耗怒气获得强化效果
- 怒气上限100，不会溢出

### Q - 冷酷猎杀 (Cull the Meek)

- **冷却**: 7 秒
- **消耗**: 无（强化时消耗）
- **范围**: 325

雷克顿挥动巨刃，对周围敌人造成伤害。
- 对小兵伤害降低
- 命中英雄/大型野怪回复生命值
- 生成怒气（强化时不生成）

**强化效果（50+怒气）**:
- 伤害提升
- 回复效果增强
- 不再生成怒气

**效果数值**:
- 基础伤害: 65/95/125/155/185/215
- 强化伤害: 95/135/175/215/255/295
- 对小兵: 50% 伤害
- 基础回复: 2.5/4/5.5/7/8.5/10 每英雄
- 强化回复: 7/10/13/16/19/22 每英雄

**隐藏机制**:
- Q的范围约为325（以雷克顿为中心的圆）
- Q命中多个英雄时回复效果可叠加
- Q的伤害享受AD加成
- 强化Q的范围略大

### W - 残暴猛击 (Ruthless Predator)

- **冷却**: 16/14/12/10/8 秒
- **消耗**: 无（强化时消耗）
- **范围**: 300

雷克顿的下次攻击造成两次打击，眩晕目标0.25秒（强化时0.75秒）。
- 强化时攻击三次
- 强化时可以击碎护盾
- 命中英雄时生成额外怒气

**强化效果（50+怒气）**:
- 攻击三次（而非两次）
- 眩晕时间延长至0.75秒
- 可以击碎护盾
- 不再生成怒气

**效果数值**:
- 基础总伤害: 120/160/200/240/280 + 1.5 总AD
- 强化总伤害: 180/240/300/360/420 + 2.25 总AD
- 基础眩晕: 0.25秒
- 强化眩晕: 0.75秒

**隐藏机制**:
- W可以重置平A
- W的基础伤害略低于普攻+技能连招
- W的眩晕可以被韧性减少
- W强化时第三次攻击可以触发贪欲吸血
- W可以打断引导技能

### E - 横冲直撞 (Slice and Dice)

- **冷却**: 16/14.5/13/11.5/10 秒
- **消耗**: 无（强化时消耗）
- **范围**: 450

雷克顿向前冲刺，造成伤害。命中敌人后可以再次施放。
- 第一次冲刺命中后获得第二次施放机会
- 强化时伤害更高，可以削减护甲

**强化效果（50+怒气）**:
- 伤害提升
- 第二段削减护甲25%，持续4秒
- 不再生成怒气

**效果数值**:
- 基础伤害: 50/70/90/110/130 + 0.9 总AD
- 强化伤害: 70/95/120/145/170 + 1.35 总AD
- 护甲削减: 25%，持续4秒
- 第二次施放时间窗口: 6秒

**隐藏机制**:
- E可以穿越薄墙
- E的位移距离约450
- E第一段命中后有6秒窗口施放第二段
- E强化时的护甲削减对野怪也生效
- E可以用来取消Q的后摇
- E闪（E + Flash）可以改变冲刺方向

### R - 统治 (Dominus)

- **冷却**: 120/100/80 秒
- **消耗**: 无
- **范围**: 20（自身）

雷克顿化身黑暗能量形态，持续12秒。
- 获得额外最大生命值
- 获得额外怒气
- 持续对周围敌人造成魔法伤害
- 每秒生成怒气

**效果数值**:
- 生命值加成: 250/450/650
- 每秒魔法伤害: 40/80/120 + 0.1 AP
- 每秒怒气生成: 5/10/15

**隐藏机制**:
- R期间怒气获取更快（配合强化技能）
- R的范围伤害可以触发贪欲吸血
- R结束后会损失额外生命值（但不低于基础生命值）
- R可以用于取消其他技能的后摇
- R持续期间每秒生成怒气，强化E/W/Q更容易

## 连招技巧

### 基础连招

**标准对线换血**:
- E第一段 → 平A → W → Q → E第二段脱离
- 或者：E → 平A → W → Q → 继续追打

**怒气利用连招**:
- 先用普通E叠怒气 → 等50+ → E → W → Q → E
- 保持强化状态

**AAW连招**:
- 平A → W（重置平A）→ 平A
- 快速打出两次普攻 + W伤害

### 进阶连招

**RW连招**:
- R开启 → 获得大量怒气 → W（强化） → E（强化）
- 大招期间尽量使用强化技能

**WE连招**:
- W起手 → 立即E第一段 → E第二段
- 确保W命中后接E追击

**E闪连招**:
- E冲刺过程中 → F → 改变方向
- 可以出其不意接近敌人

**无伤换血（利用怒气）**:
- 保持50+怒气 → E第一段 → 立即W → Q → E第二段撤退
- 强化技能提供更高伤害

### 团战连招

**团战进场**:
- R → E → W（强化，眩晕多个） → Q（强化） → E第二段
- 利用R获得大量怒气强化所有技能

**侧翼切入**:
- R → E穿越墙体 → W → Q → E
- 优先攻击敌方后排

## 怒气系统详解

### 怒气获取
- 普通攻击: 5 怒气
- Q命中英雄: +5 怒气
- E命中英雄: +5 怒气
- W命中英雄: +10 怒气
- R期间每秒: 5/10/15 怒气

### 怒气消耗
- 强化Q: -50 怒气
- 强化W: -50 怒气
- 强化E: -50 怒气

### 强化技能对比

| 技能 | 普通 | 强化 |
|------|------|------|
| Q伤害 | 100% | 150% |
| Q回复 | 100% | 300% |
| Q怒气 | +15/英雄 | 0 |
| W次数 | 2次 | 3次 |
| W眩晕 | 0.25秒 | 0.75秒 |
| W盾碎 | 无 | 有 |
| E伤害 | 100% | 150% |
| E护甲 | 无 | 25%削减 |

## 隐藏机制

1. **怒气不溢出**: 怒气上限100，达到50后仍可继续积累
2. **低血量怒气翻倍**: 生命值低于50%时，怒气获取翻倍
3. **Q范围略大于指示器**: 实际范围比显示的325略大
4. **E可以穿薄墙**: 部分薄墙（如龙坑墙）可以被E穿越
5. **W重置平A**: W可以取消平A后摇
6. **强化W碎盾**: 强化W可以击碎任何护盾
7. **E第二段可被控制打断**: 第一段命中的窗口内被控制，第二段也会被打断
8. **R结束后掉血**: R结束时额外生命值会消失，但不会低于基础生命值
9. **Q享受吸血**: Q的范围伤害可以触发贪欲吸血
10. **E强化护甲削减**: 对防御塔也生效（4秒）
11. **怒气影响模型**: 50+怒气时雷克顿的眼睛会发光
12. **R范围伤害**: R期间的范围伤害可以触发法术吸血（只限AP）

## 常见问题

**Q: 鳄鱼怎么保持怒气？**
A: 对线期多用平A和小技能积累怒气，保持50+可以随时强化技能进行对拼。

**Q: 鳄鱼强化E的护甲削减有什么作用？**
A: 25%护甲削减对物理伤害非常有效，可以显著提升自己和队友的输出。

**Q: 鳄鱼R结束后为什么会掉血？**
A: R提供的额外生命值会在R结束后消失，这是正常的。不会低于基础生命值。

**Q: 鳄鱼最强力的连招是什么？**
A: 50+怒气 → E → W(强化) → Q(强化) → E → ... 利用强化技能打出最大伤害。

**Q: 鳄鱼E闪怎么操作？**
A: 在E冲刺过程中按F（闪现），可以改变E的冲刺方向，用于出其不意的追击或撤退。
