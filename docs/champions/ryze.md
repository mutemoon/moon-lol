#  Ryze (瑞兹)

## 基础信息

- **称号**: 瑞兹 (The Rune Mage)
- **类型**: 魔法 / 远程
- **难度**: 3/3
- **角色**: 法师、战士
- **位置**: 中单

## 技能介绍

### 被动 - 奥术精通 (Arcane Mastery)

**机制**: Ryze增加自身最大法力值。

**法力加成**: 10% 每100 AP

### Q - 超负荷 (Overload)

- **冷却**: 5 秒
- **消耗**: 40/38/36/34/32 法力
- **范围**: 1000

Ryze释放符文冲击，造成伤害。

**伤害**: 75/95/115/135/155 (+ 55% AP) (+ 2% 额外法力)
**充能效果**: 消耗2层时获得额外移速

### W - 符文禁锢 (Rune Prison)

- **冷却**: 11/10.5/10/9.5/9 秒
- **消耗**: 50/60/70/80/90 法力
- **范围**: 550

Ryze禁锢敌人，减速或禁锢。

**伤害**: 60/90/120/150/180 (+ 60% AP) (+ 3% 额外法力)
**减速**: 50% (持续1.5秒)
**Flux效果**: 禁锢代替减速

### E - 符文流动 (Spell Flux)

- **冷却**: 3.5/3.25/3/2.75/2.5 秒
- **消耗**: 35/45/55/65/75 法力
- **范围**: 550

Ryze标记敌人并施加Flux效果。

**伤害**: 60/90/120/150/180 (+ 50% AP) (+ 2% 额外法力)
**Flux传播**: 可以在敌人间传播

### R -  realm warp (Realm Warp)

- **冷却**: 180/160/140 秒
- **消耗**: 100 法力
- **范围**: 3000

Ryze打开传送门，将自己和其他友军传送到指定位置。

**传送效果**: 眩晕+沉默+无敌 (0.75秒)
**Overload增强**: 50/75/100%

## 连招技巧

### 基础连招

**标准连招**:
- Q(超负荷) → W(符文禁锢) → E(符纹流动) → Q(超负荷)
- 技能循环输出

**控制连招**:
- W(符文禁锢) → E(符纹流动) → Q(超负荷)
- 禁锢后持续输出

### 进阶连招

**团战连招**:
- R(Realm Warp) → W(符文禁锢) → E(符纹流动) → Q(超负荷)
- 大招传送后团控

**逃跑连招**:
- R(Realm Warp) → Q(超负荷)
- 大招传送自己

---

## 隐藏机制

1. **被动增蓝**: 被动增加最大法力值
2. **Q充能**: Q可以生成充能层数
3. **E传播**: E可以在敌人间传播
4. **R传送**: 大招可以传送友军
5. **W双重效果**: 有Flux时变成禁锢
