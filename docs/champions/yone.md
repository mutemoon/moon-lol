# Yone (永恩)

## 基础信息

- **称号**: 永恩 (The Unforgotten)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、刺客
- **位置**: 上单、中单

## 技能介绍

### 被动 - 猎人之道 (Way of the Hunter)

**机制**: Yone暴击率翻倍。

**暴击转化**: 超出100%的暴击转化为AD
**双刃**: 交替使用钢剑和魔刃

### Q - 致命钢刺 (Mortal Steel)

- **冷却**: 4-1.33 秒
- **消耗**: 自助
- **范围**: 450/1050

Yone突刺造成伤害。

**伤害**: 25/50/75/100/125 (+ 110% AD)
**2层效果**: 旋风击飞

### W - 灵魂斩裂 (Spirit Cleave)

- **冷却**: 14-6 秒
- **消耗**: 自助
- **范围**: 600

Yone斩击造成混合伤害。

**伤害**: 10/20/30/40/50 (+ 8-12% 最大生命值)
**护盾**: 40-95 (+ 65% 额外AD)
**对英雄加成**: 护盾翻倍

### E - 灵魂游离 (Soul Unbound)

- **冷却**: 22/19/16/13/10 秒
- **消耗**: 自助
- **范围**: 300/全图

Yone进入灵魂形态。

**持续时间**: 5秒
**伤害存储**: 25-35%
**返回效果**: 造成真实伤害

### R - 命运之锁 (Fate Sealed)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 1000

Yone标记并击飞敌人。

**伤害**: 200/400/600 (+ 80% 额外AD)
**眩晕时间**: 1秒
**拉回效果**: 拉回并击飞

## 连招技巧

### 基础连招

**对线连招**:
- Q(致命钢刺) → W(灵魂斩裂) → E(灵魂游离)
- 位移输出

**单杀连招**:
- E(灵魂游离) → Q(致命钢刺) → W(灵魂斩裂) → R(命运之锁)
- 灵魂形态爆发

### 进阶连招

**团战连招**:
- R(命运之锁) → E(灵魂游离) → Q(致命钢刺) → W(灵魂斩裂)
- 大招开团

**逃脱连招**:
- E(灵魂游离) → 返回
- 位移返回

---

## 隐藏机制

1. **被动暴击**: 暴击率翻倍
2. **Q层数**: Q可以叠加2层
3. **W护盾**: W命中获得护盾
4. **E存储**: E存储伤害
5. **R控制**: R可以拉回敌人
