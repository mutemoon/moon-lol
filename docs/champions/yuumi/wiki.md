# Yuumi (悠米)

## 基础信息

- **称号**: 悠米 (The Magical Cat)
- **类型**: 魔法 / 远程
- **难度**: 1/3
- **角色**: 辅助、法师
- **位置**: 辅助

## 技能介绍

### 被动 - 猫咪友谊 (Feline Friendship)

**机制**: Yuumi周期性强化普攻。

**治疗**: 20-120 (+ 25% AP)
**好友系统**: 与队友建立好友关系

### Q - 游猎导弹 (Prowling Projectile)

- **冷却**: 6.5 秒
- **消耗**: 50/55/60/65/70/75 法力
- **范围**: 850

Yuumi发射导弹。

**伤害**: 60/95/130/165/200/235 (+ 20% AP)
**减速**: 20%
**附着加成**: 引导导弹

### W - 你和我! (You and Me!)

- **冷却**: 10/5/0 秒
- **消耗**: 自助
- **范围**: 700

Yuumi附身在友军身上。

**附身效果**: 无法被选中(除防御塔)
**治疗加成**: 4-8%

### E -  zoomies (Zoomies)

- **冷却**: 10 秒
- **消耗**: 80/90/100/110/120 法力
- **范围**: 自助

Yuumi为目标提供护盾和攻速。

**护盾**: 65/90/115/140/165 (+ 30% AP)
**攻速加成**: 25-35%
**附身效果**: 治疗友军并回复法力

### R - 最终章节 (Final Chapter)

- **冷却**: 120/110/100 秒
- **消耗**: 100 法力
- **范围**: 1100

Yuumi发射5波魔法波浪。

**治疗**: 30/40/50 (+ 10% AP)
**伤害**: 75/125/175 (+ 25% AP)
**减速**: 10-50%

## 连招技巧

### 基础连招

**对线连招**:
- Q(游猎导弹) → E(zoomies) → W(你和我!)
- 消耗治疗

**保护连招**:
- W(你和我!) → E(zoomies) → Q(游猎导弹)
- 附身保护

### 进阶连招

**团战连招**:
- R(最终章节) → W(你和我!) → E(zoomies) → Q(游猎导弹)
- 大招治疗

**反手连招**:
- W(你和我!) → R(最终章节) → Q(游猎导弹)
- 附身反打

## 隐藏机制

1. **被动治疗**: 被动周期性治疗
2. **W附身**: W附身友军无法被选中
3. **E护盾**: E提供护盾和攻速
4. **R多波**: R发射5波治疗
5. **好友系统**: 与队友建立好友获得额外效果
