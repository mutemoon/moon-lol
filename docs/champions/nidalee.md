#  Nidalee (奈德丽)

## 基础信息

- **称号**: 奈德丽 (The Bestial Huntress)
- **类型**: 物理 / 远程 (人形态) / 近战 (美洲豹形态)
- **难度**: 3/3
- **角色**: 刺客、战士
- **位置**: 打野

## 技能介绍

### 被动 - 草丛掠食 (Prowl)

**机制**: 在草丛中获得移速加成，追击敌人时移速更高。

**草丛移速**: 30%
**追击移速**: 30%
**标记持续**: 4秒

### Q (人) - 投掷标枪 (Javelin Toss)

- **冷却**: 6 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 1500

Nidalee投掷一根标枪，距离越远伤害越高。

**最小伤害**: 70/90/110/130/150 (+ 50% AP)
**最大伤害**: 基于飞行距离增加最高225%

### Q (豹) - 猛扑 (Takedown)

- **冷却**: 3 秒
- **消耗**: 自助
- **范围**: 125

Nidalee猛扑向目标，造成高额伤害。

**伤害**: 70/130/190/250 (+ 100% AD)
**强化伤害**: 对被标记目标增加30%

### W (人) - 丛林伏击 (Bushwhack)

- **冷却**: 13/12/11/10/9 秒
- **消耗**: 30/35/40/45/50 法力
- **范围**: 900

Nidalee设置一个陷阱，敌人踩中受到持续伤害。

**每跳伤害**: 10/20/30/40/50 (+ 5% AP)
**持续时间**: 4秒
**最大陷阱数**: 4/6/8/10

### W (豹) - 跳跃 (Pounce)

- **冷却**: 3 秒
- **消耗**: 自助
- **范围**: 375

Nidalee跳跃到目标位置，对附近敌人造成伤害。

**伤害**: 55/100/145/190 (+ 50% bonus AD)
**强化伤害**: 对被标记目标增加30%

### E (人) - 野性激发 (Primal Surge)

- **冷却**: 12 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 900

Nidalee治疗友军并提供攻速加成。

**治疗量**: 50/75/100/125/150 (+ 35% AP)
**攻速加成**: 30/40/50/60/70%
**强化治疗**: 目标生命值越低治疗越多

### E (豹) - 挥击 (Swipe)

- **冷却**: 3 秒
- **消耗**: 自助
- **范围**: 300

Nidalee挥击面前区域，造成伤害。

**伤害**: 70/130/190/250 (+ 40% AP)
**强化伤害**: 对被标记目标增加30%

### R - 美洲豹形态 (Aspect of the Cougar)

- **冷却**: 3 秒
- **消耗**: 自助

Nidalee在人类和美洲豹形态间切换。

**人类形态**: 远程，获得Q/W/E技能
**美洲豹形态**: 近战，获得Q/W/E技能
**技能升级**: 6/11/16级获得额外效果

---

## 连招技巧

### 基础连招

**人形态消耗**:
- 标枪(Q) → 陷阱(W) → 治疗(E)
- 远程消耗后治疗

**豹形态追击**:
- 跳跃(W) → 猛扑(Q) → 挥击(E)
- 追击收割

### 进阶连招

**团战连招**:
- R(美洲豹) → 跳跃(W) → 猛扑(Q) → 挥击(E) → R(人形态)
- 切换形态输出

**Gank连招**:
- R(美洲豹) → 跳跃(W) → 猛扑(Q) → 标枪(Q) → R(人形态) → 治疗(E)
- 豹形态起手人形态治疗

---

## 隐藏机制

1. **形态切换**: R可以自由切换人/豹形态
2. **追击标记**: 被动提供移速追击
3. **陷阱控图**: W陷阱可以限制敌方打野
4. **强化技能**: 对被标记目标技能伤害增加
5. **治疗爆发**: E治疗量基于目标损失生命值
