#  Katarina (卡特琳娜)

## 基础信息

- **称号**: 卡特琳娜 (The Sinister Blade)
- **类型**: 物理+魔法 / 近战
- **难度**: 3/3
- **角色**: 刺客
- **位置**: 中单
- **资源**: 无法力
- **攻击距离**: 125

## 技能介绍

### 被动 - 贪婪 (Voracity)

**类型**: 环境型被动
**效果范围**: 340

**贪婪 (Voracity)**: 在3秒内对英雄造成伤害后参与击杀，会减少Katarina技能当前冷却15秒。

**邪恶钢 (Sinister Steel)**: 每当Katarina捡起一把匕首，她会向周围敌人横扫，造成68-275 (基于等级) (+ 60% bonus AD) (+ 70/80/90/100% 基于等级 AP) 魔法伤害，对命中的英雄施加命中效果，并减少Shunpo当前冷却其总冷却时间的78/84/90/96% (基于等级)。

匕首在其半径内提供视野，会在地面上停留4秒后消失。

### Q - 弹射之刃 (Bouncing Blade)

- **冷却**: 11/10/9/8/7 秒
- **施法时间**: 0.25秒
- **目标范围**: 625
- **效果范围**: 450
- **速度**: 1600

**主动**: Katarina向目标敌人投掷一把匕首，造成魔法伤害。匕首到达后会弹射到最多两个额外的附近可见敌人，造成相同伤害，每次弹射到弹射目标需要0.15秒。

**魔法伤害**: 80/115/150/185/220 (+ 40% AP)

匕首在击中第一个目标后1.023秒落地，落在离目标位置约350单位的对面。

### W - 准备 (Preparation)

- **冷却**: 15/14/13/12/11 秒
- **施法时间**: 无

**主动**: Katarina将一把匕首抛向她当前位置上方的空中，获得持续衰减的额外移动速度，持续1.25秒，落地期间变为幽灵状态。

**额外移动速度**: 50/60/70/80/90%

### E - 瞬步 (Shunpo)

- **冷却**: 12/11/10/9/8 秒
- **施法时间**: 0.15秒
- **目标范围**: 725 / 775

**主动**: Katarina闪烁到光标附近最接近的单位或匕首位置，对范围内最近的敌人造成魔法伤害并施加命中效果。

**魔法伤害**: 20/30/40/50/60 (+ 40% AD) (+ 25% AP)

Katarina优先闪烁到匕首位置，然后是敌方英雄，然后是敌方非英雄，最后是任何其他单位。

Shunpo重置Katarina的普通攻击计时器。

### R - 死亡莲花 (Death Lotus)

- **冷却**: 75/60/45 秒
- **施法时间**: 无
- **效果范围**: 550
- **速度**: 2400
- **队列阈值**: 0.25秒

**主动**: Katarina引导最多2秒，每0.166秒向最近的最多3个敌人投掷一把匕首，在过程中显示自己。

每把匕首造成物理伤害和魔法伤害，施加命中效果和攻击效果，命中效果和攻击效果伤害降低，并对目标施加3秒重伤效果。

**每把匕首物理伤害**: 16% (+ 50% 每100% bonus AD) bonus AD
**每把匕首魔法伤害**: 25/37.5/50 (+ 19% AP)
**命中/攻击效果伤害有效性**: 25/30/35%

施放此技能需要500范围内有敌方英雄。目标不需要可见就能被此技能命中。

## 连招技巧

### 基础连招

**标准消耗**:
- Q(弹射之刃) → E(瞬步) → 普通攻击
- 匕首弹射后瞬步追击

**连杀连招**:
- Q → E → R(死亡莲花)
- 被动减少冷却

### 进阶连招

**团战连招**:
- R(死亡莲花) → E(瞬步) → Q(弹射之刃)
- 大招切入

**追杀连招**:
- W(准备) → E(瞬步) → Q(弹射之刃)
- 移速加成追击

## 隐藏机制

1. **无蓝条**: Katarina不需要法力
2. **被动刷新**: 参与击杀减少所有技能冷却15秒
3. **匕首机制**: Q和W产生匕首，捡起造成范围伤害并减少E冷却
4. **E位移**: E是主要位移技能，优先跳向匕首
5. **R重伤**: R施加3秒重伤效果
6. **Sinister Steel**: 捡起匕首对周围造成伤害并减少Shunpo冷却
