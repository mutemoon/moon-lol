#  Katarina (卡特琳娜)

## 基础信息

- **称号**: 卡特琳娜 (The Sinister Blade)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 刺客
- **位置**: 中单

## 技能介绍

### 被动 - 贪婪 (Voracity)

**机制**: Katarina在3秒内参与击杀英雄后，大幅减少技能冷却。她的技能击杀返还冷却。

**参与击杀减少**: 15秒
**收取匕首**: 造成范围魔法伤害

### Q - 弹射之刃 (Bouncing Blade)

- **冷却**: 11/10/9/8/7 秒
- **消耗**: 无
- **范围**: 625

Katarina投掷匕首，可以在敌人间弹射。

**伤害**: 80/115/150/185/220 (+ 40% AP)
**弹射目标**: 最多3个

### W - 准备 (Preparation)

- **冷却**: 15/14/13/12/11 秒
- **消耗**: 无

Katarina将匕首抛向空中，获得移速加成。

**移速加成**: 50/60/70/80/90%

### E - 瞬步 (Shunpo)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 无
- **范围**: 725

Katarina闪烁到目标位置，造成伤害。

**伤害**: 20/30/40/50/60 (+ 40% AD) (+ 25% AP)

### R - 死亡莲花 (Death Lotus)

- **冷却**: 75/60/45 秒
- **消耗**: 无
- **范围**: 550

Katarina快速投掷匕首，最多持续2.5秒。

**每枚伤害**: 物理+魔法
**减速**: 重伤效果

---

## 连招技巧

### 基础连招

**标准消耗**:
- Q(弹射之刃) → E(瞬步) → 普攻
- 匕首弹射后瞬步追击

**连杀连招**:
- Q → E → R(死亡莲花)
- 被动减少冷却

### 进阶连招

**团战连招**:
- R(死亡莲花) → E(瞬步) → Q(弹射之刃)
- 大招切入

**追杀连招**:
- W(准备) → E(瞬步) → Q(弹射之刃)
- 移速加成追击

---

## 隐藏机制

1. **无蓝条**: Katarina不需要法力
2. **被动刷新**: 参与击杀减少所有技能冷却15秒
3. **匕首机制**: Q和W产生匕首，捡起造成范围伤害
4. **E位移**: E是主要位移技能
5. **R重伤**: R施加重伤效果
