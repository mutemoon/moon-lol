# 潘森 (Pantheon)

## 基础信息

- **称号**: "不灭之矛" (the Unbreakable Spear)
- **类型**: 物理 / 近战
- **难度**: 1/3
- **角色**: 战士、刺客
- **位置**: 上单、打野

## 技能介绍

### 被动 - 矢志不渝 (Mortal Will)

潘森每攻击或施放技能3次，下一次技能就会强化。
- 第3次攻击或技能后，下一个Q/W/E会被强化
- 强化技能有额外效果

**强化效果**:
- Q强化: 额外伤害和减速
- W强化: 额外眩晕时间
- E强化: 额外伤害和范围扩大

**隐藏机制**:
- 被动计数在技能命中后增加
- 计数可以通过走A来叠加

### Q - 彗星Q (Comet Spear)

- **冷却**: 11/10.25/9.5/8.75/8 秒
- **消耗**: 25 法力
- **范围**: 575

潘森向选择的方向刺出或投掷他的长矛。
- 按住Q蓄力可以投掷更远的距离
- 释放Q取消蓄力可以刺击

**效果数值**:
- 刺击伤害: 55/75/95/115/135 (+0.65 AD)
- 投掷伤害: 55/75/95/115/135 (+0.65 AD)
- 强化投掷减速: 40/50/60%，持续1.5秒

**隐藏机制**:
- Q蓄力可以被打断
- Q可以用来消耗和补刀
- Q强化后有减速效果

### W - 盾牌Vault (Shield Vault)

- **冷却**: 13/12/11/10/9 秒
- **消耗**: 55 法力
- **范围**: 600

潘森向目标冲刺，造成伤害并眩晕他们。
- 冲刺到目标身边
- 眩晕目标

**效果数值**:
- 伤害: 60/100/140/180/220 (+0.75 AD)
- 眩晕时长: 0.75/1/1.25/1.5/1.75 秒
- 强化眩晕: 额外0.25秒

**隐藏机制**:
- W可以跳向小兵、英雄、眼位
- W可以配合Q取消后摇
- W可以用来穿越地形

### E - 圣盾庇护 (Aegis Assault)

- **冷却**: 22/21/20/19/18 秒
- **消耗**: 80 法力
- **范围**: 400

潘森举起盾牌，进入无敌状态，免疫来自前方的伤害，并对周围敌人反复刺击。
- 持续1.5秒
- 盾牌朝向由鼠标位置决定

**效果数值**:
- 每击伤害: 20/30/40/50/60 (+0.25 AD)
- 强化伤害: 30/45/60/75/90 (+0.4 AD)
- 强化范围: 500

**隐藏机制**:
- E免疫来自盾牌方向的所有伤害
- E可以挡技能和普通攻击
- E持续期间可以移动
- E可以用来挡皇子EQ、EZ Q等

### R - 星球坠落 (Grand Starfall)

- **冷却**: 180/165/150 秒
- **消耗**: 100 法力
- **范围**: 5500

潘森凝聚力量后跃入空中，降落在选择的位置作为一颗流星。
- 降落的准备时间约2秒
- 降落范围巨大

**效果数值**:
- 最大伤害: 300/500/700 (+2.0 AD)
- 减速: 30/40/50%，持续1秒

**隐藏机制**:
- R降落可以被控制技能打断
- R降落前有视野
- R可以用来支援/Gank
- R降落可以配合E挡伤害

## 连招技巧

### 基础连招

**对线消耗**:
- Q蓄力 → 投掷 → W → 平A

**击杀连招**:
- W → 平A → Q → E

**强化连招**:
- 被动3层 → W强化眩晕 → 平A → Q强化 → E

### 进阶连招

**WQ+E**:
- W近身 → Q取消后摇 → E挡伤害

**EW**:
- E盾牌朝向 → 免疫伤害 → W跳走

**闪现+W**:
- 闪现接近 → W眩晕

### 团战进场

**大招支援**:
- R降落 → E挡伤害 → W切入

**切后排**:
- R降落到后排 → W → 平A → Q

**反手连招**:
- E盾牌朝向 → 挡伤害 → W反击

## 隐藏机制

1. **被动强化**: 每3次攻击或技能后，下一个技能被强化
2. **Q蓄力**: 按住Q可以蓄力，增加投掷距离
3. **W跳眼**: W可以跳向眼位穿越地形
4. **E盾牌方向**: E免疫盾牌朝向的伤害
5. **被动计数**: 被动计数通过平A和技能叠加
6. **Q取消后摇**: Q可以用来取消平A后摇
7. **W重置平A**: W可以重置平A
8. **E持续移动**: E持续期间可以移动调整位置
9. **R降落视野**: R降落前可以获得目标区域视野
10. **R可以被打断**: R降落可以被控制技能打断

## 常见问题

**Q: 潘森如何对线？**
A: 利用Q远程消耗，等待敌人血量降低后W近身配合E输出。

**Q: 潘森的核心技巧是什么？**
A: E盾牌挡技能是核心，可以挡掉很多关键伤害。

**Q: 潘森如何打团？**
A: 利用R支援进场，E盾牌挡伤害，W切入后排。
