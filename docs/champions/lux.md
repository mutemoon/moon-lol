# 拉克丝 (Lux)

## 基础信息

- **称号**: 光芒四射 (The Lady of Luminosity)
- **类型**: 魔法 / 远程
- **难度**: 1/3
- **角色**: 法师、辅助
- **位置**: 中单、辅助

## 技能介绍

### 被动 - 照明 (Illumination)

**机制**: 拉克丝的技能会对敌人施加光芒标记（持续6秒）。

- 拉克丝的普攻和R技能会消耗标记
- 触发时造成额外魔法伤害
**额外伤害**: 30-220 (基于等级) (+ 35% AP)

### Q - 光之束缚 (Light Binding)

- **冷却**: 10 秒
- **消耗**: 50 法力
- **范围**: 1300

拉克丝朝目标方向发射光球，禁锢前两个命中的敌人2秒。

**伤害**: 80/120/160/200/240 (+ 75% AP)

### W - 曲光屏障 (Prismatic Barrier)

- **冷却**: 12/11.5/11/10.5/10 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 1175

拉克丝投出魔杖，飞向目标后返回。为沿途的友军提供护盾（最多叠加2次）。

**护盾值**: 40/55/70/85/100 (+ 40% AP)

### E - 透光奇点 (Lucent Singularity)

- **冷却**: 10/9.5/9/8.5/8 秒
- **消耗**: 70/80/90/100/110 法力
- **范围**: 1100

拉克丝在目标位置创造一个光芒区域，持续5秒，减速区域内敌人。可以手动再次激活引爆。

**减速**: 25/30/35/40/45%
**引爆伤害**: 65/115/165/215/265 (+ 80% AP)

### R - 最终闪光 (Final Spark)

- **冷却**: 60/50/40 秒
- **消耗**: 100 法力
- **范围**: 3400

拉克丝发射一道巨大的光束，对一条直线上的所有敌人造成魔法伤害，并暴露他们1.5秒。

**伤害**: 300/400/500 (+ 120% AP)

## 连招技巧

### 基础连招

**标准连招**:
- Q(光之束缚) → E(透光奇点) → R(最终闪光)
- Q禁锢后接E减速然后R收割

**辅助连招**:
- W(曲光屏障) → Q(光之束缚) → E(透光奇点)
- 保护友军同时控制敌人

### 进阶连招

**爆发连招**:
- Q(光之束缚) → 普攻(触发被动) → E(透光奇点) → 普攻 → R
- 利用被动的额外伤害增加爆发

**远程收割**:
- E(透光奇点) → R(最终闪光) → Q
- E先手减速，R高伤害收割

**团控连招**:
- R → Q → E → W
- R先手暴露敌人位置，Q禁锢，E减速

## 隐藏机制

1. **被动标记**: 技能先施加标记，普攻或R消耗标记触发额外伤害
2. **E手动引爆**: E可以在飞行途中或到达后手动引爆
3. **W双护盾**: W往返两次可以获得双倍护盾效果
4. **Q禁锢两人**: Q可以禁锢前两个敌人
5. **R全图距离**: R的距离非常远，可以用来清线或远程收割
6. **被动持续时间**: 被动标记持续6秒，可以刷新
