## 技能介绍

### 被动 - 勇气 (Perseverance)

**机制**: Garen 额外每秒回复1%-10%（基于等级）的最大生命值。如果受到英雄、史诗野怪、防御塔的伤害，或被敌人技能/召唤师技能命中，勇气会失去8秒，每次受到伤害和命中都会刷新。

### Q - 致命打击 (Decisive Strike)

- **冷却**: 8 秒
- **施法时间**: 无

**主动**: Garen 清除自身所有减速效果并在1-3.05秒（基于等级）内获得35%移速加成。

此外，Garen 在4秒内强化下次普攻，获得不可取消的冲锋，对目标造成额外物理伤害并沉默1.5秒。

**额外物理伤害**: 30/60/90/120/150 (+ 50% AD)

致命打击重置 Garen 的普攻计时器。

### W - 勇气 (Courage)

- **冷却**: 22/19/17/14/12 秒
- **施法时间**: 无

**被动**: 每次击杀敌人，Garen 获得一层勇气，最多叠加150层。每层提供0.25额外护甲和0.25额外魔抗，最多30点抗性。

**主动**: Garen 在4秒内减少受到的伤害。在前0.75秒内，他额外获得一个护盾和60%韧性。

**伤害减免**: 25/29/33/37/41%
**护盾强度**: 65/85/105/125/145 (+ 18% 额外生命值)

### E - 审判 (Judgment)

- **冷却**: 9/8.25/7/6.75/6 秒
- **施法时间**: 无
- **效果半径**: 325

**主动**: Garen 在3秒内快速旋转武器7次（+ 每25%攻速额外1次），变得无法进行普攻但获得幽灵状态并对附近敌人造成周期性物理伤害。审判每命中英雄6次会施加25%护甲穿透，持续6秒，每次第7次命中和之后每6次命中会刷新持续时间。

**每次旋转物理伤害**: 4/7/10/13/16 (+ 40/43/46/49/52% AD)

对最近命中敌人的伤害增加25%。

**每次旋转增伤**: 5/8.75/12/16.25/20 (+ 50/53.75/57/61.25/65% AD)

可以在1秒后手动释放，或在持续时间结束后自动释放。

审判可以暴击，造成130%+9%伤害。

### R - 德玛西亚正义 (Demacian Justice)

- **冷却**: 120/100/80 秒
- **施法时间**: 0.435秒
- **目标范围**: 400

**主动**: Garen 召唤德玛西亚的力量对付目标敌人，在施法时间开始时揭露他们并造成真实伤害。

**真实伤害**: 150/250/350 (+ 25/30/35% 目标已损失生命值)

## 连招技巧

### 基础连招

**对线消耗**:
- Q起手 → 平A → E → W → 拉开

**击杀连招**:
- Q → 平A → E → 平A → R

**E+Q取消后摇**:
- E旋转 → Q → 取消E后摇，增加伤害

### 进阶连招

**WQ+E**:
- W护盾 → Q → 平A → E

**R斩杀连招**:
- 将敌人打到低血量 → R处决

**闪现+R**:
- 闪现调整R的距离，用于收割逃跑的敌人

**W+R**:
- W护盾 → R处决

### 团战进场

**切后排连招**:
- Q加速 → E切入 → W护盾 → R瞄准后排

**反手连招**:
- W护盾 → Q移除减速 → E反击

**闪现+E**:
- 闪现到敌人身边 → E旋转

## 隐藏机制

1. **Q移除减速**: Q可以移除所有减速效果
2. **Q重置平A**: Q可以取消平A后摇
3. **E范围伤害**: E的范围可以击中多个敌人
4. **W韧性**: W提供的韧性可以减少控制时间
5. **R斩杀机制**: R的伤害基于目标已损失生命值
6. **被动脱战回复**: 被动在脱战后提供生命回复
7. **E暴击加成**: E享受暴击加成
8. **W护盾**: W的主动可以挡技能
9. **Q+E取消**: E+Q可以取消E的后摇
10. **R视野获取**: R斩杀后获得目标区域视野
