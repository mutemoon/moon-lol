#  Rek'Sai (雷克塞)

## 基础信息

- **称号**: 雷克塞 (The Void Burrower)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、打野
- **位置**: 打野

## 技能介绍

### 被动 - 虚空之怒 (Fury of the Xer'Sai)

**机制**: Rek'Sai攻击生成怒气，怒气满时技能增强。

**怒气上限**: 100
**怒气消耗**: 潜地时消耗怒气治疗
**治疗量**: 9-21% 最大生命值

### Q - 女皇之怒 (Queen's Wrath)

- **冷却**: 4/3.5/3/2.5/2 秒
- **消耗**: 自助
- **范围**: 325

Rek'Sai强化接下来3次普攻，造成额外伤害。

**额外伤害**: 30/35/40/45/50% AD
**攻速加成**: 35%
**持续时间**: 3秒

### W - 潜土 (Burrow)

- **冷却**: 4 秒
- **消耗**: 自助
- **范围**: 1500

Rek'Sai潜入地下，获得移速和特殊感知。

**移速加成**: 5/10/15/20/25
**感知能力**: 可以感知附近移动的敌人

### E - 狂野之咬 (Furious Bite)

- **冷却**: 6 秒
- **消耗**: 自助
- **范围**: 225

Rek'Sai撕咬敌人，怒气满时伤害转化为真实伤害。

**伤害**: 80/108/136/164/192 (+ 64% 额外AD)
**怒气增强**: 125%伤害，转化为真伤

### R - 虚空冲刺 (Void Rush)

- **冷却**: 120/100/80 秒
- **消耗**: 自助
- **范围**: 1500

Rek'Sai冲向被标记的目标，造成高额伤害。

**伤害**: 150/250/350 (+ 100% 额外AD) (+ 15/20/25% 最大生命值)
**标记条件**: 需要目标身上有"猎物标记"

## 连招技巧

### 基础连招

**刷野连招**:
- Q(女皇之怒) → 普攻 → E(狂野之咬)
- 强化普攻后撕咬

**Gank连招**:
- W(潜土) → 普攻 → Q(女皇之怒) → E(狂野之咬)
- 潜地接近后爆发

### 进阶连招

**团战连招**:
- R(虚空冲刺) → Q(女皇之怒) → W(潜土) → E(狂野之咬)
- 大招切入后持续输出

**单杀连招**:
- W(潜土) → Q(女皇之怒) → 普攻 → E(狂野之咬) → R(虚空冲刺)
- 潜地伏击

---

## 隐藏机制

1. **怒气系统**: 攻击生成怒气，潜地消耗怒气治疗
2. **W潜地**: 潜地后获得移速加成和感知能力
3. **E真伤**: 怒气满时E变成真实伤害
4. **R标记**: 大招需要标记目标
5. **Q强化**: Q强化普攻伤害
