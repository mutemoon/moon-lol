# Zoe (佐杨)

## 基础信息

- **称号**: 佐杨 (The Aspect of Twilight)
- **类型**: 魔法 / 远程
- **难度**: 3/3
- **角色**: 法师、刺客
- **位置**: 中单

## 技能介绍

### 被动 - 爆炸星尘 (More Sparkles)

**机制**: Zoe施放技能后强化下次普攻。

**伤害**: 20/30/40/50 (+ 25% AP)
**附加伤害**: 对低生命值目标造成额外伤害

### Q - 非凡之箭 (Paddle Star)

- **冷却**: 8/7/6/5/4 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 800

Zoe投掷非凡之箭。

**伤害**: 50/80/110/140/170 (+ 45% AP)
**转向**: 可以通过再次点击改变方向

### W - 窃法魔毯 (Portal Jump)

- **冷却**: 20/18/16/14/12 秒
- **消耗**: 40 法力
- **范围**: 575

Zoe传送到目标位置。

**伤害**: 60/100/140/180/220 (+ 50% AP)
**拾取**: 可以拾取地图上的召唤师技能和装备主动效果

### E - 气泡陷阱 (Spell Thief)

- **冷却**: 16/14/12/10/8 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 800

Zoe投掷气泡陷阱。

**伤害**: 60/100/140/180/220 (+ 50% AP)
**眩晕**: 1.3-2.1秒 (基于已飞行距离)
**易碎**: 目标受到伤害时减少护盾

### R - 折返跃迁 (Phase Dive)

- **冷却**: 22/18/14 秒
- **消耗**: 60 法力
- **范围**: 575

Zoe快速跃迁到目标位置再返回。

**伤害**: 70/110/150 (+ 35% AP)
**返回**: Zoe回到原位置

## 连招技巧

### 基础连招

**对线连招**:
- Q(非凡之箭) → E(气泡陷阱) → Q(非凡之箭)
- 超远距离消耗

**击杀连招**:
- E(气泡陷阱) → Q(非凡之箭) → R(折返跃迁)
- 远距离击杀

### 进阶连招

**团战连招**:
- W(窃法魔毯) → E(气泡陷阱) → Q(非凡之箭) → R(折返跃迁)
- 位移控制

**反杀连招**:
- R(折返跃迁) → E(气泡陷阱) → Q(非凡之箭) → R(折返跃迁)
- 瞬间返回

## 隐藏机制

1. **被动强化**: 技能后强化普攻
2. **Q转向**: 可以改变Q的飞行方向
3. **W拾取**: 拾取地图上的技能效果
4. **E易碎**: 气泡命中后目标护盾减少
5. **R折返**: R可以快速返回原位置
