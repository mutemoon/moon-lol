#  Gangplank (普朗克)

## 基础信息

- **称号**: 普朗克 (The Saltwater Scourge)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士
- **位置**: 上单

## 技能介绍

### 被动 - 火药试炼 (Trial by Fire)

**机制**: 定期地，Gangplank的下一次普攻造成额外真实伤害。

**伤害**: 50-273 (+ 100% bonus AD) 真实伤害
**移速加成**: 15-31% (+ 1% AP)

### Q - 枪火谈判 (Parrrley)

- **冷却**: 4.5 秒
- **消耗**: 50/45/40/35/30 法力
- **范围**: 625

Gangplank向目标敌人开火，造成物理伤害。击杀目标获得金币和银蛇币。

**伤害**: 10/40/70/100/130 (+ 100% AD)
**金币奖励**: 3-7
**银蛇币**: 4-8

### W - 坏血病疗法 (Remove Scurvy)

- **冷却**: 22/20/18/16/14 秒
- **消耗**: 60/70/80/90/100 法力

Gangplank消耗大量柑橘，清除所有控制效果并治疗自己。

**治疗量**: 45/70/95/120/145 (+ 90% AP) (+ 13% 损失生命值)

### E - 火药桶 (Powder Keg)

- **冷却**: 17/16/15/14/13 秒 (可充能3-5个)
- **消耗**: 1充能
- **范围**: 1000

Gangplank放置一个火药桶，可以被普攻引爆。爆炸对敌人造成伤害和减速。

**伤害**: 普攻触发伤害
**额外伤害**: 75-155 (对英雄)
**减速**: 40-80%

### R - 加农炮弹幕 (Cannon Barrage)

- **冷却**: 160/140/120 秒
- **消耗**: 100 法力
- **范围**: 全地图

Gangplank向目标位置发射信号弹，召唤船只炮击该区域。

**每波伤害**: 40/70/100 (+ 10% AP)
**总伤害**: 480/840/1200 (+ 120% AP)
**减速**: 30%，持续0.5秒

## 连招技巧

### 基础连招

**标准消耗**:
- Q(枪火谈判) → E(火药桶) → 普攻
- Q先手，E放置火药桶

**控制连招**:
- E(火药桶) → 普攻 → Q(枪火谈判) → R(加农炮弹幕)
- E减速，Q消耗

### 进阶连招

**团战连招**:
- R(加农炮弹幕) → E(火药桶) → Q(枪火谈判) → 普攻
- R先手消耗

**追杀连招**:
- E(火药桶) → 普攻 → Q(枪火谈判) → W(坏血病疗法)
- E放置火药桶并引爆

---

## 隐藏机制

1. **Q金币**: Q击杀目标获得金币
2. **E连锁**: E火药桶可以连锁爆炸
3. **W解控**: W清除所有控制效果
4. **R全图**: R可以全地图使用
5. **被动真实伤害**: 被动普攻造成真实伤害
