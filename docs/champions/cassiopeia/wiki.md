#  Cassiopeia (卡西奥佩娅)

## 基础信息

- **称号**: 卡西奥佩娅 (The Serpent's Embrace)
- **类型**: 魔法 / 远程
- **难度**: 3/3
- **角色**: 法师
- **位置**: 中单

## 技能介绍

### 被动 - 蛇眼优雅 (Serpentine Grace)

**机制**: Cassiopeia增加所有移动速度加成的效果。

**移速加成**: 6-44% (基于等级)

### Q - 毒液冲击 (Noxious Blast)

- **冷却**: 3.5 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 850

Cassiopeia在目标位置创造毒爆，对范围内敌人造成持续伤害。

**总伤害**: 75/110/145/180/215 (+ 65% AP)
**移速加成**: 30/35/40/45/50%

### W - 瘴气 (Miasma)

- **冷却**: 24/22/20/18/16 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 700

Cassiopeia喷射毒云，对区域内敌人造成持续伤害和减速。

**每秒伤害**: 20/25/30/35/40 (+ 10% AP)
**减速**: 40/50/60/70/80%

### E - 双生毒牙 (Twin Fang)

- **冷却**: 0.75 秒
- **消耗**: 40 法力
- **范围**: 700

Cassiopeia向目标发射毒牙，对中毒目标造成额外伤害和治疗。

**基础伤害**: 52-128 (基于等级) (+ 10% AP)
**额外伤害**: 20/45/70/95/120 (+ 55% AP)
**治疗量**: 10/11.5/13/14.5/16% AP

### R - 石化凝视 (Petrifying Gaze)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 850

Cassiopeia向目标方向释放石化凝视，对敌人造成伤害。面向她的敌人被眩晕，背对她的敌人被减速。

**伤害**: 150/250/350 (+ 50% AP)
**眩晕/减速时间**: 2秒
**减速效果**: 40%

## 连招技巧

### 基础连招

**标准消耗**:
- Q(毒液冲击) → E(双生毒牙) → E(双生毒牙)
- Q先手中毒，E快速输出

**控制连招**:
- R(石化凝视) → Q(毒液冲击) → E(双生毒牙) → W(瘴气)
- R先手控制，E输出

### 进阶连招

**团战连招**:
- R(石化凝视) → W(瘴气) → Q(毒液冲击) → E(双生毒牙)
- R大范围控制

**追杀连招**:
- Q(毒液冲击) → E(双生毒牙) × 3 → W(瘴气)
- Q中毒后E快速输出

## 隐藏机制

1. **E冷却极短**: E的冷却只有0.75秒，配合中毒可以快速输出
2. **R眩晕/减速**: R对面向你的敌人造成眩晕，对背向你的敌人造成减速
3. **Q移速**: Q命中后获得移速加成
4. **W减速**: W提供强力减速效果
5. **中毒判定**: E对中毒目标造成额外伤害和治疗
