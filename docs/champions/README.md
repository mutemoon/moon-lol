## 文档结构

每个英雄文档包含以下部分：

### 基础信息
- 称号（中/英文）
- 类型（物理/魔法）
- 难度等级
- 角色定位
- 主要位置

### 技能介绍
- 被动技能详解
- Q/W/E/R 四个技能
  - 冷却、消耗、范围
  - 技能描述
  - 效果数值
  - 隐藏机制

### 连招技巧
- 基础连招
- 进阶连招
- 团战连招

### 隐藏机制
- 官方文档未明确说明的机制
- 动画取消技巧
- 技能协同效果
