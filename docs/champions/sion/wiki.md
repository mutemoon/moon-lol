#  Sion (赛恩)

## 基础信息

- **称号**: 赛恩 (The Undead Juggernaut)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 坦克、战士
- **位置**: 上单、打野

## 技能介绍

### 被动 - 死亡荣耀 (Glory in Death)

**机制**: Sion死亡后变成僵尸，继续战斗。

**复活时间**: 1.5秒
**僵尸形态**: 100%最大生命值，逐渐损失
**僵尸加成**: 攻速、吸血、移速

### Q - 毁灭粉碎 (Decimating Smash)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 45 法力
- **范围**: 500-850

Sion蓄力粉碎，造成范围伤害。

**伤害**: 40-120 (+ 40-80% AD)
**蓄力效果**: 增加伤害、击飞、眩晕
**最大蓄力**: 2秒

### W - 灵魂熔炉 (Soul Furnace)

- **冷却**: 15/14/13/12/11 秒
- **消耗**: 65/70/75/80/85 法力
- **范围**: 525

Sion获得护盾，护盾结束爆炸。

**护盾值**: 60/75/90/105/120 (+ 8-16% 最大生命值)
**爆炸伤害**: 40/65/90/115/140 (+ 40% AP) (+ 14% 最大生命值)

### E - 杀手怒吼 (Roar of the Slayer)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 35/40/45/50/55 法力
- **范围**: 800/1350

Sion发射冲击波，伤害并减速敌人。

**伤害**: 65/100/135/170/205 (+ 55% AP)
**减速**: 40/45/50/55/60%
**护甲削减**: 25%

### R - 无可阻挡的冲锋 (Unstoppable Onslaught)

- **冷却**: 140/100/60 秒
- **消耗**: 100 法力
- **范围**: 全图

Sion成为不可阻挡，冲向敌人。

**伤害**: 150-450 (+ 60-120% 额外AD)
**眩晕时间**: 0.25-1.75秒
**持续时间**: 8秒

## 连招技巧

### 基础连招

**对线连招**:
- Q(毁灭粉碎) → E(杀手怒吼) → W(灵魂熔炉)
- 蓄力控制

**团战连招**:
- R(无可阻挡的冲锋) → Q(毁灭粉碎) → E(杀手怒吼) → W(灵魂熔炉)
- 大招冲锋控制

### 进阶连招

**开团连招**:
- Q(毁灭粉碎)蓄力 → R(无可阻挡的冲锋) → E(杀手怒吼)
- 蓄力后冲锋

**控制连招**:
- E(杀手怒吼) → Q(毁灭粉碎) → W(灵魂熔炉)
- 减速后控制

## 隐藏机制

1. **被动复活**: 死亡后变成僵尸
2. **Q蓄力**: Q可以蓄力增加效果
3. **W护盾**: 护盾可以爆炸
4. **E削减**: E削减敌人护甲
5. **R冲锋**: 大招可以穿墙
