#  Alistar (牛头酋长)

## 基础信息

- **称号**: 牛头酋长 (The Minotaur)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 坦克、辅助
- **位置**: 辅助

## 技能介绍

### 被动 - 胜利怒吼 (Triumphant Roar)

**机制**: Alistar在眩晕或位移敌人、小兵死亡时获得胜利层数。达到7层时消耗所有层数治疗自己和附近友军。

**自我治疗**: 5% 最大生命值
**友军治疗**: 7% Alistar最大生命值

### Q - 大地粉碎 (Pulverize)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 375

Alistar锤击地面，对范围内敌人造成魔法伤害并击飞和眩晕1秒。

**伤害**: 60/100/140/180/220 (+ 80% AP)

### W - 野蛮冲撞 (Headbutt)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 650

Alistar向目标方向冲刺，撞击敌人时造成伤害、击退和眩晕。

**伤害**: 55/110/165/220/275 (+ 100% AP)
**击退距离**: 700
**眩晕时间**: 0.75秒

### E - 践踏 (Trample)

- **冷却**: 12/11.5/11/10.5/10 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 350

Alistar践踏地面，每0.5秒对附近敌人造成魔法伤害。被命中的敌人会获得践踏层数。5层时下一次对英雄的普攻造成额外伤害并眩晕。

**每跳伤害**: 8/11/14/17/20 (+ 7% AP)
**额外伤害**: 20-305 (基于等级) 魔法伤害
**眩晕时间**: 1秒

### R - 坚定意志 (Unbreakable Will)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力

Alistar清除所有控制效果并获得伤害减免。

**伤害减免**: 55/65/75%
**持续时间**: 7秒

## 连招技巧

### 基础连招

**标准控制**:
- W(野蛮冲撞) → Q(大地粉碎)
- W将敌人撞向己方，Q击飞眩晕

**消耗连招**:
- E(践踏) → W(野蛮冲撞) → Q(大地粉碎)
- E接近，W控制，Q击飞

### 进阶连招

**团控连招**:
- R(坚定意志) → Flash → Q(大地粉碎) → W(野蛮冲撞)
- R解除控制，闪现Q击飞

**保护连招**:
- W(野蛮冲撞) → Q(大地粉碎) → E(践踏)
- W控制敌人，Q眩晕，E持续输出

---

## 隐藏机制

1. **胜利怒吼**: 7层时自动触发治疗，团队收益高
2. **W+Q连招**: W可以将敌人撞向Q的范围
3. **E移速**: E使Alistar获得移速加成并穿越单位
4. **R免疫**: R可以清除所有现有控制效果
5. **被动回血**: 胜利怒吼的被动提供团队续航
