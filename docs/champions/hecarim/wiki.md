# 赫卡里姆 (Hecarim)

## 基础信息

- **称号**: 战争影子 (the Shadow of War)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、刺客
- **位置**: 打野

## 技能介绍

### 被动 - 暗影冲击 (Shadow of the Blade)

赫卡里姆获得基于自身移动速度的额外物理伤害。
- 额外伤害 = 移动速度 × 0.15

**隐藏机制**:
- 这个被动让赫卡里姆在冲刺时伤害更高
- 配合R的位移可以打出高额伤害

### Q - 暴走 (Rampage)

- **冷却**: 4 秒
- **消耗**: 28/26/24/22/20 法力
- **范围**: 350

赫卡里姆砍击附近所有敌人，造成物理伤害。如果命中至少一个敌人，他会增加后续暴走的伤害并降低冷却。
- 每次命中叠加伤害和冷却减免
- 最多叠加4层

**效果数值**:
- 基础伤害: 60/85/110/135/160 (+0.6 AD)
- 每层叠加: +15% 伤害，-0.5秒冷却
- 最大叠加: 4层

**隐藏机制**:
- Q的范围伤害可以击中多个敌人
- 叠加层数在Q命中后立即获得
- 层数会在不攻击后衰减

### W - 恐惧之灵 (Spirit of Dread)

- **冷却**: 16/15.5/15/14.5/14 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 525

赫卡里姆获得护甲和魔法抗性提升。赫卡里姆对附近敌人造成魔法伤害，并获得相当于这些敌人受到伤害一定比例的生命回复。
- 持续5秒
- 附近敌人受到伤害的10%转化为赫卡里姆的生命

**效果数值**:
- 护甲/魔抗加成: 20/35/50/65/80
- 每秒伤害: 20/30/40/50/60 (+0.2 AD)
- 伤害转化生命: 10/12/14/16/18%

**隐藏机制**:
- W的范围伤害可以触发贪欲九头蛇
- 生命偷取效果对附近所有敌人生效
- 护甲和魔抗加成在早期非常显著

### E - 毁灭冲锋 (Devastating Charge)

- **冷却**: 18 秒
- **消耗**: 60 法力
- **范围**: 300

赫卡里姆获得大量移动速度提升，可以穿越单位，持续一小段时间。他的下一次攻击会将目标击退，并根据冲锋距离造成额外物理伤害。
- 持续4秒
- 冲锋期间可以穿越小兵和英雄
- 冲锋距离越远，伤害越高

**效果数值**:
- 移速加成: 25/35/45/55/65%
- 额外伤害: 30/60/90/120/150 (+0.5 AD)
- 击退距离: 100

**隐藏机制**:
- E期间可以穿越地形
- E配合R可以实现超远距离冲锋
- 冲锋距离会影响额外伤害
- E可以用来穿墙

### R - 暗影冲击 (Onslaught of Shadows)

- **冷却**: 140/120/100 秒
- **消耗**: 100 法力
- **范围**: 50000

赫卡里姆召唤幽灵骑士向前冲锋，在一条直线上造成魔法伤害。冲锋结束时，赫卡里姆创造一道冲击波，使附近敌人恐惧逃跑。
- 冲锋距离极远
- 冲锋结束才造成恐惧

**效果数值**:
- 冲锋伤害: 150/250/350 (+0.8 AD)
- 恐惧时长: 0.75/1/1.25 秒
- 恐惧范围: 300

**隐藏机制**:
- R冲锋的路径是赫卡里姆到目标点的直线
- 恐惧在冲锋结束时才生效
- R+E可以在冲锋途中使用E加速
- R可以用来穿墙

## 连招技巧

### 基础连招

**Gank连招**:
- E起手加速 → R冲锋 → Q → W → 平A

**突进连招**:
- E加速 → 平A击退 → Q → W

**逃跑连招**:
- E加速 → 穿越地形 → R穿墙

### 进阶连招

**E+R取消冲锋动画**:
- E冲锋途中 → R → 取消E冲锋的后摇

**RE连招**:
- R冲锋 → E加速追击 → Q倾泻

**闪现+R**:
- 闪现调整R的角度
- 用于抓取绕后位置的敌人

**E+闪+R**:
- E冲锋 → 闪到敌人身后 → R冲锋逃离

### 团战进场

**后排切入**:
- E加速 → R冲锋到敌方后排 → Q → W

**团控连招**:
- R冲锋 → 结束恐惧 → E追击 → Q

**反手连招**:
- 等待敌人先手 → R冲锋反打

## 隐藏机制

1. **被动加速伤害**: 被动伤害与移动速度成正比
2. **E可以穿墙**: E冲锋可以穿越地形
3. **Q叠加机制**: Q命中敌人后叠加层数，增加伤害和降低冷却
4. **W生命偷取**: W对附近敌人造成的伤害会按比例转化为生命
5. **R冲锋路径**: R的冲锋路径是直线，可以穿墙
6. **R恐惧延迟**: 恐惧在冲锋结束后才生效
7. **E+Q取消**: E后接Q可以取消E的后摇
8. **Q范围伤害**: Q的范围可以击中多个敌人
9. **W护甲魔抗**: W提供的双抗加成在早期很显著
10. **R可以用来逃跑**: R的冲锋距离极远，可以用于穿墙逃跑

## 常见问题

**Q: 赫卡里姆如何快速清野？**
A: 利用Q的范围伤害和W的AOE伤害，可以快速清理野怪。

**Q: 赫卡里姆的Gank时机是什么？**
A: E加速配合R冲锋是Gank的核心，利用距离优势抓人。

**Q: 赫卡里姆如何打团？**
A: 利用E加速切入后排，R恐惧后用Q倾泻伤害。
