# 纳尔 (Gnar)

## 基础信息

- **称号**: 迷失之牙 (the Missing Link)
- **类型**: 物理 / 近战-远程混合
- **难度**: 3/3
- **角色**: 战士、坦克
- **位置**: 上单

## 技能介绍

### 被动 - 狂怒基因 (Genes)

纳尔可以通过战斗积累怒气。怒气值随时间增长，在战斗中和受到伤害时会更快积累。
- 当怒气值达到最大值时，下一个技能会将纳尔变身为巨型形态（Megar）
- Mega Gnar 拥有不同的技能效果，更肉但更笨重
- 怒气衰减：脱离战斗后怒气会逐渐降低

**变身阈值**: 100 怒气
**变身持续时间**: 约 15 秒

### Q - 回旋鏢 / 巨石投掷 (Boomerang Throw / Boulder Toss)

- **冷却**: 20/17.5/15/12.5/10 秒
- **消耗**: 0
- **范围**: 1100（小）/ 590（Mega）

**小纳尔 - 回旋鏢**:
- 投掷回旋鏢，对路径上的敌人造成物理伤害和减速
- 回旋鏢会返回，捡起后减少冷却时间
- 最多命中3个目标

**Mega Gnar - 巨石投掷**:
- 投掷巨石，在第一个击中的单位处停止
- 对周围所有敌人造成物理伤害和减速
- 可以捡起巨石减少冷却

**效果数值**:
- 小纳尔伤害: 5/35/65/95/125 (+0.4 AD)
- 减速: 15/20/25/30/35%，持续 2 秒
- Mega 伤害: 5/55/105/155/205 (+0.6 AD)
- Mega 减速: 45/50/55/60/65%，持续 2 秒

**隐藏机制**:
- 小纳尔的Q可以穿透小兵
- Q的回旋轨迹可以被走位躲避
- Mega Q捡起后减少50%冷却

### W - 亢奋 / 敲击 (Hyper / Wallop)

- **冷却**: 7 秒
- **消耗**: 0
- **范围**: 0

**小纳尔 - 亢奋**:
- 纳尔的攻击和技能使他亢奋，造成额外伤害并获得移动速度
- 第三次攻击或技能命中时触发额外伤害
- 移动速度加成持续一段时间

**Mega Gnar - 敲击**:
- Mega 纳尔太过愤怒无法亢奋
- 用后腿站立，向前方区域猛击
- 对区域内敌人造成物理伤害和眩晕

**效果数值**:
- 小纳尔额外伤害: 15/30/45/60/75 (+0.06 AP) 魔法伤害
- 移动速度: 15/20/25/30/35%
- Mega 眩晕: 0.25/0.5 秒
- Mega 伤害: 50/100/150/200/250 (+0.8 AD)

**隐藏机制**:
- 小纳尔W的额外伤害可以通过E和Q触发
- Mega W的眩晕时间很短，需要预判
- W的亢奋效果可以叠加

### E - 跳跃 / 碎裂 (Hop / Crunch)

- **冷却**: 22/19.5/17/14.5/12 秒
- **消耗**: 0
- **范围**: 475

**小纳尔 - 跳跃**:
- 向一个位置跳跃
- 如果落在敌人头上，可以再次弹跳，跳跃更远的距离
- 弹跳可以造成额外伤害

**Mega Gnar - 碎裂**:
- Mega 纳尔太大无法弹跳
- 降落在目标位置，在周围区域造成伤害

**效果数值**:
- 小纳尔伤害: 50/75/100/125/150 (+0.4 AD)
- 弹跳伤害: 50/75/100/125/150 (+0.4 AD)
- Mega 伤害: 50/100/150/200/250 (+0.6 AD)

**隐藏机制**:
- 小纳尔E可以跳眼、跳小兵、跳英雄
- E跳英雄可以实现超远距离位移
- E的弹跳可以穿越地形
- Mega E的范围伤害很大

### R - 呐啊！ (GNAR!)

- **冷却**: 90/60/30 秒
- **消耗**: 0
- **范围**: 590

Mega Gnar 将周围的敌人向一个方向投掷，造成伤害并减速。任何撞到墙的敌人都会被眩晕并受到额外伤害。
- 只有 Mega Gnar 可以使用
- 投掷方向由鼠标位置决定
- 撞墙的敌人受到双倍伤害和眩晕

**效果数值**:
- 伤害: 200/300/400 (+0.8 AD)
- 减速: 45/55/65%，持续 2.5 秒
- 眩晕（撞墙）: 1.25/1.5/1.75 秒
- 撞墙额外伤害: 100/150/200

**隐藏机制**:
- R可以将敌人投掷到墙上造成眩晕
- R的范围是360度，但方向由鼠标决定
- R可以配合Mega E使用
- R可以用来保护队友，将敌人从己方后排投开
- 被投掷的敌人会获得短暂的减速效果

## 形态切换机制

### 小纳尔特点
- 灵活，位移能力强
- 远程攻击（平A是远程）
- 机动性好，可以跳小兵/英雄/眼
- 伤害输出相对较低

### Mega Gnar 特点
- 高坦度，额外生命值
- 近战攻击
- 技能伤害更高
- 更笨重，移速较慢
- 拥有强大的团控能力

### 怒气管理
- 怒气在脱战后衰减
- 可以通过主动掉血来积攒怒气
- 变身的时机是团战胜负的关键

## 连招技巧

### 基础连招（小纳尔）

**消耗连招**:
- Q → 走位捡回 → W → 拉开
- E跳英雄近身 → 平A → Q → E跳走

**追击连招**:
- E跳小兵/英雄 → 弹跳接近 → Q → 平A → W

### Mega Gnar 连招

**团控连招**:
- E进入战场 → R投掷到墙上 → W敲击 → Q巨石
- 目标是尽可能多的人撞墙

**E+R连招**:
- E落地 → 立即R → 将敌人投掷到地形上眩晕

**闪现+R**:
- Mega 状态下可以闪现调整R的角度

### 变身时机

**何时变身**:
- 团战即将爆发时变身入场
- 需要用R控制时
- 追杀重要目标时

**保持小纳尔**:
- 对线期保持小纳尔形态更灵活
- 需要快速支援时保持小纳尔

## 隐藏机制

1. **Q回旋轨迹**: 回旋鏢的回旋路径可以被走位躲避
2. **E跳眼穿墙**: 小纳尔E可以跳向墙边的眼来穿越地形
3. **怒气溢出**: 怒气达到100后不会立即变身，需要等下一个技能
4. **Mega E范围**: Mega E的伤害范围比视觉效果大
5. **R投掷优先级**: R会将所有击中的敌人向鼠标方向投掷
6. **WHyper叠加**: WHyper可以叠加，但有内置冷却
7. **变身无敌帧**: 变身瞬间有短暂无敌（无法验证）
8. **Q减速可以叠加**: Mega Q的减速可以叠加
9. **E弹跳高度判定**: E弹跳需要落在目标"头上"，有时会被阻挡
10. **Mega R墙体判定**: R的眩晕要求敌人真正撞到墙，不是地形边缘

## 常见问题

**Q: 纳尔如何快速积攒怒气？**
A: 主动承受伤害可以快速积攒怒气，但要注意不要被消耗太多血量。

**Q: Mega 纳尔和小的纳尔如何选择？**
A: 团战需要控制时变身，对线消耗保持小形态。

**Q: 纳尔的Q如何使用？**
A: 小纳尔Q是远程消耗技能，Mega Q是近距离范围伤害。Mega Q要注意捡起冷却。
