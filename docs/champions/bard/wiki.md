#  Bard (琴瑟仙女)

## 基础信息

- **称号**: 琴瑟仙女 (The Wandering Caretaker)
- **类型**: 魔法 / 远程
- **难度**: 3/3
- **角色**: 辅助、法师
- **位置**: 辅助

## 技能介绍

### 被动 - 时光乐章 (Traveler's Call)

**机制**: 地图上随机生成"音符"，收集音符恢复法力并获得经验。收集音符还会增强Meep小兵。

**收集效果**:
- 恢复12%最大法力
- 获得20+经验值
- 获得24%脱战移速加成

**Meep强化**:
- 5个音符: Meep减速敌人
- 15个音符: Meep造成范围伤害
- 35个音符: 范围增大

### Q - 宙始之弦 (Cosmic Binding)

- **冷却**: 11/10/9/8/7 秒
- **消耗**: 60 法力
- **范围**: 850

Bard向目标方向发射能量弹，对首个敌人造成伤害和减速。如果命中地形或第二个敌人，眩晕所有目标。

**伤害**: 80/120/160/200/240 (+ 80% AP)
**减速**: 60%，持续1-1.8秒
**眩晕时间**: 1/1.2/1.4/1.6/1.8 秒

### W - 守护者之赞 (Caretaker's Shrine)

- **冷却**: 18 秒 (可充能2次)
- **消耗**: 70 法力
- **范围**: 800

Bard在目标位置创造一个神殿，5秒后充满能量。友军经过时获得治疗和移速加成，敌军摧毁神殿。

**最小治疗**: 25/50/75/100/125 (+ 40% AP)
**移速加成**: 20/22.5/25/27.5/30%

### E - 神奇旅程 (Magical Journey)

- **冷却**: 22/20.5/19/17.5/16 秒
- **消耗**: 30 法力
- **范围**: 900

Bard打开一个穿越地形的单向通道，友军通过时移速增加。

**通道移速**: 33%加速

### R - 时空速结 (Tempered Fate)

- **冷却**: 110/95/80 秒
- **消耗**: 100 法力
- **范围**: 3400

Bard向目标区域发送能量，使所有单位进入凝滞状态2.5秒。英雄、小兵和防御塔都会被影响。

**效果**: 全体凝滞2.5秒

## 连招技巧

### 基础连招

**标准控制**:
- Q(宙始之弦) → W(守护者之赞)
- Q先手控制，W治疗

**保护连招**:
- E(神奇旅程) → W(守护者之赞)
- E逃跑，W治疗

### 进阶连招

**团控连招**:
- R(时空速结) → Q(宙始之弦)
- R先手大范围控制

**追击连招**:
- E(神奇旅程) → Q(宙始之弦) → W(守护者之赞)
- E穿越地形，Q控制

---

## 隐藏机制

1. **音符收集**: 音符是Bard的核心机制，影响Meep强度
2. **Q二段**: Q命中小兵后继续飞行，可以晕眩多个敌人
3. **W充能**: W有2层充能，可以连续放置两个神殿
4. **E通道**: E创造的通道是单向的，但友军可以通过
5. **R范围**: R可以影响防御塔和史诗级野怪
6. **Meep强化**: 收集更多音符可以显著增强Meep
