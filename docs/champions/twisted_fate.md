# Twisted Fate (崔斯特)

## 基础信息

- **称号**: 崔斯特 (The Card Master)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、射手
- **位置**: 中单、ADC

## 技能介绍

### 被动 - 命运骰 (Loaded Dice)

**机制**: Twisted Fate击杀敌人获得金币加成。

**金币加成**: 1-6 (基于概率)

### Q - 万能牌 (Wild Cards)

- **冷却**: 6/5.75/5.5/5.25/5 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 1450

Twisted Fate投掷三张卡牌。

**伤害**: 60/105/150/195/240 (+ 50% 额外AD) (+ 85% AP)

### W - 选牌 (Pick a Card)

- **冷却**: 6 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 325

Twisted Fate选择一张卡牌。

**蓝牌**: 伤害+回蓝
- 伤害: 40/60/80/100/120 (+ 100% AD) (+ 100% AP)
- 回蓝: 70/90/110/130/150

**红牌**: 范围伤害+减速
- 伤害: 30/45/60/75/90 (+ 100% AD) (+ 70% AP)
- 减速: 30/35/40/45/50%

**金牌**: 伤害+眩晕
- 伤害: 15/22.5/30/37.5/45 (+ 100% AD) (+ 50% AP)
- 眩晕: 1/1.25/1.5/1.75/2秒

### E - 叠加 (Stacked Deck)

**被动**: Twisted Fate获得攻速加成。

**攻速加成**: 15/25/35/45/55%
**叠层伤害**: 65/90/115/140/165 (+ 20% 额外AD) (+ 40% AP)

### R - 命运 (Destiny)

- **冷却**: 170/140/110 秒
- **消耗**: 100 法力
- **范围**: 全图

Twisted Fate标记敌人并获得传送。

**持续时间**: 6/8/10秒
**传送时间**: 1.5秒

## 连招技巧

### 基础连招

**对线连招**:
- W(选牌) → 金牌 → Q(万能牌)
- 控制后消耗

**消耗连招**:
- Q(万能牌) → W(选牌) → 蓝牌
- 远程消耗

### 进阶连招

**单杀连招**:
- R(命运) → W(选牌) → 金牌 → Q(万能牌)
- 大招支援后控制

**团战连招**:
- R(命运) → W(选牌) → 蓝牌 → Q(万能牌)
- 全球支援

## 隐藏机制

1. **W选牌**: 三张卡牌效果不同
2. **金牌控制**: 金牌眩晕敌人
3. **R全图**: R可以全图传送
4. **E叠加**: 叠满3层有额外伤害
5. **蓝牌回蓝**: 蓝牌可以回蓝
