# 李青 (Lee Sin)

## 基础信息

- **称号**: 盲僧 (the Blind Monk)
- **类型**: 物理 / 近战
- **难度**: 3/3
- **角色**: 战士、刺客
- **位置**: 打野

## 技能介绍

### 被动 - 铁血/控风 (Flurry / Wind)

李青释放技能后的2次平A获得攻速加成。
- 每次释放技能后获得2层充能
- 每层充能提供一次额外平A
- 额外平A获得额外伤害

**隐藏机制**:
- 被动提供的攻速加成让李青在技能后可以快速打出伤害
- 可以用来快速清野

### Q -  Sonic / 共鸣 (Sonic Wave / Resonating Strike)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 50 法力
- **范围**: 1100

**Sonic Wave**:
- 李青释放一道音波，定位敌人并造成物理伤害
- 如果命中，李青可以在3秒内施放共鸣

**Resonating Strike**:
- 李青向被Sonic命中的敌人冲刺
- 造成基于目标已损失生命值的物理伤害

**效果数值**:
- Sonic伤害: 55/75/95/115/135 (+0.4 AD)
- 共鸣伤害: 55/75/95/115/135 (+0.4 AD) (+6/7/8/9/10% 目标已损失生命值)

**隐藏机制**:
- Sonic可以击中隐身单位
- 共鸣可以追杀低血量敌人
- Q的第二段享受目标已损失生命值加成

### W - 防御/铁头 (Safeguard / Iron Will)

- **冷却**: 12 秒
- **消耗**: 50 法力
- **范围**: 700

**Safeguard**:
- 李青向目标友军冲刺，为自己提供护盾
- 如果目标是英雄，也会为该友军提供护盾
- 使用后3秒内可以施放铁头

**Iron Will**:
- 李青在战斗中成长，获得生命偷取和法术吸血
- 持续4秒

**效果数值**:
- 护盾值: 40/80/120/160/200 (+0.8 AD)
- 生命偷取: 10/14/18/22/26%
- 法术吸血: 5/7/9/11/13%

**隐藏机制**:
- W可以跳眼、跳小兵、跳友军英雄
- W+眼是李青的核心位移技巧
- 护盾可以用来挡技能

### E - 拍击/虚弱 (Tempest / Cripple)

- **冷却**: 9 秒
- **消耗**: 50 法力
- **范围**: 425

**Tempest**:
- 李青击打地面，释放冲击波，对命中敌人造成魔法伤害并暴露他们
- 如果命中，可以在3秒内施放虚弱

**Cripple**:
- 李青虚弱被Tempest命中的附近敌人，减少他们的移动速度
- 移速在持续期间逐渐恢复

**效果数值**:
- Tempest伤害: 70/115/160/205/250 (+1.0 AD)
- 虚弱减速: 20/30/40/50/60%

**隐藏机制**:
- E可以暴露隐身单位
- E的范围伤害可以清野
- E配合Q可以增加命中率

### R - 猛龙摆尾 (Dragon's Rage)

- **冷却**: 110/85/60 秒
- **消耗**: 0
- **范围**: 375

李青施展强力的回旋踢，将目标踢向后方，对目标和目标碰撞到的所有敌人造成物理伤害。被目标碰撞的敌人会被击飞。
- 可以用目标碰撞其他敌人
- 击退距离约600

**效果数值**:
- 伤害: 200/400/600 (+2.0 AD)
- 碰撞伤害: 100/200/300 (+1.0 AD)
- 碰撞击飞: 0.5 秒

**隐藏机制**:
- R可以配合Q和E使用
- R可以用来保护队友，将敌人踢开
- R可以用来抢龙/抢buff
- R的击退可以用来打断敌方技能

## 核心技巧

### 眼位跳 (Ward Hop)

李青可以在释放W的瞬间插眼并跳向眼位。
- W落地前插眼 → 跳向眼位
- 可以穿越地形
- 可以在空中插眼

### Insec Kick

李青最经典的技巧，将敌人踢向己方阵容。
- Q命中 → W跳眼 → R踢回
- Q命中 → 插眼 → W跳眼 → R踢回
- 需要练习眼位和角度

### 伤害最大化连招

**标准伤害连招**:
- Q1 → 平A → W跳眼 → 平A → Q2 → 平A → E → 平A → R → 平A

**闪现+R**:
- 闪现调整R的角度
- 可以将敌人踢向己方阵容

**R+闪+R**:
- R踢出 → 闪现到另一位置 → R继续踢

## 连招技巧

### 基础连招

**消耗连招**:
- Q1 → 平A → W跳眼 → 平A → Q2

**Gank连招**:
- W跳眼接近 → 平A → E → Q1 → 平A → Q2

**击杀连招**:
- Q1 → 平A → W跳眼 → 平A → Q2 → 平A → E → R

### Insec 连招

**标准Insec**:
- Q1命中 → 插眼 → W跳眼 → R踢回 → 平A → Q2

**快速Insec**:
- Q1命中 → W跳眼 → R踢回（不用平A）

**绕后Insec**:
- 从后方绕 → 插眼 → W跳眼 → R踢回

### 团战技巧

**切后排**:
- W跳眼 → R踢回敌方后排 → Q追击

**保护队友**:
- R踢开敌方刺客 → W护盾

**抢龙/抢buff**:
- Q1 → 惩戒 → W跳眼 → Q2斩杀

## 隐藏机制

1. **W跳眼位移**: W可以跳向眼位，包括敌方眼
2. **Q第二段斩杀**: Q2的伤害基于目标已损失生命值
3. **E暴露隐身**: E可以暴露隐身单位
4. **R击退控制**: R可以将敌人击退约600距离
5. **被动攻速**: 释放技能后获得2次攻速加成的平A
6. **W护盾**: W为李青和目标友军提供护盾
7. **Q可以探隐**: Q可以击中隐身单位
8. **R可以打断**: R可以打断引导技能
9. **W+眼穿墙**: 在墙边插眼并跳向眼位可以穿越地形
10. **Q+W无后摇**: Q接W可以取消Q的后摇

## 常见问题

**Q: 李青如何快速清野？**
A: 利用Q的范围伤害和W的护盾，以及被动的攻速加成。

**Q: 李青的核心技巧是什么？**
A: 眼位跳和Insec踢是李青的核心技巧，需要大量练习。

**Q: 李青如何打团？**
A: 利用眼位跳接近后排，Insec踢将敌人踢向己方阵容。
