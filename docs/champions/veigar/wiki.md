# Veigar (维迦)

## 基础信息

- **称号**: 维迦 (The Tiny Master of Evil)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、刺客
- **位置**: 中单

## 技能介绍

### 被动 - 现象级邪恶力量 (Phenomenal Evil Power)

**机制**: Veigar技能命中敌人获得AP加成。

**AP加成**: 每层+1技能伤害
**参与击杀**: 获得5层

### Q - 邪恶冲击 (Baleful Strike)

- **冷却**: 6/5.5/5/4.5/4 秒
- **消耗**: 30/35/40/45/50 法力
- **范围**: 1050

Veigar发射邪恶能量。

**伤害**: 80/120/160/200/240 (+ 50-70% AP)
**击杀加成**: 击杀获得层数

### W - 黑暗物质 (Dark Matter)

- **冷却**: 8-0 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 950

Veigar召唤黑暗物质。

**伤害**: 85/140/195/250/305 (+ 70-110% AP)
**延迟**: 1.221秒
**冷却减免**: 基于层数减少

### E - 事件视界 (Event Horizon)

- **冷却**: 20/18.5/17/15.5/14 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 725

Veigar创造一个禁锢区域。

**眩晕时间**: 1.5/1.75/2/2.25/2.5秒
**区域大小**: 411范围

### R - 原始爆发 (Primordial Burst)

- **冷却**: 120/90/60 秒
- **消耗**: 100 法力
- **范围**: 650

Veigar对目标发射原始能量。

**伤害**: 175/250/325 (+ 65-75% AP)
**斩杀加成**: 低血量敌人伤害增加

## 连招技巧

### 基础连招

**对线连招**:
- Q(邪恶冲击) → E(事件视界) → W(黑暗物质)
- 控制后输出

**消耗连招**:
- W(黑暗物质) → Q(邪恶冲击) → E(事件视界)
- 远程消耗

### 进阶连招

**团战连招**:
- E(事件视界) → W(黑暗物质) → R(原始爆发) → Q(邪恶冲击)
- 团控输出

**单杀连招**:
- Q(邪恶冲击) → E(事件视界) → R(原始爆发)
- 爆发伤害

---

## 隐藏机制

1. **被动AP**: 层数越多AP越高
2. **W延迟**: W有1秒多延迟
3. **E控制**: E可以阻挡敌人
4. **R斩杀**: R对低血量敌人伤害翻倍
5. **Q击杀**: Q击杀获得额外层数
