# Zyra (荆棘之兴)

## 基础信息

- **称号**: 荆棘之兴 (Rise of the Thorns)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、辅助
- **位置**: 辅助、中单

## 技能介绍

### 被动 - 荆棘花园 (Garden of Thorns)

**机制**: Zyra周围会周期性地生长植物。

**机制**: Zyra击杀敌人后在死亡位置生成植物
**植物持续时间**: 6秒

### Q - 致命绽放 (Deadly Bloom)

- **冷却**: 7/6.5/6/5.5/5 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 800

Zyra在目标位置创造植物。

**伤害**: 60/95/130/165/200 (+ 50% AP)
**植物伤害**: 基础植物攻击

### W - 生长之种 (Rampant Thorns)

- **冷却**: 0.5 秒
- **消耗**: 20 法力
- **范围**: 850

Zyra种植一颗种子。

**种子持续时间**: 30秒
**种子数量上限**: 3/4/5/6/7
**冷却**: 冷却时间基于Q技能等级

### E - 缠绕之根 (Strangleweeds)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 70/75/80/85/90 法力
- **范围**: 1100

Zyra召唤缠绕之根。

**伤害**: 60/105/150/195/240 (+ 60% AP)
**禁锢**: 1/1.25/1.5/1.75/2秒
**击退**: 植物会被击退

### R - 荆棘呼啸 (Strangleweed Finale)

- **冷却**: 130/115/100 秒
- **消耗**: 100 法力
- **范围**: 700

Zyra召唤巨大的荆棘浪潮。

**伤害**: 180/265/350 (+ 70% AP)
**击飞**: 0.5秒
**范围**: 宽度500

## 连招技巧

### 基础连招

**对线连招**:
- W(生长之种) → Q(致命绽放) → E(缠绕之根)
- 植物控制

**消耗连招**:
- E(缠绕之根) → W(生长之种) → Q(致命绽放)
- 禁锢后植物输出

### 进阶连招

**团战连招**:
- R(荆棘呼啸) → W(生长之种) → Q(致命绽放) → E(缠绕之根)
- 大招起手

**保护连招**:
- W(生长之种) → R(荆棘呼啸) → Q(致命绽放)
- 植物保护

---

## 隐藏机制

1. **被动植物**: 击杀敌人后生成植物
2. **Q植物**: Q创造植物攻击敌人
3. **W种子**: W种植种子，可被Q/E激活
4. **E禁锢**: E禁锢并击退敌人和植物
5. **R击飞**: R击飞区域内敌人
