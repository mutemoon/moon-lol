#  Fizz (菲兹)

## 基础信息

- **称号**: 菲兹 (The Tidal Trickster)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 刺客
- **位置**: 中单

## 技能介绍

### 被动 - 灵活战士 (Nimble Fighter)

**机制**: Fizz永久处于幽灵状态，减少所有受到伤害的4-14点。

**伤害减免**: 4 (+ 1% AP)，最多50%
**野怪额外减免**: 14 (+ 1% AP)

### Q - 海石冲刺 (Urchin Strike)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 50 法力
- **范围**: 550

Fizz向目标敌人突进，造成魔法伤害和100% AD物理伤害。

**伤害**: 10/25/40/55/70 (+ 55% AP)

### W - 海皇三叉戟 (Seastone Trident)

- **冷却**: 7/6/5/4/3 秒
- **消耗**: 30/40/50/60/70 法力

被动: Fizz的普攻造成持续魔法伤害。
主动: 强化下一次普攻，造成额外魔法伤害。

**被动总伤害**: 30/45/60/75/90 (+ 25% AP)
**主动伤害**: 50/75/100/125/150 (+ 45% AP)

### E - 顽皮 (Playful)

- **冷却**: 16/14/12/10/8 秒
- **消耗**: 75/80/85/90/95 法力
- **范围**: 400

Fizz突进到目标位置，变得无法被选取，在三叉戟上保持0.75秒，然后落下造成范围伤害和减速。

**伤害**: 80/130/180/230/280 (+ 95% AP)
**减速**: 40/45/50/55/60%

### R - 深海之噬 (Chum the Waters)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 1300

Fizz向目标位置投下诱饵，召唤鲨鱼。对敌人造成伤害、击退和减速。诱饵飞行距离越远，鲨鱼越大。

**伤害**: 180-630 (+ 60-90% AP)
**减速**: 40-80%

## 连招技巧

### 基础连招

**标准爆发**:
- Q(海石冲刺) → E(顽皮) → W(海皇三叉戟) → R(深海之噬)
- Q接近，E躲避

**消耗连招**:
- R(深海之噬) → Q(海石冲刺) → E(顽皮)
- R先手标记

### 进阶连招

**团战连招**:
- E(顽皮) → R(深海之噬) → Q(海石冲刺) → W(海皇三叉戟)
- E接近，R控制

**追杀连招**:
- Q(海石冲刺) → W(海皇三叉戟) → E(顽皮)
- Q追击，W输出

---

## 隐藏机制

1. **E无法选取**: E让Fizz无法被选取
2. **被动减速**: W被动对敌人造成持续伤害
3. **R飞行距离**: R的伤害基于诱饵飞行距离
4. **Q位移**: Q是Fizz的主要位移技能
5. **被动减伤**: Fizz受到所有伤害都会减少
