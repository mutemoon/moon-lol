#  Braum (布隆)

## 基础信息

- **称号**: 布隆 (The Heart of the Freljord)
- **类型**: 魔法 / 近战
- **难度**: 1/3
- **角色**: 坦克、辅助
- **位置**: 辅助

## 技能介绍

### 被动 - 震荡猛击 (Concussive Blows)

**机制**: Braum的普攻和技能对敌人施加震荡猛击层数。4层时触发，造成魔法伤害和眩晕。

**伤害**: 26-216 (基于等级) 魔法伤害
**眩晕时间**: 1.25-1.75 (基于等级) 秒

### Q - 寒冬之咬 (Winter's Bite)

- **冷却**: 8/7.5/7/6.5/6 秒
- **消耗**: 45/50/55/60/65 法力
- **范围**: 1050

Braum从盾牌发射冰块，对首个敌人造成伤害和减速。

**伤害**: 75/120/165/210/255 (+ 2.5% 最大生命值)
**减速**: 70%，持续2秒

### W - 挺身而出 (Stand Behind Me)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 40 法力
- **范围**: 650

Braum跳到目标友军旁边，为自己和该友军提供护甲和魔抗加成。

**护甲加成**: 20/25/30/35/40 (+ 12% 额外护甲)
**魔抗加成**: 20/25/30/35/40 (+ 12% 额外魔抗)

### E - 坚不可摧 (Unbreakable)

- **冷却**: 16/14/12/10/8 秒
- **消耗**: 30/35/40/45/50 法力

Braum举起盾牌朝向目标方向，拦截所有敌方弹道并减少通过盾牌的伤害。

**伤害减免**: 35/40/45/50/55%
**持续时间**: 3/3.25/3.5/3.75/4 秒

### R - 冰川裂隙 (Glacial Fissure)

- **冷却**: 130/115/100 秒
- **消耗**: 100 法力
- **范围**: 1200

Braum跃起并锤击地面，创造一道裂缝对敌人造成伤害和击飞，并在路径上留下减速冰面。

**伤害**: 150/250/350 (+ 60% AP)
**击飞时间**: 0.6-2秒
**减速**: 40/50/60%

## 连招技巧

### 基础连招

**标准控制**:
- Q(寒冬之咬) → 普攻 × 4 → 被动眩晕
- Q先手减速，配合普攻叠加被动

**保护连招**:
- W(挺身而出) → E(坚不可摧)
- W跳向队友，E举起盾牌保护

### 进阶连招

**团控连招**:
- Q(寒冬之咬) → R(冰川裂隙) → W(挺身而出)
- Q先手，R大范围击飞

**反手连招**:
- E(坚不可摧) → Q(寒冬之咬) → R(冰川裂隙)
- E抵挡技能，Q反击

---

## 隐藏机制

1. **被动眩晕**: 4层被动触发眩晕，是Braum的控制来源
2. **Q减速**: Q提供强力减速，方便叠加被动
3. **E盾牌**: E可以拦截所有弹道技能
4. **W双抗**: W为队友和自己提供双抗加成
5. **R击飞**: R根据距离决定击飞时长
