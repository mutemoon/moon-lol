# 凯隐 (Kayn)

## 基础信息

- **称号**: 暗影收割者 (the Shadow Reaper)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、刺客
- **位置**: 打野

## 技能介绍

### 被动 - 暗裔之镰 (The Darkin Scythe)

凯隐携带一把远古武器，内部有暗裔 拉亚斯特 正在争夺控制权。
- 凯隐在战斗中与敌人对抗，决定谁能掌控战局
- 变身形态取决于前几分钟对英雄造成的伤害类型

**两种形态**:

**暗裔杀手（红凯）**:
- 施法伤害的一部分转化为治疗
- Q造成基于目标最大生命值的额外伤害
- R造成基于目标最大生命值的伤害并治疗

**暗影刺客（蓝凯）**:
- 前几秒与敌方英雄战斗时造成额外伤害
- W期间可以移动
- R施放距离更远，冷却更短

### Q - 收割斩 (Reaping Slash)

- **冷却**: 7/6.5/6/5.5/5 秒
- **消耗**: 40 法力
- **范围**: 350

凯隐冲刺并挥舞镰刀，对穿过的敌人造成物理伤害，然后对附近敌人再次造成伤害。

**红凯强化**:
- 造成额外最大生命值物理伤害
- 额外伤害基于已损失生命值

**效果数值**:
- 基础伤害: 75/105/135/165/195 (+0.85 AD)
- 红凯额外伤害: 8/9/10/11/12% 最大生命值

**隐藏机制**:
- Q可以穿越地形
- Q对沿途敌人造成两段伤害
- Q配合E可以超远距离位移

### W - 刀锋之掠 (Blade's Reach)

- **冷却**: 13/12/11/10/9 秒
- **消耗**: 55/50/45/40/35 法力
- **范围**: 700

凯隐向上挥砍，造成物理伤害并减速。

**蓝凯强化**:
- W期间可以移动
- W范围增加

**红凯强化**:
- 额外击飞命中敌人0.75秒

**效果数值**:
- 伤害: 85/130/175/220/265 (+1.1 AD)
- 减速: 90%，持续1.5秒
- 击飞: 0.75秒（红凯）

**隐藏机制**:
- W有约0.25秒前摇
- 蓝凯W期间移动是核心机制
- W可以配合E穿墙

### E - 影步 (Shadow Step)

- **冷却**: 21/19/17/15/13 秒
- **消耗**: 40 法力
- **范围**: 400

凯隐获得移速加成，变为幽灵状态，可以穿越地形，持续7-9.5秒。
- 第一次进入地形时回复生命值
- 被控制或连续1.5秒不在地形中则技能结束

**蓝凯强化**:
- 额外70%移速
- 免疫减速
- 冷却大幅减少

**效果数值**:
- 移速加成: 40%
- 持续时间: 7/7.5/8/8.5/9/9.5 秒
- 回复生命值: 80/90/100/110/120/130 (+0.4 AP)

**隐藏机制**:
- E是凯隐的核心位移技能
- E可以穿越所有墙体
- 蓝凯E期间免疫减速，追击极强
- 红凯E主要用于刷野和调整位置

### R - 暗影寄生 (Umbral Trespass)

- **冷却**: 120/100/80 秒
- **消耗**: 40 法力
- **范围**: 550

被动：被凯隐伤害的英雄会被标记3.15秒。
主动：凯隐寄生于被标记的敌人，变为不可选中，2.5秒后爆发造成伤害。

**蓝凯强化**:
- R施放距离更远
- 爆发后距离更远
- 拉亚斯特之镰（被动）冷却刷新

**红凯强化**:
- 造成基于目标最大生命值的伤害
- 回复生命值

**效果数值**:
- 寄生持续: 2.5秒
- 伤害: 300/350/400 (+1.5 AD)
- 红凯额外伤害: 20% 最大生命值
- 红凯治疗量: 50% 伤害量

**隐藏机制**:
- R必须对有标记的敌人施放
- R期间凯隐完全不可选中
- R可以用于躲避关键技能
- R结束后凯隐会出现在目标身后

## 变身形态详解

### 变身前

凯隐在游戏前15分钟需要通过造成伤害来"解锁"形态：
- 对物理伤害英雄造成伤害 → 解锁暗影刺客
- 对魔法伤害英雄造成伤害 → 解锁暗裔杀手

**变身前特点**:
- 两种形态的强化效果都不生效
- 只能使用基础技能
- 需要尽快解锁形态

### 变身后对比

| 属性 | 暗影刺客（蓝凯） | 暗裔杀手（红凯） |
|------|------------------|------------------|
| Q额外伤害 | 无 | 最大生命值8-12% |
| W期间移动 | 可移动 | 不可移动 |
| W额外效果 | 增加范围 | 击飞0.75秒 |
| E冷却 | 大幅减少 | 正常 |
| E移速 | +70% | 正常 |
| E减速免疫 | 是 | 否 |
| R距离 | 更远 | 正常 |
| R额外效果 | 刷新被动CD | 治疗 |

## 连招技巧

### 蓝凯（暗影刺客）

**标准击杀连招**:
- E穿越地形 → W → Q → 平A → R → R重施
- E进入战场，W命中，Q追击

**E+W Combo**:
- E穿越墙体 → W（移动中） → Q
- 追击时保持移动

**远距离秒杀**:
- E → W → Q → R → 平A → R重施
- 利用R躲避技能同时击杀

**团战切入**:
- E穿越前排 → Q穿过 → R敌方后排 → 爆发 → R退出

### 红凯（暗裔杀手）

**标准对拼**:
- E → Q → W → 平A → R → R重施
- 利用R躲避伤害同时打出最大伤害

**追击连招**:
- E → Q → W（击飞） → 平A → R → 治疗 → R退出

**团战连招**:
- E穿越地形 → Q → W击飞 → 平A → R → R重施
- 利用R的治疗保持存活

### 万能连招（两种形态通用）

**基础Gank**:
- E穿越地形 → W → Q → 平A

**有R时击杀**:
- E → W → Q → 平A → R → R重施

## 隐藏机制

1. **Q两段伤害**: Q的冲刺和旋转分别造成伤害
2. **Q可以穿墙**: Q位移可以穿越薄墙
3. **W前摇**: W有约0.25秒前摇
4. **E免疫减速**: 蓝凯E期间免疫减速
5. **E进入地形回血**: 红凯E第一次穿墙回复生命
6. **R必须标记**: R只能对有标记的敌人施放
7. **R不可选中**: R寄生期间凯隐完全不可选中
8. **R退出位置**: R结束后凯隐出现在目标身后
9. **变身判定**: 游戏前15分钟决定最终形态
10. **Q配合E**: Q和E可以组合实现超远距离位移
11. **W蓝凯移动**: 蓝凯W期间可以自由移动
12. **红凯Q治疗**: 红凯Q造成额外伤害并治疗

## 常见问题

**Q: 凯隐应该怎么选择变身？**
A: 根据阵容和局势。如果敌方脆皮多，选蓝凯；如果需要续航和持续作战，选红凯。

**Q: 凯隐变身前怎么打？**
A: 变身前尽量保持两种伤害都造成一些。对线期多观察敌方英雄类型。

**Q: 蓝凯和红凯哪个更强？**
A: 蓝凯爆发更高，红凯续航更强。不同局势有不同选择。

**Q: 凯隐E怎么用最好？**
A: 蓝凯E用于追击和逃跑，红凯E主要用于刷野和调整位置。

**Q: 凯隐R怎么用？**
A: 可以用来躲避技能、击杀敌人、保命。注意R只能对有标记的目标施放。

**Q: 凯隐解锁形态需要多久？**
A: 通常需要3-5分钟，取决于造成伤害的频率。
