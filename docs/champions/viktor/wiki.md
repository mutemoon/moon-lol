# Viktor (维克托)

## 基础信息

- **称号**: 维克托 (The Herald of the Arcane)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师
- **位置**: 中单

## 技能介绍

### 被动 - 光荣进化 (Glorious Evolution)

**机制**: Viktor可以通过碎片强化技能。

**碎片获取**: 击杀敌人获得
**碎片上限**: 100点/技能

### Q - 虹吸能量 (Siphon Power)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 45/50/55/60/65 法力
- **范围**: 600

Viktor投掷虹吸装置。

**伤害**: 60/75/90/105/120 (+ 40% AP)
**护盾**: 40-140 (+ 25% AP)
**强化效果**: 加速和强化护盾

### W - 重力场 (Gravity Field)

- **冷却**: 17/16/15/14/13 秒
- **消耗**: 65 法力
- **范围**: 800

Viktor创造重力场。

**减速**: 33/36/39/42/45%
**眩晕时间**: 1.5秒 (5层时)
**强化效果**: 其他技能减速敌人

### E -  hextech Ray (Hextech Ray)

- **冷却**: 12/11/10/9/8 秒
- **消耗**: 60/70/80/90/100 法力
- **范围**: 550

Viktor发射能量射线。

**伤害**: 70/110/150/190/230 (+ 50% AP)
**强化效果**: 路径爆炸

### R - 奥术风暴 (Arcane Storm)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 700

Viktor召唤移动风暴。

**伤害**: 100/175/250 (+ 50% AP)
**每跳伤害**: 65/105/145 (+ 35% AP)
**强化效果**: 参与击杀扩大风暴

## 连招技巧

### 基础连招

**对线连招**:
- Q(虹吸能量) → E(hextech Ray) → W(重力场)
- 技能连招

**消耗连招**:
- E(hextech Ray) → Q(虹吸能量) → W(重力场)
- 远程消耗

### 进阶连招

**团战连招**:
- R(奥术风暴) → W(重力场) → Q(虹吸能量) → E(hextech Ray)
- 风暴输出

**强化连招**:
- Q(虹吸能量) → E(hextech Ray) → W(重力场) → R(奥术风暴)
- 强化所有技能

## 隐藏机制

1. **被动进化**: 三个技能可以强化
2. **Q护盾**: Q提供护盾和强化普攻
3. **W控制**: W有5层叠加眩晕
4. **E爆炸**: E强化后有二次爆炸
5. **R追踪**: R可以追踪敌人
