# Smolder (斯莫德)

## 基础信息

- **称号**: 斯莫德 (The Exemplar of the幼龙)
- **类型**: 物理 / 远程
- **难度**: 2/3
- **角色**: 射手
- **位置**: ADC

## 技能介绍

### 被动 - 幼龙吐息 (Bayoor's Breath)

**机制**: Smolder攻击积累火焰层数。

**火焰层数**: 0-100
**火焰伤害**: 1-50 (+ 10% AD) (+ 5% AP)
**强化**: 层数越高火焰伤害越高

### Q - 龙息 (Flame Breath)

- **冷却**: 6/5.5/5/4.5/4 秒
- **消耗**: 25 法力
- **范围**: 600

Smolder喷射龙息。

**伤害**: 45/65/85/105/125 (+ 40% AD)
**火焰叠加**: 5层火焰

### W - 龙卷 (Dragon)

- **冷却**: 14/13/12/11/10 秒
- **消耗**: 50 法力
- **范围**: 350

Smolder创造龙卷风。

**伤害**: 70/110/150/190/230 (+ 60% AD)
**击退**: 击退路径上的敌人

### E - 幼龙飞升 (Dragon's Greed)

- **冷却**: 22/20/18/16/14 秒
- **消耗**: 80 法力
- **范围**: 550

Smolder向目标位置俯冲。

**伤害**: 80/120/160/200/240 (+ 70% AD)
**位移**: 穿越地形
**火焰叠加**: 15层火焰

### R - 超级龙息 (Mega Dragon)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 1500

Smolder释放超级龙息。

**伤害**: 200/300/400 (+ 80% AD) (+ 60% AP)
**范围**: 全图宽度
**燃烧**: 持续燃烧伤害

## 连招技巧

### 基础连招

**对线连招**:
- Q(龙息) → W(龙卷) → E(幼龙飞升)
- 火焰叠加

**消耗连招**:
- E(幼龙飞升) → Q(龙息) → W(龙卷)
- 位移消耗

### 进阶连招

**团战连招**:
- R(超级龙息) → E(幼龙飞升) → Q(龙息) → W(龙卷)
- 大招起手

**追击连招**:
- E(幼龙飞升) → R(超级龙息) → Q(龙息)
- 位移追击

## 隐藏机制

1. **被动火焰**: 攻击积累火焰层数
2. **Q叠加**: Q技能叠加火焰层
3. **W击退**: W可以击退敌人
4. **E穿越**: E可以穿越地形
5. **R全图**: R可以攻击全图
