# Swain (斯维因)

## 基础信息

- **称号**: 斯维因 (The Noxian Grand General)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 法师、坦克
- **位置**: 中单、辅助

## 技能介绍

### 被动 - 饥饿之鸦 (Ravenous Flock)

**机制**: 敌人死亡时掉落灵魂碎片，Swain收集后获得治疗和生命值加成。

**治疗**: 6% 最大生命值
**生命值加成**: 每个碎片+15生命值
**碎片持续时间**: 16秒

### Q - 死亡之手 (Death's Hand)

- **冷却**: 7/6/5/4/3 秒
- **消耗**: 40/45/50/55/60 法力
- **范围**: 725

Swain释放五道邪能冲击波。

**伤害**: 60/90/120/150/180 (+ 45% AP)
**后续伤害**: 25%加成

### W - 帝国视界 (Vision of Empire)

- **冷却**: 22/21/20/19/18 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 5500/6000/6500/7000/7500

Swain召唤恶魔之眼，提供视野并减速敌人。

**伤害**: 70/105/140/175/210 (+ 60% AP)
**减速**: 50% (持续1.5秒)
**视野时间**: 2秒

### E - 永动 (Nevermove)

- **冷却**: 12/11.5/11/10.5/10 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 850

Swain发射恶魔波浪，束缚敌人。

**伤害**: 90/130/170/210/250 (+ 70% AP)
**束缚时间**: 1.5秒
**重施**: 可以将敌人拉回

### R - 恶魔升华 (Demonic Ascension)

- **冷却**: 120 秒
- **消耗**: 100 法力
- **范围**: 650

Swain释放恶魔之力，持续伤害并治疗。

**每跳伤害**: 7.5/12.5/17.5 (+ 2% AP)
**每跳治疗**: 7.5/15/22.5 (+ 2.5% AP) (+ 0.75% 额外生命值)
**持续时间**: 直到恶魔能量耗尽

## 连招技巧

### 基础连招

**对线连招**:
- E(永动) → Q(死亡之手) → W(帝国视界)
- 控制后输出

**消耗连招**:
- W(帝国视界) → Q(死亡之手) → E(永动)
- 远程消耗

### 进阶连招

**团战连招**:
- R(恶魔升华) → E(永动) → Q(死亡之手) → W(帝国视界)
- 大招持续输出

**反手连招**:
- E(永动) → R(恶魔升华) → Q(死亡之手)
- 控制后开大

## 隐藏机制

1. **被动收集**: 灵魂碎片可以提供视野
2. **W探照**: W可以提供全图视野
3. **E控制**: E可以束缚并拉回敌人
4. **R治疗**: 大招持续治疗
5. **恶魔能量**: 参与击杀延长持续时间
