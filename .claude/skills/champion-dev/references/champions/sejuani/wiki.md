#  Sejuani (瑟庄妮)

## 基础信息

- **称号**: 瑟庄妮 (Fury of the North)
- **类型**: 魔法 / 近战
- **难度**: 2/3
- **角色**: 坦克、战士
- **位置**: 打野、辅助

## 技能介绍

### 被动 - 北地之怒 (Fury of the North)

**机制**: Sejuani的技能对被眩晕敌人附加冰冻效果。

**冰冻伤害**: 10% 最大生命值
**冰霜护甲**: 受到伤害后获得双抗加成

### Q - 极冰冲撞 (Arctic Assault)

- **冷却**: 18/16.5/15/13.5/12 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 650

Sejuani冲锋，伤害并击飞敌人。

**伤害**: 90/140/190/240/290 (+ 75% AP)
**击飞时间**: 0.5秒

### W - 凛冬之怒 (Winter's Wrath)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 60 法力
- **范围**: 600

Sejuani挥动暴风之锤，造成范围伤害。

**伤害**: 5-85 (+ 30-60% AP) (+ 4-8% 最大生命值)
**减速**: 75% (持续0.25秒)

### E - 永冻 (Permafrost)

- **冷却**: 1.5 秒
- **消耗**: 20 法力
- **范围**: 600

Sejuani投掷冰冻陷阱，眩晕被标记的敌人。

**伤害**: 55/105/155/205/255 (+ 70% AP)
**眩晕时间**: 1秒
**冰霜层数**: 需要4层冰霜

### R - 冰川监狱 (Glacial Prison)

- **冷却**: 120/105/90 秒
- **消耗**: 100 法力
- **范围**: 1300

Sejuani投掷冰球，眩晕并创造冰冻区域。

**伤害**: 125-400 (+ 40-80% AP)
**眩晕时间**: 1-1.5秒
**冰冻区域**: 持续减速

## 连招技巧

### 基础连招

**Gank连招**:
- Q(极冰冲撞) → W(凛冬之怒) → E(永冻)
- 冲锋后持续控制

**团战连招**:
- R(冰川监狱) → Q(极冰冲撞) → W(凛冬之怒) → E(永冻)
- 大招起手团控

### 进阶连招

**控制连招**:
- E(永冻) → Q(极冰冲撞) → W(凛冬之怒)
- 冰冻后冲锋

**追击连招**:
- Q(极冰冲撞) → E(永冻) → R(冰川监狱)
- 冲锋后控制

---

## 隐藏机制

1. **冰冻效果**: 被动对眩晕敌人附加冰冻
2. **冰霜护甲**: 被动在脱战后提供双抗
3. **Q冲锋**: Q可以穿墙
4. **E冰冻**: E需要冰霜层数触发眩晕
5. **R团控**: 大招可以远程开团
