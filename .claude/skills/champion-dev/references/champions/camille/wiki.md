# 青钢影 (Camille)

## 基础信息

- **称号**: 钢铁影刃 (the Steel Shadow)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、刺客
- **位置**: 上单

## 技能介绍

### 被动 - 自适应防御 (Adaptive Defenses)

卡蜜尔对英雄的普通攻击会获得一个短暂存在的护盾，护盾值基于她的最大生命值，对抗物理或魔法伤害。
- 护盾持续2秒
- 护盾值: 6% 最大生命值

### Q - 精密协议 (Precision Protocol)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 40 法力
- **范围**: 325

卡蜜尔的下一次攻击造成额外物理伤害，并获得移动速度加成，持续1秒。这个技能可以在2.25秒内重施。

**第二段机制**:
- 如果第二段攻击在第一段后至少0.75秒命中，额外伤害提升
- 额外伤害的一部分转化为真实伤害

**效果数值**:
- 基础伤害: 60/85/110/135/160
- 第二段加成伤害: 70/115/160/205/250
- 移动速度: 40%
- AD加成: 0.9

**隐藏机制**:
- Q可以重置平A
- Q的两段都可以触发攻击特效
- Q的位移距离约为325（加上攻击距离）
- 0.75秒延迟是独立计算的，不受技能急速影响

### W - 战术横扫 (Tactical Sweep)

- **冷却**: 15/14/13/12/11 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 610

卡蜜尔蓄力后横扫，造成物理伤害。
- 内圈伤害较低
- 外圈命中敌人会减速，并受到额外最大生命值物理伤害
- 卡蜜尔回复对英雄造成外圈伤害的75%

**效果数值**:
- 内圈伤害: 80/120/160/200/240
- 外圈额外伤害: 6/7/8/9/10% 最大生命值
- 减速: 90%，持续2秒

**隐藏机制**:
- W有约0.25秒的前摇
- 外圈范围大约占整个扇形的60%
- W的回血只针对英雄，不对小兵生效
- W可以触发贪欲吸血等攻击特效

### E - 钩索 (Hookshot)

- **冷却**: 16/15/14/13/12 秒
- **消耗**: 70 法力
- **范围**: 800

卡蜜尔向地形发射钩索，附着1秒，可以重施。

**第一段**:
- 向地形飞行并附着
- 可以重施一次

**第二段**:
- 从墙壁冲向第一个命中的敌方英雄
- 造成伤害并眩晕
- 获得攻击速度加成
- 朝向敌方英雄时位移距离翻倍

**效果数值**:
- 冲刺伤害: 80/110/140/170/200
- 眩晕时间: 1秒
- 攻击速度: 50/55/60/65/70%，持续5秒

**隐藏机制**:
- E可以附着在任何地形上（防御塔、墙体、果实）
- E附着后有1秒窗口期可以重施
- 朝着敌方英雄方向E时，距离翻倍（1600范围）
- E可以被任何控制技能打断（附着期间）
- E冲刺过程中可以使用闪现（E+F combo）
- E第一段的飞行速度极快，几乎不可能被闪避

### R - 海克斯科技绝杀 (The Hextech Ultimatum)

- **冷却**: 140/115/90 秒
- **消耗**: 100 法力
- **范围**: 475

卡蜜尔短暂变为不可选中，跳向一个敌方英雄并将其困在一个区域内，持续4秒。
- 打断目标正在引导的技能
- 目标无法通过任何方式离开该区域（闪现、位移、传送都不行）
- 其他敌人被击退
- 对被困目标的攻击附加额外魔法伤害

**效果数值**:
- 持续时间: 4秒
- 额外伤害: 5/10/15% 当前生命值（每击）

**隐藏机制**:
- R不可选中窗口约0.3秒
- R可以打断引导技能（如厄运小姐大招）
- 目标被困时无法被任何技能选中（保护作用）
- 维克托的R可以用自身R带出区域
- 塔姆E可以保护队友不被困住
- 卡蜜尔自己在R期间可以被攻击

## 连招技巧

### 基础连招

**标准E连招**:
- E → E（第一段附着，第二段冲向敌人） → 平A → Q → 平A → Q → W

**W起手连招**:
- W → 平A → Q → R → E → E → 平A → Q

**远程消耗**:
- E → E（双E穿越墙体接近） → 平A → Q → Q → W → 撤退

### 进阶连招

**E+W+E Combo**:
- E第一段附着 → W蓄力 → 立即E第二段
- 这样W会在E落地位置释放，提高命中率

**E+E+F Combo**:
- E第一段 → 立即E第二段并在中途F（闪现）
- 用于超远距离突进
- F的位置决定落地位置

**RWQ连招**:
- R跳向目标 → 立即W → Q追砍
- 确保在R的4秒窗口期内打出最大伤害

### 团战进场

**完美团战**:
- E附着地形 → 观察时机 → E第二段冲向敌方后排
- R控制敌方核心 → 平A → Q → W
- 如果被困目标试图逃跑，可以再次E追击

## 隐藏机制

1. **E可附着的地形**: 包括墙体、防御塔、龙/男爵坑、小兵、果实
2. **E冲锋距离翻倍**: 朝向敌方英雄E时，实际距离约为1600
3. **Q两段分离**: Q的两段可以分开使用，不需要连续按
4. **R不可选中**: R有约0.3秒的不可选中窗口
5. **W蓄力可取消**: W蓄力时可以用E或R取消
6. **Q可以取消W后摇**: W后可以快速接Q
7. **被动护盾**: 对线期对拼时，被动护盾是重要换血资本
8. **E附着时间**: 附着时间只有1秒，需要快速决策
9. **Q第二段延迟**: 0.75秒延迟可以配合其他技能使用
10. **R困住目标**: 目标无法使用任何位移技能，但可以被推拉
11. **E可以E闪**: 在E冲刺过程中可以使用闪现改变落点
12. **W外圈判定**: 扇形外圈大约占60%的范围

## 常见问题

**Q: E可以钩哪些地形？**
A: 所有墙体、防御塔本体、龙坑/男爵坑壁、果实体、小兵和英雄（虽然不常用）。

**Q: 如何提高E的命中率？**
A: 预判敌人走位，或者利用墙体从侧后方包抄。E的飞行速度极快，近距离几乎无法反应。

**Q: R的被困效果可以被什么解除？**
A: 没有任何技能可以解除被困效果，但可以被推拉技能（如蔚的R）带出区域。

**Q: 青钢影对线怎么换血？**
A: 利用被动护盾吸收伤害，EAQA打一套然后W减速撤退。
