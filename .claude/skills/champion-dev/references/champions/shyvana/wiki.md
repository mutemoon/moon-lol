#  Shyvana (希瓦娜)

## 基础信息

- **称号**: 希瓦娜 (The Half-Dragon)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士、打野
- **位置**: 打野、上单

## 技能介绍

### 被动 - 龙鳞 (Scalemail)

**机制**: Shyvana击杀敌人获得护甲和魔抗。

**击杀叠加**: 大型野怪+1层，英雄+3层
**每层属性**: +0.3 护甲和魔抗

### Q - 灰烬燃烧 (Emberstrike)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 自助
- **范围**: 275

Shyvana强化接下来几次普攻，造成额外伤害。

**伤害**: 10-30 (+ 110% AD) (+ 25% AP)
**强化次数**: 人形态2次，龙形态3次

### W - 烈火护盾 (Inferno Aegis)

- **冷却**: 14/13.5/13/12.5/12 秒
- **消耗**: 自助
- **范围**: 350

Shyvana获得护盾，护盾结束时爆炸。

**护盾值**: 60/80/100/120/140 (+ 5% 最大生命值)
**爆炸伤害**: 80/100/120/140/160 (+ 40% 额外AD) (+ 20% AP)

### E - 熔岩爆发 (Molten Burst)

- **冷却**: 12/11.5/11/10.5/10 秒
- **消耗**: 自助
- **范围**: 150/300

Shyvana发射火球，伤害并减速敌人。

**伤害**: 80/110/140/170/200 (+ 35% 额外AD) (+ 70% AP)
**减速**: 20/25/30/35/40%

### R - 龙之 descent (Dragon's Descent)

- **冷却**: 自助
- **消耗**: 100 龙之怒气
- **范围**: 400/1050

Shyvana变身为龙，获得额外属性并恐惧敌人。

**伤害**: 150/250/350 (+ 100% AP)
**恐惧时间**: 0.75秒
**龙形态加成**: +150-350生命值，+25-75射程

## 连招技巧

### 基础连招

**人形态连招**:
- Q(灰烬燃烧) → W(烈火护盾) → E(熔岩爆发)
- 强化输出

**龙形态连招**:
- R(龙之 descent) → Q(灰烬燃烧) → E(熔岩爆发) → W(烈火护盾)
- 变身爆发

### 进阶连招

**团战连招**:
- R(龙之 descent) → Q(灰烬燃烧) → W(烈火护盾) → E(熔岩爆发)
- 变身团控

**单杀连招**:
- Q(灰烬燃烧) → E(熔岩爆发) → W(烈火护盾) → R(龙之 descent)
- 强化后变身

## 隐藏机制

1. **怒气系统**: 变身消耗怒气
2. **Q强化**: Q强化普攻伤害
3. **W护盾**: 护盾可以爆炸
4. **E减速**: E造成范围减速
5. **R变身**: 大招变身龙形态
