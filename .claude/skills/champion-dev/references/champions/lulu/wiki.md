#  Lulu (璐璐)

## 基础信息

- **称号**: 璐璐 (The Fae Sorceress)
- **类型**: 魔法 / 远程
- **难度**: 2/3
- **角色**: 辅助、法师
- **位置**: 辅助

## 技能介绍

### 被动 - Pix (皮克斯)

**机制**: Lulu有一个小精灵Pix跟随，协助攻击。

**Pix伤害**: 5-43 (+ 5% AP) 每发
**3连发**: Pix发射3发螺栓

### Q - 闪光之矛 (Glitterlance)

- **冷却**: 7 秒
- **消耗**: 50/55/60/65/70 法力
- **范围**: 950

Lulu和Pix各发射一个魔法螺栓。

**伤害**: 60/95/130/165/200 (+ 50% AP)
**减速**: 80%

### W - 奇思妙想 (Whimsy)

- **冷却**: 18 秒
- **消耗**: 65 法力
- **范围**: 650

Lulu可以变形敌人或强化友军。

**敌方施放**: 变形1.2-2秒
**友军施放**: 攻速20-30% + 移速25%

### E - 帮助，皮克斯！ (Help, Pix!)

- **冷却**: 10/9.5/9/8.5/8 秒
- **消耗**: 60/65/70/75/80 法力
- **范围**: 650

Lulu可以护盾友军或对敌人造成伤害。

**敌方伤害**: 70/110/150/190/230 (+ 50% AP)
**友军护盾**: 70/110/150/190/230 (+ 50% AP)

### R - 野性生长 (Wild Growth)

- **冷却**: 120/100/80 秒
- **消耗**: 100 法力
- **范围**: 900

Lulu增大自身或友军，击飞附近敌人。

**击飞时间**: 1秒
**额外生命值**: 275/425/575 (+ 55% AP)
**减速**: 30-60%

---

## 连招技巧

### 基础连招

**保护连招**:
- E(帮助皮克斯) → W(奇思妙想)
- 护盾后加速

**进攻连招**:
- Q(闪光之矛) → E(帮助皮克斯) → W(奇思妙想)
- 减速后变形

### 进阶连招

**团战连招**:
- R(野性生长) → Q(闪光之矛) → E(帮助皮克斯)
- 大招保护后输出

**反手连招**:
- W(奇思妙想) → R(野性生长) → Q(闪光之矛)
- 变形敌人后保护

---

## 隐藏机制

1. **Pix协助**: Pix发射额外螺栓协助攻击
2. **W双模式**: W可以对敌人或友军施放
3. **E双模式**: E可以伤害敌人或护盾友军
4. **R增大**: R增加体型和生命值
5. **Q减速**: Q提供强力减速效果
