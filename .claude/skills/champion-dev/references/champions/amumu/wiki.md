## 技能介绍

### 被动 - 诅咒之触 (Cursed Touch)

**机制**: Amumu的普攻和绷带投掷会标记敌人为"诅咒"状态，持续3秒，刷新后续应用和绝望技能的周期性伤害。诅咒目标受到所有即将到来的**魔法伤害的10%额外真实伤害**（减免前）。

### Q - 绷带射击 (Bandage Toss)

- **冷却(充能)**: 16/15/14/13/12 秒
- **消耗**: 50 法力
- **施法时间**: 0.25秒
- **范围**: 1100
- **宽度**: 160
- **速度**: 2000 / 1800

Amumu向目标方向投掷绷带，对第一个敌人造成魔法伤害、眩晕1秒并将自己拉向目标，同时目标被显形。

**伤害**: 70/95/120/145/170 (+ 85% AP)

Amumu会周期性储存绷带投掷的充能，最多2层。绝望技能可以在绷带投掷的突进过程中释放。

### W - 绝望之念 (Despair)

- **消耗**: 8 法力/秒
- **范围**: 350
- **切换技能**: 无施法时间

Amumu哭泣产生眼泪池，每0.5秒对附近敌人造成魔法伤害。

**每跳伤害**: 基于目标最大生命值的百分比 (+ 0.25% AP)

绝望会在Amumu无法支付法力成本时自动关闭。

### E - 悲伤木乃伊 (Tantrum)

- **冷却**: 9/8/7/6/5 秒
- **消耗**: 35 法力
- **施法时间**: 0.25秒
- **范围**: 350

**被动**: Amumu减少所有即将到来的物理伤害，减少量基于额外护甲和魔抗，上限为每次伤害实例的50%。

**物理伤害减少**: 5/7/9/11/13 (+ 3% 额外护甲) (+ 3% 额外魔抗)

**主动**: 对附近敌人造成魔法伤害。每次Amumu被普攻命中时，悲伤木乃伊的当前冷却时间减少0.75秒。

**伤害**: 65/95/125/155/185 (+ 50% AP)

### R - 木乃伊之咒 (Curse of the Sad Mummy)

- **冷却**: 150/125/100 秒
- **消耗**: 100/150/200 法力
- **施法时间**: 0.25秒
- **范围**: 550

Amumu诅咒并束缚附近敌人，造成魔法伤害、击飞和眩晕1.5秒。

**伤害**: 200/300/400 (+ 80% AP)

## 连招技巧

### 基础连招

**标准团控**:
- Q(绷带射击) → R(木乃伊之咒) → W(绝望之念) → E(悲伤木乃伊)
- Q先手眩晕，R大范围控制

**消耗连招**:
- Q(绷带射击) → E(悲伤木乃伊) → W(绝望之念)
- Q接近目标，E造成伤害

### 进阶连招

**团战连招**:
- R(木乃伊之咒) → Q(绷带射击) → W(绝望之念) → E(悲伤木乃伊)
- R先手大范围控制

**蹲草连招**:
- W(绝望之念) → Q(绷带射击) → R(木乃伊之咒)
- 草丛先开W持续伤害

## 隐藏机制

1. **诅咒增伤**: 被动使目标受到10%额外真实伤害（基于即将到来的魔法伤害）
2. **Q充能**: Q有2层充能，可以连续使用两次
3. **E减伤**: E被动减少物理伤害，是Amumu坦度的关键，取决于额外护甲和魔抗
4. **W消耗**: W持续消耗法力但伤害可观，会自动关闭如果法力不足
5. **R范围**: R的范围比指示器显示的更大
