# 被动技能与联动

## 职责与设计

- 实体挂载：被动技能通过 `PassiveSkillOf` 组件挂载在英雄实体上。
- 执行路径：被动技能一般不走主动施法管线，而是通过观察者机制、周期系统、伤害事件或普攻事件与主动技能进行联动。

## 联动与伤害响应模式

- 被动 Buff 消耗：主动施法时给自身附带特定被动状态组件，后续普攻触发事件中检测并消耗层数，提供额外输出或效果。
- 伤害与命中监听：通过注册监听 `EventDamageCreate` 或 `EventAttackEnd` 事件的观察者，在造成伤害时为目标挂载增减益效果、触发治疗或附加标记。

## 典型实现案例

- 锐雯被动：普攻结束事件触发时，检测并消耗被动 Buff 实体层数，附带额外物理伤害。
- 菲奥娜被动：通过周期性扫描系统，在敌方英雄身上挂载要害组件，检测伤害命中事件的发生方向以判定要害击破。
- 达勒斯被动：在伤害创建观察者中判定施法者身份，通过伤害命中事件增加出血叠层，达到上限时触发血怒属性增益 Buff。
