# 技能数值读取机制

- 资产来源：技能的所有伤害、比例系数和基础成长数值都存储在 `Spell` 资产中。
- 检索方法：为了简化在业务代码中解析资产的复杂度，系统提供了两个主要的获取助手函数。

# 属性计算值获取

- `get_skill_value`：用于获取经过公式计算并附加人物属性加成后的最终数值。
- 参数规则：传入 `Spell` 资产句柄引用、在 calculations 中定义的公式名、当前技能等级，以及一个接收属性 ID 并返回数值的回调闭包。
- 计算求和：该函数会自动解析公式中包含的基础效果值和各项属性乘数系数，执行叠加乘算并返回求和后的结果。

# 原始数据值获取

- `get_skill_data_value`：用于直接检索配置中特定的原始数值，不进行任何人物属性系数计算。
- 匹配忽略规则：参数中的键名在检索时大小写无关，并且会自动忽略所有下划线。例如传入 `CD_Refund_Percent` 与匹配 `CDRefundPercent` 等效。
- 等级映射：等级参数为当前技能学成等级。在函数内部，数组查找会自动折算为以 0 开始的索引以读取对应的数组元素。

# 读取选用规范

- calculations 公式名：若需要计算物理、魔法或真实伤害等附带人物属性系数的伤害或盾量，建议使用 `get_skill_value`。
- dataValues 数据名：若仅需获取技能内置的持续时间、基础减速比例、攻击速度提升值或冷却缩减百分比等独立参数，建议使用 `get_skill_data_value`。
