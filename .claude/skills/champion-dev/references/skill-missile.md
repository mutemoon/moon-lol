# 飞弹投射物系统

- 运行定义：通过派发 `CommandMissileCreate` 构造飞弹实体。飞弹分为跟踪型和直线型两类。
- 数据载体：飞弹运行时附加了 `MissileState` 组件，并在直线型飞弹中附加 `LinearMissile` 组件。

# 飞弹创建参数

- `CommandMissileCreate`：创建飞弹事件。
- `target`：当不为空时，创建跟踪型飞弹，追踪目标实体的骨骼节点。
- `destination`：直线型飞弹的飞往目标坐标点。
- `spell`：飞弹绑定的 `Spell` 资产句柄。
- `damage`：直线型飞弹对路径上敌人造成的真实折算物理或魔法伤害值。对于跟踪型飞弹，伤害直接由源实体的基础伤害属性读取。
- `speed`：若传入，则覆盖技能资产中的飞弹飞行速度。
- `particle_hash`：若传入，则覆盖技能资产中的飞弹粒子效果特效哈希值。

# 飞弹碰撞与局限性

- 直线飞弹碰撞：`LinearMissile` 带有伤害判定宽度和已被击中敌人实体的去重列表。飞弹在固定更新中飞行并检测与敌人的距离，碰撞后触发单体伤害并加入去重列表。
- 到达销毁：飞弹接近目标点或追踪目标实体的距离小于阈值时自动销毁实体。
- 命中回调局限：当前飞弹系统主要由底层直线移动逻辑判定伤害，未原生提供命中回调函数或闭包支持，需注意此局限性。
