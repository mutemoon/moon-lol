# 单段技能模式

- 模式特征：只有一个施法动作、一次位移或一次范围判定。施法后即进入主冷却，状态较少或没有后续的技能流转状态。
- 冷却配置：默认使用自动冷却模式 `SkillCooldownMode::AfterCast`。

# 核心开发与实现流程

- 技能注册：在英雄模块初始化时，将技能与对应的槽位及技能配置文件进行绑定。
- 观察者配置：注册监听 `EventSkillCast` 事件的观察者。
- 英雄实体过滤：在观察者系统中使用查询过滤触发的英雄实体，判断其是否带有该英雄的特定标记组件，从而拦截并防止误判。
- 动作派发：在观察者逻辑内部，根据触发的槽位，直接向世界派发播放技能动画、位移或造成伤害的原子指令。

# 典型实现参考

- 盖伦 E 技能：首段施法触发后直接播放旋转动画，同时向施法者附着一个随身持续范围伤害场，随英雄移动，持续时间结束后伤害场自动销毁，管线在施法初始即自动激活并启动冷却计时。
