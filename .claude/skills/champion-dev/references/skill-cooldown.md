# 技能冷却机制

## 冷却模式类型

- `AfterCast` 模式：施法动作完成后自动由管线启动默认冷却。适用于普通单段技能，或按下后即刻进入冷却的技能。
- `Manual` 模式：手动控制冷却的启动时机。适用于分段技能、二段技能、或在终段才开始进入主冷却的技能。

## 冷却控制

- 计时器控制：在游戏逻辑中通过 `cooldown.timer` 更新和减少冷却的剩余时间。
- 重施窗口冲突：在 `SkillRecastWindow` 激活的重施窗口期内，通用管线不会执行冷却校验。
- 刷新与减少：命中刷新或击杀刷新机制通过直接修改技能实体的 `cooldown.timer` 属性实现，例如无双剑姬的 Q 技能命中敌人后返还冷却时间。
