#  Jarvan IV (嘉文四世)

## 基础信息

- **称号**: 嘉文四世 (The Exemplar of Demacia)
- **类型**: 物理 / 近战
- **难度**: 2/3
- **角色**: 战士
- **位置**: 打野
- **资源**: 法力
- **攻击距离**: 175

## 技能介绍

### 被动 - 战争律动 (Martial Cadence)

**类型**: 触发型被动
**目标免疫**: 每6/5/4/3秒 (基于等级)

Jarvan IV的普通攻击会获得强化，造成相当于目标当前生命值8%的额外物理伤害，最小20，对非英雄目标上限400。

这个效果对同一目标不能频繁触发。

### Q - 龙之打击 (Dragon Strike)

- **冷却**: 10/9/8/7/6 秒
- **消耗**: 45/50/55/60/65 法力
- **施法时间**: 0秒
- **目标范围**: 65-785 (边缘)
- **碰撞半径**: 180 (中心)
- **宽度**: 136-300

Jarvan IV向目标方向伸展长矛，对命中的敌人造成物理伤害，并使其护甲减少3秒。

**物理伤害**: 90/130/170/210/250 (+ 145% bonus AD)
**护甲减少**: 目标护甲的10/14/18/22/26%

如果长矛命中已部署的Demacian Standard，Jarvan IV会向其位置突进，沿途和附近敌人被击飞0.75秒。

Jarvan IV可以在突进期间施放任何技能。

### W - 黄金圣盾 (Golden Aegis)

- **消耗**: 30 法力
- **冷却**: 无 (静态)
- **施法时间**: 无
- **效果范围**: 600

**主动**: Jarvan IV减速所有附近敌人2秒。

**减速**: 15/20/25/30/35%

Jarvan IV也为自己获得一个持续4秒的护盾，每个被Golden Aegis命中的敌方英雄增加1%最大生命值的护盾强度。

**护盾强度**: 60/80/100/120/140 (+ 70% bonus AD)

### E - 德玛西亚旗帜 (Demacian Standard)

- **冷却**: 12/11.5/11/10.5/10 秒
- **消耗**: 55 法力
- **施法时间**: 无
- **目标范围**: 860
- **效果范围**: 200 (中心) / 1000 (视野)

**被动**: Jarvan IV获得额外攻击速度。

**额外攻击速度**: 20/22.5/25/27.5/30%

**主动**: Jarvan IV向目标位置投掷德玛西亚旗帜，对区域内的敌人造成魔法伤害。

旗帜会保留8秒，提供周围视野，并提供一个光环，向周围友方英雄和Jarvan IV提供Demacian Standard被动的额外攻击速度。

**魔法伤害**: 80/120/160/200/240 (+ 80% AP)

### R - 毁灭之刃 (Cataclysm)

- **冷却**: 120/105/90 秒
- **消耗**: 100 法力
- **施法时间**: 无
- **目标范围**: 650
- **效果范围**: 350
- **视野范围**: 1700
- **速度**: 目标距离 ÷ 0.35
- **不可通行边界**: 255 (路径半径) / 445

**主动**: Jarvan IV以位移免疫状态跃向目标敌人位置，在0.35秒内到达，落地时对所有附近敌人造成物理伤害。

**物理伤害**: 200/325/450 (+ 180% bonus AD)

落地时，他创造一个圆形不可通行地形，在0.15秒内将区域内敌人击出，如果敌人在外围边缘则被击出圈外，否则被拉向圈中心。地形持续3.5秒，并提供区域视野。

Cataclysm可以在0.75秒后再次施放，只要地形存在。

**再次施放**: Jarvan IV摧毁Cataclysm创造的地形。

## 连招技巧

### 基础连招

**标准Gank**:
- E(德玛西亚旗帜) → Q(龙之打击) → W(黄金圣盾)
- 旗帜配合Q击飞

**对线换血**:
- Q(龙之打击) → 普通攻击 → W(黄金圣盾)
- 被动配合输出

### 进阶连招

**团战连招**:
- E(德玛西亚旗帜) → Q(龙之打击) → R(毁灭之刃)
- 旗帜配合突进控制

**闪击连招**:
- R(毁灭之刃) → Q(龙之打击) → E(德玛西亚旗帜)
- 大招开团

### 隐藏机制

1. **EQ连招**: E放旗帜，Q命中旗帜可以突进并击飞
2. **被动伤害**: 被动对低血量目标伤害更高，最小20，最大400 (对非英雄)
3. **R地形**: R创造的地形可以阻挡敌人移动，外围敌人被击出，中心敌人被拉入
4. **W护盾**: W护盾值基于自身最大生命值增加
5. **E攻速**: E旗帜提供范围攻速加成光环
6. **R位移免疫**: R冲刺过程中具有位移免疫
