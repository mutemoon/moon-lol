## 技能介绍

### 被动 - 嗜火 (Pyromania)

**机制**: Annie每次用碎裂之火命中敌人或释放其他技能时，会获得一层嗜火层数，最多叠加4层。满层时Annie进入"充能"状态，下次释放碎裂之火、焚烧或召唤提伯斯时会眩晕命中的敌人。

**眩晕时长**: 1.25/1.5/1.75 秒 (基于等级)

Annie游戏开始时和复活时会获得满层嗜火。死亡时会失去"充能"状态和所有嗜火层数。

### Q - 碎裂之火 (Disintegrate)

- **消耗**: 60/65/70/75/80 法力
- **施法时间**: 0.25秒
- **目标范围**: 625
- **速度**: 1400
- **队列阈值**: 0

Annie向目标敌人投掷火球，造成魔法伤害。

**伤害**: 80/125/170/215/260 (+ 80% AP)

如果击杀目标，碎裂之火的冷却时间减少50%，且返还法力消耗。

### W - 焚烧 (Incinerate)

- **消耗**: 70/75/80/85/90 法力
- **施法时间**: 0.25秒
- **效果范围**: 600
- **角度**: 49.52度
- **队列阈值**: 0

Annie向目标方向释放锥形火焰，对范围内敌人造成魔法伤害。

**伤害**: 70/110/150/190/230 (+ 80% AP)

### E - 熔岩护盾 (Molten Shield)

- **冷却**: 10秒
- **消耗**: 60/65/70/75/80 法力
- **施法时间**: 无
- **目标范围**: 800
- **队列阈值**: 0

Annie为自己或目标友军英雄和提伯斯提供护盾，持续3秒，并获得20%-53.53%（基于等级）的移速加成（1秒内衰减）。

**护盾值**: 60/95/130/165/200 (+ 40% AP)

当熔岩护盾激活时，对护盾造成伤害的敌人会受到魔法伤害。每次激活每个敌人只能触发一次。

**伤害反弹**: 25/35/45/55/65 (+ 40% AP)

### R - 召唤提伯斯 (Summon: Tibbers)

- **冷却**: 130/115/100 秒
- **消耗**: 100 法力
- **施法时间**: 0.25秒（无后摇）
- **目标范围**: 600
- **效果范围**: 250 / 350

**被动**: Annie获得魔法穿透。

**魔法穿透**: 10/15/20%

**主动**: Annie在目标位置召唤提伯斯，对范围内的敌人造成魔法伤害。

**初始伤害**: 150/275/400 (+ 75% AP)

提伯斯会在战场上作为宠物存在最多45秒，可以被控制。

提伯斯存活时可以再次释放此技能，让提伯斯移动到目标位置。

## 连招技巧

### 基础连招

**标准爆发**:
- 被动满层 → W(焚烧) → Q(碎裂之火) → R(提伯斯)
- 被动满层时先手控制

**消耗连招**:
- Q(碎裂之火) → W(焚烧)
- Q击杀小兵时冷却刷新

### 进阶连招

**团战连招**:
- R(提伯斯) → W(焚烧) → Q(碎裂之火)
- R先手召唤提伯斯

**反手连招**:
- E(熔岩护盾) → R(提伯斯) → W(焚烧)
- 护盾保护后反击

### 技巧

1. **被动眩晕**: 被动满层时下次伤害技能眩晕敌人
2. **Q击杀刷新**: Q击杀目标返还冷却和蓝耗
3. **E护盾反弹**: E可以对攻击护盾的敌人造成伤害
4. **提伯斯控制**: 提伯斯可以手动控制移动

## 隐藏机制

1. **被动眩晕**: 被动满层时下次伤害技能眩晕敌人
2. **Q击杀刷新**: Q击杀目标返还50%冷却和法力
3. **E护盾反弹**: 每次激活每个敌人只能触发一次反弹伤害
4. **提伯斯宠物**: 提伯斯可以控制，有火焰光环伤害
5. **R可在控制下释放**: 召唤提伯斯可以在被沉默时释放
